# FLY VoltUp Power

    FLY-VoltUp Power该升压模块专为高压驱动设计，用户无需额外购买高压电源即可实现高压驱动，有助于在设计3D打印机时节省空间

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-power/img/power.webp').default} size="50%" align="left" />


## 产品特性

    * 输出电源48V/60V可选
    * 板载风扇口,并且自带4010风扇固定，电源输入多少V则输出多少V
    * 双电源接口扩展：提供一个电源接口，允许用户同时扩展两个电源输出，并且可单独制定某一个电源输出48V或者60V
    * 易于安装：采用标准尺寸设计，适用于多种环境，无需额外驱动安装。
    * 紧凑尺寸：扩展板尺寸合理，不会占用过多空间，方便集成到各种项目中。

## 产品规格：

    * 电源接口：12V/24V
    * 输出接口：两路可选电源 48V/60V  （通过跳线帽选择）
    * 风扇接口：两个XH2.54 2P  VCC FAN


## 注意事项：

    * 安装前断开电源并关闭所有设备
    * 在48V输出时，电机电流最大可设置为2A，而在60V输出时，电机电流最大可设置为1A
    * 两个VCC FAN接口的电压与电源输入一致，输出电压与电源电压相同
    * 模块预留了4010风扇的固定孔位，由于升压模块产生的热量较大，必须配备风扇进行散热
    * 内置防反接防导管芯片，防止避免客户操作失误导致零件损坏。
    * 不要同时插入电压选择跳线帽避免元器件损坏。
    * 正确接入正负极避免损坏元器件。
