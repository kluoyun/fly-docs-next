# CAN 브리지

CAN 버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스로, 트위스트 페어 케이블을 사용하여 신호를 전송할 수 있으며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신을 위해 사용되어, 비용이 많이 들고 무거운 배선 다발을 대체합니다. 이 프로토콜의 견고성 덕분에 다른 자동화 및 산업 애플리케이션으로 그 용도가 확장되었습니다. CAN 프로토콜의 특성에는 직렬 데이터 통신의 완전성, 실시간 지원 제공, 최대 1Mb/s의 전송 속도, 11비트의 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터는 CAN 버스를 사용하여 프린트 헤드와 메인보드 간의 연결선 수를 줄입니다. 원래 필요했던 수십 개의 선을 CAN을 사용하면 단 4개의 선만으로도 충분하여 배선 수를 크게 줄이고 배선 작업을 단순화합니다.
이 장에서는 브리지 CAN의 사용에 대해 간략히 개요만 설명합니다.

## 툴보드가 CAN을 통해 메인보드에 연결

    * 툴보드의 전원선은 CAN 포트에 연결되지 않고, 12V-24V 전원선에 연결됩니다.
