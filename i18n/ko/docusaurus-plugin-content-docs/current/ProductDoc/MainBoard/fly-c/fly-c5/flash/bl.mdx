# BL 펌웨어

    :::팁

    * 출하 시 이미 `부트로더`가 미리 플래시되어 있으므로 다시 플래시할 필요가 없습니다.
    * C5는 자체적으로 제공하는 상위기계를 사용해야 합니다.
    :::

## 상위기계에 SSH 연결

    :::경고
    - BL 펌웨어를 플래시하기 전에 `와이파이를 통해 상위기계에 SSH 연결`해야 합니다.
    :::

## 펌웨어 플래시


    * 다운 스위치를 위쪽으로 옮기고 하위기계의 USB를 상위기계에 연결합니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * 아래 명령어를 입력하여 BL을 플래시 시작합니다.


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```

    * 다음 그림과 같이 표시되면 플래시가 성공했으며 BOOT 버튼을 놓아도 됩니다.
