# BL Firmware

    :::tip

    * Please note that the `bootloader` has been pre-flashed at the factory and does not need to be flashed again.
    * C5 can only use the built-in host software for flashing.
    :::

## SSH Connection to Host Machine

    :::danger Warning
    - The BL firmware must be burned after using `WiFi connection to the host machine SSH`.
    :::

 * First, please: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh#using-wifi-to-connect-to-the-host-ssh">Use WiFi to Connect to the Host SSH</Button>
 ---
 <br/>

## Flashing Firmware


    * Set the dip switch upward and connect the downstream device USB to the host machine.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * Enter the following command to start flashing the BL


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```

    * When the following image appears, it proves that the flashing is successful and you can release the BOOT key.
       <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/hid3.webp').default} size="70%" align="center" />
