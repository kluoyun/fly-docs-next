# 와이파이 연결

    :::info 중요한 안내

    * 이 제품의 온보드 안테나는 수신만 가능하며, 송신 안테나는 직접 설치해야 하며, 안테나 단자는 `ipex3`입니다.
    * 안테나를 제대로 설치하세요!!! 그렇지 않으면 신호가 너무 약해서 심각한 발열을 초래할 수 있습니다!!!
    * 처음 사용할 때는 AirClick 핫스팟에 수동으로 연결한 후, 해당 WIFI에 연결해야 합니다.

    :::


## AirClick 와이파이 연결

    * AirClick의 핫스팟에 연결합니다. 핫스팟 이름은 `FLY-WBTN-XXXXXX`입니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
    * 핫스팟에 연결한 후 브라우저를 통해 [http://192.168.4.1/](http://192.168.4.1/#/setting)에 접속합니다.
    * 이때 WIFI 설정 인터페이스가 표시되며, 프린터와 동일한 라우터에 수동으로 연결하고 비밀번호를 입력한 후 아래쪽 파란색 상자에 연결하기를 클릭하여 WIFI에 연결합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
    * WIFI에 연결한 후 AirClick가 WIFI에 연결한 후의 IP 주소가 표시되며, 아래쪽 파란색 상자를 클릭하여 기기를 재시작할 수 있습니다.
        * 번호 1로 둘러싸인 것은 AirClick의 IP 주소입니다.
        * 번호 2로 둘러싸인 IP를 복사하고 기기를 재시작합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
    * 자신의 현재 네트워크를 해당 라우터로 수동으로 전환해야 하며, 그렇지 않으면 접속할 수 없음에 주의하세요.
    * 연결에 성공한 후에는 AirClick를 전원을 끄고 다시 켜는 것을 권장합니다.
