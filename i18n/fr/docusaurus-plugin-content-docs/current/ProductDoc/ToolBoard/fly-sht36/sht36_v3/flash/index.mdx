# Notice sur la programmation de la mise à jour du firmware
    
    * La compilation et la programmation du firmware nécessitent l'utilisation d'un ordinateur principal
    * SHT36 V3 peut être connecté à l'ordinateur principal via les méthodes RS232 ou CAN
    * Les connexions RS232 et CAN doivent utiliser le câble de données `XT30(2+2)-F`
    * Voici le modèle du câble de données `XT30(2+2)-F` ci-dessous
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Informations importantes
* La programmation du firmware de la carte mère doit passer par Katapult pour programmer Klipper
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, il n'est pas nécessaire d'effectuer la dernière étape de programmation du firmware Katapult, cela ne sert qu'à prévenir en cas d'échec de la programmation initiale
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour vérifier si une LED clignote sur la carte mère. Si c'est le cas, cela signifie que la carte mère est dans le firmware Katapult
:::

## Comment savoir quel firmware vous avez besoin

    * SHT36 V3 peut se connecter à l'ordinateur principal de la manière suivante
    * <Button variant="contained" disableElevation href="./can">Firmware CAN</Button>
        * Avantage : Si le firmware Katapult est installé, vous pouvez mettre à jour le firmware sans toucher à la carte mère et il est compatible avec la communication en bus
        * Inconvénient : L'utilisation conjointe de KlipperScreen peut entraîner des erreurs telles que le dépassement de délai lors du retour à la position zéro
        * Indication : Si vous souhaitez utiliser CAN, assurez-vous que l'ordinateur principal prend en charge CAN et qu'il est utilisé avec UTOC ou une carte mère flashée avec le firmware de pont
    * <Button variant="contained" disableElevation href="./rs232">Firmware RS232</Button>
        * Avantage : Nécessite aucun logiciel spécial pour fonctionner, communication fiable et stable
        * Inconvénient : Pour mettre à jour le firmware, il faut soit entrer en mode DFU manuel, soit appuyer deux fois sur le bouton RST pour entrer dans Katapult
        * Indication : Nécessite la combinaison avec G2T ou UTOR pour fonctionner
