# Connexion à RRF

    :::info Remarque importante
    * AirClick doit être connecté au même réseau que l'imprimante cible
    :::

1. Cliquez sur le paramètre système à gauche

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. Sélectionnez `RRF` pour `Printer Type`
3. Entrez l'adresse IP de la machine RRF dans `RepRapFirmware IP`

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/rrf.webp').default} size="80%" align="Center" />

4. Après avoir configuré comme indiqué ci-dessus, cliquez sur `Soumettre` dans le coin inférieur droit pour enregistrer la configuration
5. Après une connexion correcte, un message s'affichera en haut de la page web

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

6. Si la connexion échoue
    - Vérifiez si l'adresse IP est correcte
    - Vérifiez si l'imprimante et AirClick sont connectés au même réseau
    - Vérifiez si la carte mère RRF fonctionne normalement
    - Vérifiez si le navigateur peut ouvrir la page web de RRF
