# Connexion WiFi

    :::info Indication importante

    * L'antenne embarquée de ce produit ne peut être utilisée que pour la réception, l'antenne d'émission doit être installée par vous-même, le terminal d'antenne est `ipex3`.
    * Veuillez installer l'antenne correctement!!! Sinon, un mauvais signal entraînera une surchauffe grave!!!
    * Lors de la première utilisation, il est nécessaire de se connecter manuellement au point d'accès AirClick, puis de se connecter au WIFI correspondant.

    :::


## Connexion WIFI d'AirClick

    * Se connecter au point d'accès d'AirClick, le nom du point d'accès est `FLY-WBTN-XXXXXX`.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
    * Après la connexion au point d'accès, accédez via le navigateur à [http://192.168.4.1/](http://192.168.4.1/#/setting).
    * À ce moment-là, l'interface de configuration du WIFI s'affichera, vous devez vous connecter manuellement au routeur identique à celui de l'imprimante, saisir le mot de passe, puis cliquer sur la case bleue en bas pour vous connecter au WIFI.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
    * Après la connexion au WIFI, l'adresse IP d'AirClick après la connexion au WIFI sera indiquée, cliquez sur la case bleue en bas pour redémarrer l'appareil.
        * Le numéro 1 entouré est l'adresse IP d'AirClick.
        * Le numéro 2 entouré, copiez l'IP et redémarrez l'appareil.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
    * Veuillez noter qu'il est nécessaire de basculer manuellement votre réseau actuel vers le routeur correspondant, sinon vous ne pourrez pas y accéder.
    * Après une connexion réussie, il est recommandé de redémarrer l'AirClick en le mettant hors tension.
