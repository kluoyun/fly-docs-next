# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.png').default} size="40%" align="left" />


## Product Features

    * Supports CAN and power distribution to four, allowing for the simultaneous expansion of four CAN tool boards.
    * The board is equipped with an RJ11 interface, designed for connection and communication with systems using the CAN protocol. Although RJ11 is not a standard interface in CAN systems, our product provides a smooth connection between the two.
    * User-friendly design, which can be directly mounted on a rail using a printed part.

## Product Specifications

    * Communication Interface: RJ11*1, XH2.54*5
    * Power Interface: KF128-5.028-2P*5
    * Input Voltage: 5V-24V
    * Compatibility: RRF/Klipper/etc. CAN

## Precautions

    :::warning Precautions

    * Please note that the output voltage will be the same as the input voltage.
    * Please ensure that the wiring sequence is correct.
    * Ensure that the main controller is powered off before installing the expansion board.
    * Avoid using the expansion board in humid or dusty environments.
    * When installing the device, pay attention to the voltage specifications of the tool board to avoid damage.

    :::
