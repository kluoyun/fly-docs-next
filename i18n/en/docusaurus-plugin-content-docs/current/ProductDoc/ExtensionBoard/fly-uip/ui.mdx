# Boot Screen

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/uip.webp').default} size="100%" align="left" />

## First Screen
    :::warning Notice

    * The system needs to support ECM or Rndis, otherwise it cannot be used normally
    * This product is designed specifically for Klipper and requires the Moonraker service to display the IP and resources; otherwise, you need to open the background and configure SSH mode to use it
    
    :::
    * The IP display will show the information read from the connected machine; if the IP cannot be read or is not read, a prompt will appear
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ip.webp').default} size="100%" align="left" />
    * If read successfully, the following screen will appear (the IP is displayed according to the host machine)
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ip1.webp').default} size="100%" align="left" />

## Second Screen

    * Displays the CPU and memory usage of the connected machine
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/cpu.webp').default} size="100%" align="left" />

## Third Screen

    * Displays the services started on the host machine; note that only services related to Klipper are shown
    * If not started, a red dot will appear before the service
    * If started, a blue dot will appear before the service
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/service.webp').default} size="100%" align="left" />
