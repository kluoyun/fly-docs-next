화면 연결 및 설정

정보
  • lite2.1은 SSH를 통해 시스템에 연결한 후 아래 명령어를 입력해야 Fly Tools 화면 설정을 사용할 수 있습니다
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
정보
  • Fly Tools는 상위 컴퓨터가 네트워크에 연결된 후 상위 컴퓨터의 IP+:9999로 접근할 수 있습니다
  • 예를 들어 192.168.6.110:9999
  • TFT 화면
  • HDMI 설정
  • FLY TOOLS에서 KS를 켜야 합니다
  • TFT 정전식 화면 설정
  • TFT 저항식 화면 설정

방법 1​

  • 정전식 설정은 아래 명령어를 사용하여 fly-tft-v2-c로 설정해야 합니다
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
  • 화면 뒷면의 스위치가 Cap으로 설정되어 있는지 확인하십시오
  • 아래 명령어를 사용하여 설정이 성공적으로 수정되었는지 확인하십시오
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
  • 정전식 화면을 저항식 화면으로 변경하려면 아래 명령어를 사용하십시오
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt

방법 2​

  • 사용 도구: 카드 리더기, lite2 이미지가 설치된 SD/TF 카드
  • SD/TF 카드를 카드 리더기에 삽입한 후 컴퓨터 USB 포트에 연결하면 boot 드라이브가 나타납니다
  • 마우스 오른쪽 버튼으로 boot 드라이브를 클릭하면 아래 이미지 파일이 나타납니다
  • FlyOS-Env.txt라는 파일을 찾습니다
  • FlyOS-Env.txt 파일을 오른쪽 더블 클릭하여 엽니다
  • 이미지에서 8행 11행을 찾아 이미지에 따라 설정을 수정합니다
