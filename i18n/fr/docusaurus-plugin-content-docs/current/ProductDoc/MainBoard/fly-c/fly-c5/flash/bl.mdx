# Firmwares BL

    :::tip

    * Notez bien que le "bootloader" est préinstallé à la sortie d'usine, pas besoin de le réinstaller.
    * Le C5 ne peut utiliser que l'application propriétaire pour le flashage.
    :::

## Connexion SSH à l'ordinateur principal

    :::danger Attention
    - Avant de flasher les firmwares BL, veillez à vous connecter via `WiFi` à l'ordinateur principal via SSH.
    :::

## Flashage des firmwares


    * Positionnez l'interrupteur de commutation vers le haut et connectez l'appareil inférieur à l'ordinateur principal via USB.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * Entrez la commande ci-dessous pour démarrer le flashage du BL


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```

    * Lorsque vous voyez l'image ci-dessous, cela signifie que le flashage a réussi et que vous pouvez relâcher la touche BOOT.
       <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/hid3.webp').default} size="70%" align="center" />
