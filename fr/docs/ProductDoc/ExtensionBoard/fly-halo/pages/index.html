Accueil​

  • Les zones opérationnelles de l'image 1 ci-dessus correspondent aux numéros en dessous
  1. Clic : Entrer dans la configuration de température d'extraction, ajuster la température en glissant la barre circulaire, cliquer à nouveau sur la zone 1 pour confirmer la température définie
  2. Clic : Entrer dans la configuration de température du lit chauffant, ajuster la température en glissant la barre circulaire, cliquer à nouveau sur la zone 2 pour confirmer la température définie

Page d'extrusion​

  • Les zones opérationnelles de l'image 1 ci-dessus correspondent aux numéros en dessous
  1. Clic : Augmenter la température cible de l'extrudeur de 1 degré
  2. Appui long : Entrer dans la configuration de température d'extraction, ajuster la température en glissant la barre circulaire, cliquer à nouveau pour confirmer la température définie
  3. Clic : Diminuer la température cible de l'extrudeur de 1 degré
  4. Clic : Extrusion
  5. Glisser vers le haut ou vers le bas : Sélectionner la longueur d'extraction ou de retrait, unité mm
  6. Clic : Retrait
  7. Appui long : Entrer dans la configuration de vitesse d'extraction de l'image 2 ci-dessus, sélectionner la vitesse d'extraction en glissant vers le haut ou vers le bas, unité mm/s, cliquer sur une zone vide autour après la sélection pour quitter

Page des services système​

  • Cliquez sur les services correspondants de l'image 1 ci-dessus pour faire apparaître l'image 2 avec une fenêtre contextuelle, pour démarrer, redémarrer, arrêter les services systèmes correspondants
  • La mise à jour OTA ultérieure prendra en charge tous les services pouvant être manipulés dans Moonraker
