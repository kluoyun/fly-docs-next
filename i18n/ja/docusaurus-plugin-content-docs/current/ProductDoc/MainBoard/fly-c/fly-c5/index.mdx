## 製品特長

- 32ビット ARM Cortex-M4、550MHz、STM32H723VGT6 チップ
- TMC ドライバ対応の SPI、UART、STEP/DIR モードをサポート
- DIAG 関数ピンは限界スイッチを占有せず、設定後に無限限界リターンが可能
- デジタル制御ファンが 24V、12V、5V をサポートし、追加の降圧モジュールが不要
- 断電時の自動シャットダウンと断電再印刷がサポートされています（デフォルトで自動シャットダウンが有効）
- USB Klipperファームウェアが工場出荷時にインストール済み
- マルチポート電源管理、ポート、限界スイッチ、ドライバのショートは上位機器の動作に影響しません
- 5V RGB インターフェースを予備搭載
- TFT および HDMI スクリーンをサポートするワンケーブル接続
- システムには klipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグイン numpy が事前にインストールされており、ソース変更が不要で初心者にも簡単です（TFカードまたはM2WEの購入が必要）

## 製品仕様

### 上位機器

- **CPU**：全志 H618、4コア 64ビット Cortex-A53
- **GPU**：Arm Mali-G31
- **RAM**：1GB DDR4
- **ROM**：最大128GB SDカードに対応
- **周辺機器**：
    - TFT x1
    - USB 2.0 x2
    - Micro HDMI x1
    - エチルネットポート（Eth）x1（100M）
    - FPC-HDMI x1
    - UART x1
- **インターフェース**：M.2 ボード内蔵、Wi-FiやeMMCなど拡張可能（プライベートプロトコルのため、非FLY専用のM.2デバイスには接続しないでください）
- **サポート**：TFTスクリーンとHDMIスクリーンFPCワンケーブル接続
- **システム事前インストール**：klipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpy、ソース変更不要で初心者にも簡単（TFカードまたはM2WEの購入が必要）

### 下位機器

- **MCU**：STM32H723VGT6
- **ファームウェア**：KLIPPER/RRF/Marlin
- **電源供給**：12-24V DC電源をサポートし、降圧モジュールが不要
- **ヒートベッドポート最大出力電流**：15A
- **ヒーター棒ポート最大出力電流**：6A
- **ファンポート最大出力電流**：1A
- **ドライバ動作モード**：SPI、UART、STEP/DIR
- **サポートドライバ**：TMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988など
- **拡張インターフェース**：BLTouch（Servos、Probe）、RGB x1、UART x1、CAN x1
- **ファン**：2線式デジタル制御ファン x4（24V、12V、5Vを選択可能）
- **温度センサ**：3路NTC100K
- **サポート構造**：Cartesian、Delta、Kossel、Ultimaker、CoreXYなど
