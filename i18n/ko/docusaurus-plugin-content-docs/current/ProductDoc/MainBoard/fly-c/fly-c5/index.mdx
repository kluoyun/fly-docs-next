## 제품 특징

- 32비트 ARM Cortex-M4, 550MHz, STM32H723VGT6 칩셋
- TMC 드라이버 지원 SPI, UART, STEP/DIR 작업 모드
- DIAG 기능 핀은 리미터를 차지하지 않으며 설정 후 무한리미터 리세팅 가능
- CNC 팬 지원 24V, 12V, 5V, 추가적인 다운레벨 모듈 필요 없음
- 전원 꺼짐 시 자동 종료 및 전원 꺼짐 후 재인쇄 기능 지원 (기본적으로 전원 꺼짐 자동 종료 기능 활성화)
- USB Klipper 펌웨어 출하 시 기본 설치
- 다중 전원 관리, 포트, 리미터, 드라이버 단락이 상위 컴퓨터 작동에 영향을 미치지 않음
- 5V RGB 인터페이스 예비 장착
- TFT 및 HDMI 화면 지원 One-Wire 기능
- Klipper, Moonraker, Mainsail, Fulldid, Klipper-Screen, Crowsmest, Resonance Compensation 플러그인 Numpy 시스템 기본 설치, 소스 변경 필요 없음, 사용 쉬움 (TF 카드 또는 M2WE 구매 필요)

## 제품 사양

### 상위 컴퓨터

- **CPU**：Allwinner H618, 4코어 64비트 Cortex-A53
- **GPU**：Arm Mali-G31
- **RAM**：1GB DDR4
- **ROM**：최대 128GB SD카드 지원
- **외부 장치**：
    - TFT x1
    - USB 2.0 x2
    - Micro HDMI x1
    - 유선랜 포트 (Eth) x1 (100M)
    - FPC-HDMI x1
    - UART x1
- **인터페이스**：내장 M.2, Wi-Fi, eMMC 등 확장 가능 (프라이빗 프로토콜 사용, 비 FLY 전용 M.2 장치 연결 금지)
- **지원**：TFT 화면과 HDMI 화면 FPC One-Wire 기능
- **시스템 기본 설치**：Klipper, Moonraker, Mainsail, Fulldid, Klipper-Screen, Crowsmest, Resonance Compensation 플러그인 Numpy, 소스 변경 필요 없음, 사용 쉬움 (TF 카드 또는 M2WE 구매 필요)

### 하위 컴퓨터

- **MCU**：STM32H723VGT6
- **펌웨어**：KLIPPER/RRF/Marlin
- **전원 공급**：12-24V DC 전원 지원, 다운레벨 모듈 제거
- **열 침대 포트 최대 출력 전류**：15A
- **히터 포트 최대 출력 전류**：6A
- **팬 포트 최대 출력 전류**：1A
- **드라이버 동작 모드**：SPI, UART, STEP/DIR
- **지원 드라이버**：TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등
- **확장 인터페이스**：BLTouch (Servos, Probe), RGB x1, UART x1, CAN x1
- **팬**：2선 CNC 팬 x4 (24V, 12V, 5V 선택 가능)
- **온도 센서**：3개의 NTC 100K
- **지원 구조**：Cartesian, Delta, Kossel, Ultimaker, CoreXY 등
