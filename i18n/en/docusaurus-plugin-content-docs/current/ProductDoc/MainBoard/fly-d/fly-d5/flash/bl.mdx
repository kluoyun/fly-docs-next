## Entering Flash Mode

    :::tip

    Left is the `BOOT button`, right is the `RESET button`

    :::

    * There are two ways to enter flash mode
    * First method: With the main board completely powered off, hold the BOOT button and connect one end of the data transfer cable to the main board and the other end to your device
    * Second method: Connect one end of the data transfer cable to the main board and the other end to your device, then hold the BOOT button and press the RST button, release the RST button first, then release the BOOT button
