连接上位机SSH

  • 可以使用串口或者WiFi连接到设备
  • 请提前准备好MobaXterm_Personal或其他SSH终端工具
  • 请确定SD卡或者M2WE已经烧录与上位机相匹配的系统
  • 如果上位机不能正常启动请把驱动，限位，风扇，等外设拆除！！！
注意
  • 如果您的上位机为第一次启动，请务必使用串口连接到上位机SSH
  • 编译固件时，请务必使用WiFi连接到上位机SSH，以方便固件编译和烧录！
  • FLYOS-Armbian系统与FLYOS-FAST系统 连接至SSH的操作有所区别，请根据您烧录的系统选择相应的教程！！！

FLYOS-Armbian WiFi连接到上位机SSH​

  • 请确保上位机系统已经正常启动

  • 请确保明确上位机的用户名与密码。FLY系统默认用户名fly, 默认密码mellow

  • 通过WiFi连接到上位机SSH的前提是：您的上位机已经连接到路由器，并且知道上位机的IP

  • 打开MobaXterm，设置如下图所示，Remote host处则填写您在串口连接时查询到的上位机IP，也可以通过路由器后台查询上位机IP

注意：这里第4步输入的上位机的用户名

提示
  • 第四步输入上位机的用户名。fly上位机且使用FLYOS-Armbian系统的则填写fly，其他上位机请根据实际情况填写正确的用户名
  • 第一次连接回弹出这个窗口，点击Accept即可
  • 提示输入密码，输入上位机密码后回车
提示
