Katapult 펌웨어

Katapult 펌웨어 설명​

팁

Katapult의 원래 이름은 CanBoot였지만 나중에 이름이 바뀌었습니다.

  • Katapult 펌웨어는 부트로더의 일종으로, Klipper와 함께 사용할 수 있습니다. DFU 등의 방식으로 번인 후 명령어로 펌웨어를 일괄적으로 업데이트할 수 있습니다.
  • 장점: 두 번 리셋하거나 빠르게 두 번 전원을 차단하면 Katapult 펌웨어로 진입할 수 있으며, 사전 설정된 LED가 깜박이기 시작합니다. LED가 없으면 깜박이지 않습니다.
  • Katapult 펌웨어는 펌웨어를 번인하는 데만 사용되며, klipper에 연결할 수 없습니다!!!
  • Katapult 펌웨어는 상위 컴퓨터와 연결하는 세 가지 방법을 지원합니다.
    • CAN 연결
      • 장점: KLIPPER 펌웨어를 컴파일한 후 명령어로 직접 펌웨어를 업데이트할 수 있습니다.
      • 단점: 펌웨어 컴파일 속도를 미리 알아야 합니다.
    • RS232 연결
      • 장점: KLIPPER 펌웨어를 컴파일한 후 명령어로 펌웨어를 업데이트할 수 있습니다.
      • 단점: 명령어로 일괄적으로 펌웨어를 업데이트할 수 없으며, 수동으로 Katapult 펌웨어로 들어가야 합니다.
    • USB 연결
      • 장점: KLIPPER 펌웨어를 컴파일한 후 명령어로 직접 펌웨어를 업데이트할 수 있습니다.
      • 단점: 너무 길면 간섭을 받기 쉽습니다.
팁
  • D7은 USB 연결을 사용합니다.
  • D7은 2024년 9월 1일 출고 시 이미 Katapult 펌웨어가 사전 플래싱되어 있습니다.

플래싱 모드 진입​

팁

왼쪽이 BOOT 버튼, 오른쪽이 RESET 버튼입니다.

  • 플래싱 모드로 진입하는 방법은 두 가지가 있습니다.
  • 첫 번째: 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채로 데이터 전송이 가능한 케이블을 메인보드와 장치에 연결합니다.
  • 두 번째: 데이터 전송이 가능한 케이블을 메인보드와 장치에 연결한 후, BOOT 버튼을 누른 채로 RST 버튼을 누르고, RST 버튼을 먼저 놓고 마지막으로 BOOT 버튼을 놓습니다.

펌웨어 플래싱​

  • 윈도우 플래싱 가이드
  • 호스트 머신 플래싱 가이드
  • D7의 Katapult 펌웨어 자동 다운로드
주의 사항
  • 시스템 호환성 문제로 인해 컴퓨터의 USB2.0을 사용하여 플래시
  • 그래도 플래시가 되지 않으면 다른 호스트 컴퓨터를 사용하여 플래시
  1. 플래싱 도구 다운로드 링크
  2. SetupSTM32CubeProgrammer와 STM32 DFU 드라이버를 다운로드 필요
  3. 앞서 컴파일한 펌웨어(klipper.bin) 또는 사전 컴파일된 Katapult 펌웨어를 컴퓨터의 임의 디렉토리로 복사
  4. Type-C 데이터 케이블을 사용하여 메인보드를 컴퓨터에 연결, DFU 플래싱 모드로 진입했는지 확인
  5. STM32CubeProgrammer에서 USB 모드 선택 후 새로고침, 연결
  6. 오류 창이 나타나지 않으면 연결 성공
  7. 왼쪽 다운로드 아이콘 클릭 후 아래 몇 가지 선택
  8. 화살표가 가리키는 Browse 클릭 후 플래싱할 펌웨어 선택
  9. Start address 선택 후 0x08000000 입력
  10. 아래의 Start automatic mode 클릭하여 펌웨어 플래싱 시작
  11. 아래와 같은 메시지가 나타나면 펌웨어 플래싱 완료
  1. 설치 프로그램 설치
  • FLY 상위 기기는 설치 프로그램을 실행할 필요가 없습니다.
  • 설치가 필요한 경우 주의사항: 상위 기기는 인터넷 연결이 필요합니다.
  • 이미 설치한 경우 다시 설치할 필요가 없습니다.
sudo apt update
sudo apt install dfu-util -y
  1. 아래 명령어를 실행하여 연결이 성공했는지 확인하십시오.
lsusb
  • DFU 장치가 없으면 번인 모드로 다시 진입해 보십시오.
  • 시스템 버전 문제로 다른 이름이 표시될 수 있지만, 0483:df11이 있으면 번인 모드로 진입한 것입니다.
  1. 미리 컴파일된 펌웨어 가져오기
  • 여기에는 미리 컴파일된 펌웨어가 모두 포함되어 있습니다.
  • 상위 기기는 인터넷 연결이 필요합니다.
  • 상위 기기에서 다음 명령어를 실행하여 펌웨어 패키지를 다운로드하십시오.
cd && git clone https://github.com/FengHua741/FLY_Katapult.git
  • 이미 펌웨어 패키지를 다운로드한 경우, 업데이트 후 나머지 작업을 진행하는 것이 좋습니다.
cd ~/FLY_Katapult && git pull
  1. 펌웨어 플래싱
팁

하나의 펌웨어만 플래싱하면 됩니다.

  • 아래는 펌웨어 플래싱 명령어입니다. 
    sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/D/FLY_D7.bin

  1. 화살표로 표시된 File downloaded successfully 메시지가 나타나면 플래싱이 성공한 것입니다.

팁

플래싱이 완료된 후, 메인보드를 완전히 한 번 전원을 꺼야 합니다.
