# Обновление прошивки
    :::danger Важные замечания

    * Для использования необходима система Windows
    * Способы подключения могут быть только следующие два, и выбирать нужно только один из них, других способов подключения нет!!!

    :::
## Скачивание прошивки

    <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/usblcd/fly-uip.uf2/version/52147655/files">Адрес загрузки прошивки</Button>
    * Нажмите на список файлов
    * Затем нажмите левой кнопкой мыши на ползунок внизу и удерживая его, потяните вправо
    * Затем нажмите кнопку "Скачать"
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/firmware.webp').default} size="100%" align="left" />


## Метод обновления
    * Подключите U-IP к компьютеру и в адресной строке браузера введите ``192.168.4.1``
    * Найдите системное управление и нажмите "Перезагрузить в режим программирования"
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

### Процесс обновления
    
    * После нажатия "Перезагрузить в режим программирования" на компьютере появится диск `RPI-RP2`
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf2.webp').default} size="100%" align="center" />
    * Скопируйте нужную прошивку в диск
    * Ссылка
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf21.webp').default} size="100%" align="center" />
    * После копирования прошивки дождитесь примерно ``десяти секунд``, после чего диск `RPI-RP2` автоматически исчезнет, что означает завершение процесса прошивки!
