상위기기 SSH 연결

::: danger

  • 이 문서는 FLY 상위기기에만 적용됩니다. 다른 장치의 경우 참고용입니다.
  • 펌웨어를 컴파일하고 플래시하려면 네트워크 연결 튜토리얼을 참조하세요. 시리얼 포트 연결 튜토리얼은 참고하지 마세요.

:::

  • 장치에 시리얼 또는 Wi-Fi로 연결할 수 있습니다.
  • MobaXterm_Personal 또는 다른 SSH 터미널 도구를 미리 준비해 주세요.
  • SD 카드 또는 M2WE에 해당 상위기기의 시스템이 플래시되었는지 확인하세요.
  • 만약 상위기기가 정상적으로 시작되지 않으면 드라이버, 리미터, 팬 등 외부 장치를 제거하세요!!!
  • 마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 컴파일할 수 있도록 하세요!!
  • 펌웨어를 컴파일하려면 Wi-Fi 또는 이더넷 케이블을 통해 상위기기와 연결하는 튜토리얼을 참조하세요.
  • 시리얼 연결
  • 네트워크 연결

상위기기 SSH에 시리얼을 사용하여 연결하기​

  • USB Type-c 케이블을 사용하여 FLY 상위기기를 컴퓨터와 연결하세요.
  • 다른 상위기기는 연결 방법을 자세히 확인하세요!!
  • FLY-C8
  • FLY-Pi v2
  • FLY-Pi(π)
  • FLY-Gemini
  • FLY-MiniPad
  • FLY-Pi Lite2
  • FLY-C8 포트
  • FLY-Pi v2 포트
  • FLY-Pi 포트
  • FLY-Gemini 포트
  • FLY-MiniPad 포트
  • FLY-Pi Lite2 포트
  • 메인보드와 컴퓨터가 정상적으로 연결되면 장치 관리자를 열어 CH340 포트를 볼 수 있습니다.
  • MobaXterm을 열고 아래와 같이 설정합니다. 포트는 앞서 장치 관리자에서 본 포트를 선택하세요.
  • 모든 것이 정상적으로 작동하면 시작 화면을 볼 수 있습니다.
  • 잠시 기다리면 이 화면이 나타나면 시작된 것입니다.

몇 분 후에도 다음 그림처럼 아무런 표시가 없다면 몇 번의 엔터 키를 누르세요

엔터 키를 누른 후에는 다음 그림과 같은 내용이 표시됩니다

  • IP를 확인하려면 SSH에서 다음 명령어를 입력하세요: ip a | grep inet 엔터, 반환된 내용 중 라우터와 동일한 세그먼트의 IP를 찾아 접속하세요

상위기기 SSH에 네트워크 IP를 사용하여 연결하기​

  • 시스템이 정상적으로 시작되었는지 확인하세요.
  • 상위기기와 연결된 라우터 관리자에 로그인하세요. (상위기기의 IP 주소를 알고 있다면 필요 없습니다.)
  • 상위기기의 사용자 이름과 비밀번호를 알고 있는지 확인하세요. FLY 시스템의 기본 사용자는 fly, 기본 비밀번호는 mellow입니다.
  • 메인보드가 시작되고 네트워크에 연결되었다면 다음 방법으로 SSH에 연결할 수 있습니다.
  • MobaXterm을 열고 아래와 같이 설정하세요. IP는 앞서 확인한 것을 입력하세요. (라우터 관리자에서도 확인 가능)

주의: 여기서 4단계에 상위기기의 사용자 이름을 입력하세요

  • 첫 번째 연결 시 이 창이 나타납니다. Accept를 클릭하세요.
  • 비밀번호를 입력하라는 메시지가 표시되면 상위기기의 비밀번호를 입력하고 엔터를 누르세요

주의: 여기서 입력하는 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르세요
