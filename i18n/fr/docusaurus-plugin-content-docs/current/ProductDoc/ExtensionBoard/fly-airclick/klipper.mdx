# Connexion à Klipper

    :::info Remarque importante
    * AirClick doit être connecté au même réseau que l'imprimante cible
    :::

1. Cliquez sur les paramètres du système à gauche

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. Sélectionnez `Klipper` pour `Printer Type`
3. Entrez l'adresse IP de la machine Klipper dans `Moonraker IP`
4. Entrez le port de l'hôte Klipper supérieur dans `Moonraker Port`, par défaut, c'est `7125`. Si l'hôte supérieur utilise un autre port, vous devez entrer le port correspondant.
5. Ignorez `Apikey`, il n'est pas pris en charge pour le moment.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. Une fois la configuration effectuée comme indiqué ci-dessus, cliquez sur `Soumettre` en bas à droite pour enregistrer la configuration.
7. Après une connexion correcte, un message s'affichera en haut de la page Web.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. Si la connexion échoue
    - Vérifiez si l'adresse IP et le port sont corrects.
    - Vérifiez si l'imprimante et AirClick sont connectés au même réseau.
    - Vérifiez si l'hôte Klipper supérieur fonctionne normalement.
    - Vérifiez si le navigateur peut ouvrir la page Web de l'hôte Klipper supérieur.
