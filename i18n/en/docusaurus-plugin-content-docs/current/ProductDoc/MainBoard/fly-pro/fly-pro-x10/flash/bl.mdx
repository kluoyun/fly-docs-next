# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at the factory and does not need to be re-flashed.

    :::

## Entering Burn Mode


    * There are two methods to enter burn mode
    * Method 1: With the motherboard completely powered off, short the BOOT pin and connect one end of the data transfer cable to the motherboard and the other end to your device.
    * Method 2: Connect one end of the data transfer cable to the motherboard and the other end to your device, then short the BOOT pin and press the RST button, then release the RST button, and finally disconnect the BOOT jumper.
