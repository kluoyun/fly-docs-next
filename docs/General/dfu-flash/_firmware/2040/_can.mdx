    下面介绍如何编译固件：
    * 连接到SSH后输入下面的命令并回车：

        ```bash
        cd ~/klipper && rm -rf .config ./out && make menuconfig
        ```

    * 其中` rm -rf .config ./out`是删除之前编译数据与固件`make menuconfig`是编译固件,执行后应该出现下方界面

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="left" />

    * 选择**Enable extra low-level configuration options**并且回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="left" />

    * 进入菜单**Micro-controller Architecture**然后选择**STMicroelectronics STM32**回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="left" />

    * 进入菜单**Processor model**，选择**STM32F072**回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="left" />

    * 选择**Bootloader offset**，选择：**8KiB bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.png').default} size="100%" align="left" />

    * 选择**Communication interface**，选择：**CAN bus (on PB8/PB9)**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/can.png').default} size="100%" align="left" />
