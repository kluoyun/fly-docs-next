# Способ использования

## Метод подключения
    :::warning Важное замечание

    * Подключаться можно только двумя указанными методами, и выбирать нужно только один из них! Нет других способов подключения!!!

    :::

    ``Метод 1:`` Подключите USB A (мужской разъем) U-IP к USB A (женский разъем) хост-компьютера.
    
    ``Метод 2:`` Используйте переходник (USB к TYPE-C). Одно из концов подключите к USB A (женскому разъему) хост-компьютера, а другое к U-IP.

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/host.webp').default} size="100%" align="left" />


    ``Затем:`` После подключения просто дождитесь нормальной готовности.

# Метод настройки U-IP

    :::danger Важное замечание

    * Для пользователей Apple данный функционал временно недоступен
    * Требуется система WINDOWS для работы
    * Подключение возможно только двумя указанными методами, и выбирать нужно только один из них! Нет других способов подключения!!!

    :::

    ``Метод 1:`` Подключите USB A (мужской разъем) U-IP к USB A (женскому разъему) компьютера.
    
    ``Метод 2:`` Используйте переходник (USB к TYPE-C). Одно из концов подключите к USB A (женскому разъему) компьютера, а другое к U-IP.
    
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/pc.webp').default} size="100%" align="left" />


    ``Затем:`` После подключения подождите ``пять секунд``, затем откройте браузер в системе WINDOWS и введите в адресную строку ``http://192.168.4.1/``, нажмите Enter, после чего появится следующий интерфейс:

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ui.webp').default} size="100%" align="left" />

## Описание интерфейса

### Настройка сети
    * Можно задать IP-адрес, MAC-адрес, имя хоста и порт административного интерфейса U-IP.
    * Не рекомендуется изменять параметры без необходимости.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/network.webp').default} size="100%" align="left" />

### Настройка отображения
    * Можно задать направление отображения, яркость подсветки и показ IP-адреса устройства подключенного через IPv6.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/display.webp').default} size="100%" align="left" />

### Управление системой
    * Можно задать версию прошивки и выполнить перезагрузку устройства или вход в режим программирования U-IP.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

## Настройка соединения системы

### Настройка Moonraker
    * Первый параметр — это включение соединения с Moonraker, по умолчанию включено.
    * Второй параметр — автоматически присвоенный IP-адрес устройству.
    * Третий параметр — порт Moonraker по умолчанию 7125.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/mmoonraker.webp').default} size="100%" align="left" />

### Настройка SSH
    * Для специальных систем можно попробовать этот метод, но обратите внимание, что не все устройства могут работать корректно с этой функцией.
    * При включении SSH подключения введите учетные данные SSH устройства, порт по умолчанию 22.
    * Этот параметр позволяет подключиться к устройству через SSH и получить его IP-адрес и ресурсы системы. Если получение информации невозможно, устройство будет застревать на экране загрузки.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ssh.webp').default} size="100%" align="left" />
