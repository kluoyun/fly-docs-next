# BL固件

    :::tip

    * 请注意出厂已经预刷了`bootloader`无需在刷
    * C5只能使用自带上位机烧录
    :::

    :::danger 警告
    - BL固件烧录前需要使用`wifi连接到上位机SSH`

## 烧录固件


    * 拨码开关朝上拨，将下位机USB连接上位机
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * 输入下方指令开始烧录BL


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```
