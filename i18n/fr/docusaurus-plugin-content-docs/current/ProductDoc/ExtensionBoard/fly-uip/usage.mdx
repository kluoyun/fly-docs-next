# Méthode d'utilisation

## Méthode de connexion
    :::avertissement Remarque

    * Les méthodes de connexion ne peuvent utiliser que les deux méthodes suivantes, et il faut choisir l'une des deux, sans autre méthode de connexion possible !!!

    :::

    ``Méthode un :`` Connecter le port USB A (masculin) de U-IP au port USB A (féminin) du poste de travail.
    
    ``Méthode deux :`` Utiliser un câble de données (USB vers TYPE-C), une extrémité connectée au port USB A (féminin) du poste de travail et l'autre extrémité connectée à U-IP.

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/host.webp').default} size="100%" align="left" />


    ``Ensuite :`` Attendez normalement après avoir branché les fils.

# Méthode de configuration de U-IP

    :::danger Attention

    * Les utilisateurs Apple ne sont pas encore pris en charge.
    * Un système WINDOWS est nécessaire pour utiliser cela.
    * Les méthodes de connexion ne peuvent utiliser que les deux méthodes suivantes, et il faut choisir l'une des deux, sans autre méthode de connexion possible !!!

    :::
    ``Méthode un :`` Connecter le port USB A (masculin) de U-IP au port USB A (féminin) de l'ordinateur.
    
    ``Méthode deux :`` Utiliser un câble de données (USB vers TYPE-C), une extrémité connectée au port USB A (féminin) de l'ordinateur et l'autre extrémité connectée à U-IP.
    
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/pc.webp').default} size="100%" align="left" />


    ``Ensuite :`` Attendez ``cinq secondes`` puis utilisez le navigateur WINDOWS et entrez ``http://192.168.4.1/`` dans la barre d'URL, appuyez sur Entrée pour afficher l'interface ci-dessous.

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ui.webp').default} size="100%" align="left" />

## Présentation de l'interface

### Paramètres réseau
    * Peut définir l'adresse IP, l'adresse MAC, le nom d'hôte et le port de gestion de fond de U-IP.
    * Ne modifiez pas au hasard.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/network.webp').default} size="100%" align="left" />

### Paramètres d'affichage
    * Peut définir la direction de l'affichage, la luminosité du rétroéclairage et si l'adresse IPv6 de la machine connectée doit être affichée, etc.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/display.webp').default} size="100%" align="left" />

### Contrôle du système
    * Peut définir la version du firmware, etc., et peut redémarrer l'appareil ou entrer en mode de gravure de U-IP.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

## Configuration de connexion du système

### Configuration Moonraker
    * Premièrement, activer ou non la connexion Moonraker, activé par défaut.
    * Deuxièmement, l'adresse IP attribuée automatiquement à la machine.
    * Troisièmement, le numéro de port Moonraker, par défaut 7125.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/mmoonraker.webp').default} size="100%" align="left" />

### Configuration SSH
    * Pour les systèmes spéciaux, vous pouvez essayer cette méthode, mais notez que ce n'est pas applicable à tous les appareils.
    * Après avoir activé la connexion SSH, entrez le compte et le mot de passe SSH de la machine, le numéro de port par défaut est 22.
    * Cette option se connecte via SSH à la machine et obtient l'IP et les ressources du système. Si cela échoue, elle restera en boucle sur l'écran de démarrage.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ssh.webp').default} size="100%" align="left" />
