# Notice sur la programmation de la mise à jour du firmware
    
    * La compilation et la programmation du firmware nécessitent l’utilisation d’un ordinateur de bord
    * La version V3 de SHT36 peut être connectée à un ordinateur de bord via deux méthodes : RS232 ou CAN
    * Pour les connexions RS232 et CAN, il est nécessaire d’utiliser un câble de données `XT30(2+2)-F`
    * Voici l’apparence du câble de données `XT30(2+2)-F` ci-dessous
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Informations importantes
* La mise à jour du firmware de la carte mère doit passer par Katapult pour programmer Klipper
* Si vous êtes certain que la carte mère a déjà été programmée avec Katapult, il n’est pas nécessaire de procéder à la dernière étape de programmation du firmware Katapult, cela ne serait qu’une mesure préventive en cas d’échec de la programmation précédente
* Il est possible de vérifier rapidement si la carte mère entre en mode Katapult en appuyant deux fois sur le bouton de réinitialisation ; si une LED clignote, cela signifie que la carte mère est en mode Katapult
:::

## Comment déterminer quel type de firmware vous avez besoin

    * La version V3 de SHT36 peut être connectée à un ordinateur de bord de la manière suivante
    * <Button variant="contained" disableElevation href="./can">Firmware CAN</Button>
        * Avantage : Si le firmware Katapult est installé, il est possible de mettre à jour le firmware sans toucher la carte mère, et il prend en charge les connexions en bus
        * Inconvénient : L'utilisation conjointe de KlipperScreen peut entraîner des erreurs telles que la déconnexion lors du retour à zéro
        * Astuce : Si vous souhaitez utiliser CAN, assurez-vous que l’ordinateur de bord supporte CAN, et il doit être utilisé avec UTOC ou une carte mère avec un firmware bridge installé
    * <Button variant="contained" disableElevation href="./rs232">Firmware RS232</Button>
        * Avantage : Pas besoin de logiciel spécial pour l’utiliser, communication fiable et stable
        * Inconvénient : Pour réinstaller le firmware, il faut soit entrer en mode DFU manuel, soit appuyer deux fois sur le bouton RST pour entrer dans Katapult et mettre à jour le firmware
        * Astuce : Il doit être utilisé avec G2T ou UTOR
