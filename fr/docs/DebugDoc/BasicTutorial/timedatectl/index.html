Modifier le fuseau horaire du système

Veuillez noter que si vous utilisez la version 3.3.4 de FLY OS sur le logiciel FLY, vous pouvez directement utiliser FLY-Tools pour modifier le fuseau horaire. Pour les autres logiciels, veuillez consulter les opérations ci-dessous.

sudo timedatectl set-timezone <TIMEZONE>

Remplacez <TIMEZONE> par le code de fuseau horaire que vous souhaitez utiliser. Vous pouvez trouver les codes de fuseau horaire sur le site iana.org. Par exemple, si vous souhaitez changer le fuseau horaire du système pour Beijing (Asie/Shanghai), vous pouvez entrer la commande suivante :

sudo timedatectl set-timezone Asia/Shanghai

Maintenant, le fuseau horaire de votre système a été modifié avec succès à l'heure de Shanghai. Vous pouvez vérifier les paramètres de fuseau horaire avec la commande suivante :

date
