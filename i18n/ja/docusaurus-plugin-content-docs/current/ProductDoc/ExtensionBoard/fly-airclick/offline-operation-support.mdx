# 有線接続 Klipper 設定

:::danger 注意事項

* STM32F072 にファームウェアを書き込んでから klipper に接続する必要があります。そうしないと有線接続操作はサポートされません。

:::

## コンパイルを開始する

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押し、**Save configuration** が表示されたら、`Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインインターフェースに戻ったはずです
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 書き込みモードに入る

    * AirClick の電源を切り、STM32 の BOOT ボタンを押しながら、STM32 の書き込みポートを Typec で上位機と接続します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/boot.webp').default} size="100%" align="center" />

## 書き込みモードに入ったか確認する
