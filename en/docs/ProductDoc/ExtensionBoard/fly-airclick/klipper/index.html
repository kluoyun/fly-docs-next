Connect to Klipper

Important Note
  • AirClick must be connected to the same network as the target printer
  1. Click on System Settings on the left

  1. Select Klipper for Printer Type

  2. Enter the IP address of the Klipper machine for Moonraker IP

  3. Enter the port of the Klipper host computer for Moonraker Port, the default is 7125, if the host computer uses a different port, you need to enter the corresponding port manually

  4. Ignore Apikey, it is not supported yet

  5. After configuring as shown above, click Submit in the lower right corner to save the configuration

  6. A prompt will appear at the top of the web page if the connection is successful

  7. If the connection fails

    • Check if the IP address and port are correct
    • Check if the printer and AirClick are connected to the same network
    • Check if the Klipper host computer is running normally
    • Check if the browser can open the web page of the Klipper host computer
