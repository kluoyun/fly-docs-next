Raspberry Piイメージ

手記

SDカードの注意事項:

  • 容量は16GB以上128GB以下のTFカードでなければならない
  • TFカードの転送速度はclass10以上でなければならない。サンディスクなどのブランドのTFカードの使用を推奨
注意
  • サーバーの問題により、システムやKlipperの更新を勝手に行わないでください
  • Raspberry Pi 3B/3B+/4B/CM4/5Bに対応したシステムイメージ
  • このイメージはbookwormを基に各種サービスをインストール
  • このイメージはTFTスクリーンをデフォルトでインストールおよび有効にしています。TFTスクリーンを使用しない場合は/boot/config.txtを手動で変更する必要があります
重要な注意事項
  • ダウンロードしたシステムは解凍する必要があります、直接拡張子を変更したり直接書き込んだりしないでください！！！
  • 解凍せずに書き込むと起動できません！！！
