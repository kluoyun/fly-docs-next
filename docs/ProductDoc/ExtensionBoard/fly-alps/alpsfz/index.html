如何调整模块触发值

打开调平模块​

Klipper控制台发送SET_PIN PIN=_probe_ready VALUE=1，观察触发指示灯，调整阈值由小到大至指示灯不亮。输入下面命令：

SET_PIN PIN=_probe_ready VALUE=1

调整压力值​

串口工具助手获取
  • 使用usb数据线连接模块跟电脑，打开串口工具助手，选择对应的端口号，波特率115200，点击连接。

串口工具输入下面命令调整触发值：

set 触发值

默认触发值：20000 数值越小越灵敏，数值越大越不灵敏。

  • 例如：set 20000，设置触发值为20000，数值越小越灵敏，数值越大越不灵敏。

  • 串口工具使用命令:

  • 串口发送v,实时输出abc值。a:滤波前；b:滤波后。

  • 串口发送uv，关闭输出。

  • 串口发送 rt,查看当前阈值。
