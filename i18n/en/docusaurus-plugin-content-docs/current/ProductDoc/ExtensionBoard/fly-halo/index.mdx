# Product Introduction

    * The main control uses ESP32-S3R8, which has better performance than ESP32. It has 8MB PSRAM and 16MB Flash. The screen is an IPS screen with high resolution, brightness, wide viewing angle, and delicate colors. It supports capacitive touch.

## Product Features

    :::tip

    * Currently, there are 6 pages, swipe left, right, up, and down to switch between them.

    :::

    * Home (displays standby, printing, halted, error statuses)

    * Extrusion (controls extrusion, retraction, extrusion temperature)

    * System Services (controls the startup, shutdown, and restart of services such as host, Klipper, Moonraker)

    * Settings (used for setting various parameters of halo, WiFi, IP, language, backlight brightness, etc.)

    * Device Information (displays information about halo, firmware version, check for updates, etc.)

    * Filament Detection: filament runout and blockage detection, filament length detection, after a specified length of blockage, it can automatically pause and raise GPIO47 (all these are disabled by default, can be enabled in settings)

    * Print Completion Shutdown: supports setting a delay time after print completion to shut down the host and raise GPIO48 (all these are disabled by default, can be enabled in settings)
