# 펌웨어 업데이트
    :::danger 주의사항

    * 사용하려면 WINDOWS 시스템이 필요합니다.
    * 연결 방법은 아래 두 가지 방법 중 하나만 사용할 수 있으며, 다른 연결 방식은 없습니다!!!

    :::
## 펌웨어 다운로드

    <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/usblcd/fly-uip.uf2/version/52147655/files">펌웨어 다운로드 주소</Button>
    * 파일 목록 클릭
    * 아래 스크롤 바를 왼쪽 마우스로 길게 누른 후 오른쪽으로 드래그
    * 다운로드 버튼 클릭
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/firmware.webp').default} size="100%" align="left" />


## 업데이트 방법
    * U-IP를 컴퓨터에 연결한 후 브라우저 주소 표시줄에 ``192.168.4.1`` 입력
    * 시스템 제어 찾기 > 재부팅하여 부트로더 모드로 전환
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

### 펌웨어 업데이트
    
    * 부트로더 모드로 재부팅하면 컴퓨터에 `RPI-RP2` 디스크가 나타납니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf2.webp').default} size="100%" align="center" />
    * 업데이트할 펌웨어를 복사
    * 참고
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf21.webp').default} size="100%" align="center" />
    * 펌웨어를 복사한 후 약 ``10초`` 후에 `RPI-RP2` 디스크가 자동으로 사라지며, 이때 펌웨어 업데이트가 완료되었습니다!
