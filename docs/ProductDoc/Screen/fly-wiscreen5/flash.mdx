# 开机图片修改
:::tip
    请严格按着`分辨率`以及`文件名`制作图片，否则系统将不会识别

    将制作好的图片，放入到SD卡/U盘根目录，在屏幕开机状态下，插入屏幕，点击升级即可
:::
## 开机LOGO修改

    * 图片要求`800*480`
    * 名字要求`boot_logo.JPG`
    * 不得大于`105KB`




## 第一界面图片修改

    * 图片要求`500*400`
    * 名字要求`print.png`
    * 不得大于`45KB`



## 打印界面图片修改

    * 图片要求`311*480`
    * 名字要求`print_smail.png`
    * 不得大于`45KB`
