상위기기 SSH 연결

  • 시리얼 포트 또는 WIFI를 사용하여 장치에 연결할 수 있습니다.
  • 미리 MobaXterm_Personal 또는 기타 SSH 터미널 도구를 준비해 두세요.
  • SD 카드 또는 M2WE에 상위기기의 시스템이 번들되어 있는지 확인하세요.
  • 만약 상위기기가 정상적으로 시작되지 않으면, 드라이버, 제한, 팬 등 외부 장치들을 제거하세요!!
  • 마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 컴파일하기 쉽게 해주세요!!
  • 펌웨어를 컴파일하려면, WIFI 또는 유선 네트워크를 통해 상위기기에 연결하는 방법을 참조하세요.
  • 시리얼 연결
  • 네트워크 연결

상위기기 SSH에 시리얼로 연결하기​

  • USB Type-c 케이블을 사용하여 FLY 상위기기를 컴퓨터와 연결합니다.
  • 다른 상위기기는 연결 방법을 찾아보세요!!
  • FLY-MiniPad
  • FLY-MiniPad 포트
  • 만약 메인보드가 컴퓨터와 정상적으로 연결되었다면, 장치 관리자에서 CH340의 포트를 볼 수 있습니다.
  • MobaXterm을 열고, 앞서 장치 관리자에서 본 포트를 선택하도록 설정합니다.
  • 모든 것이 정상이라면 시작 화면을 볼 수 있습니다.
  • 잠시 기다리면 이 화면이 나타나면 시작된 것입니다.

몇 분 동안 기다렸는데도 화면에 아무런 표시가 없다면, 엔터 키를 몇 번 누르세요.

엔터 후에는 아래와 같은 내용이 나타납니다.

  • IP 주소를 확인하려면, SSH에서 ip a | grep inet을 입력하고 엔터를 누릅니다. 반환된 내용 중 라우터와 같은 세그먼트의 IP를 찾으세요.

상위기기 SSH에 네트워크 IP로 연결하기​

  • 상위기기가 연결된 라우터 후방에 로그인하세요.
  • 시스템이 정상적으로 시작되었는지 확인하세요.
  • 자신이 상위기기의 사용자 이름과 비밀번호를 알고 있는지 확인하세요. FLY 시스템의 기본 사용자 이름은 fly, 기본 비밀번호는 mellow입니다.
  • 메인보드가 시작되고 네트워크에 연결되어 있다면, IP를 통해 SSH에 연결할 수 있습니다.
  • MobaXterm을 열고, 앞서 본 IP를 입력하세요. 또는 라우터 후방에서 확인할 수도 있습니다.

주의: 여기서 4단계에 입력되는 상위기기의 사용자 이름

  • 첫 연결에서는 이 창이 나타납니다. Accept를 클릭하면 됩니다.
  • 비밀번호를 입력하라는 메시지가 표시됩니다. 상위기기의 비밀번호를 입력하고 엔터를 누릅니다.

주의: 여기서 입력하는 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르세요.

FLY 시스템의 기본, 사용자 이름은 fly, 비밀번호는 mellow입니다.
