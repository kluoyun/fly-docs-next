# Введение в систему и загрузка

## Описание

* FlyOS-Fast — это облегченная система на базе Linux, специально адаптированная для 3D-печати и поддерживающая все платформы верхнего уровня Fly.
* FlyOS-Fast облегченная и быстро запускается.
* Ядро оптимизировано для обеспечения реальности Klipper.
* Оптимизация файловой системы снижает риск невозможности запуска после внезапного отключения питания.
* Быстрый старт — Klipperscreen запускается за 30 секунд.
* Предустановлены службы Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen и другие.
* Поддерживается переключение между Klipper и RRF.
* Поддерживается онлайн-переключение между Mainsail и Fluidd.
* Поддерживает печать с USB-накопителя.
* Поддерживает продолжение печати после отключения питания.
* Поддерживает экраны на основе последовательного порта.
* Поддерживает автопрограммирование прошивки при старте (fly-c8, fly-geminiv3).
* Поддерживает предварительную установку языка по умолчанию для интерфейсов Fluidd и других, а также имя принтера по умолчанию.
* Для дополнительных функций OEM-поставки, таких как логотип при старте, предустановленные конфигурационные файлы, скрытие вкладок Mainsail и т. д., свяжитесь с нами.

## Описание имени образа системы

* Правило именования файлов системы: `FlyOS-Fast-[серия_чипов]-[номер_версии].img`, например: `FlyOS-Fast-H5-v1.0.0.img`.
* Исходный файл имеет формат `.img`, в ссылках на скачивание доступны только сжатые файлы в форматах `.exe` и `.7z` для удобства загрузки.
* В системах Windows скачивайте `.exe`-файл, который является самораспаковывающимся архивом 7z. После скачивания двойной клик на файле распакует его, и вы получите файл `.img` (можно использовать только в Windows).
* В других системах, таких как MacOS или Linux, скачивайте файл `.7z`, который является архивом 7z. После скачивания используйте инструменты, поддерживающие формат 7z, для распаковки, чтобы получить файл `.img`.
* Из-за большого размера системных файлов временно не предоставляются другие форматы сжатия и исходные образы.

:::tip Выбор системы

* fly-C5, продукты Fly с чипом H618, выбирайте `FlyOS-Fast-H618.exe` или `FlyOS-Fast-H618.7z`.
* Серии fly-pi, fly-c8, fly-gemini и другие продукты Fly с чипом H5, выбирайте `FlyOS-Fast-H5.exe` или `FlyOS-Fast-H5.7z`.
* Серии fly-pi-lite2, fly-miniPad и другие продукты Fly с чипом H3, выбирайте `FlyOS-Fast-H3.exe` или `FlyOS-Fast-H3.7z`.
* Модели чипов различаются, поэтому файлы системы тоже разные. Пожалуйста, выбирайте в зависимости от конкретной модели.

:::

## Скачивание образа системы

* Официальная ссылка для скачивания FlyOS-Fast: [Библиотека制品 FlyOS-Fast](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/packages)

- Имя пользователя по умолчанию: `root`
- Пароль по умолчанию: `mellow`

:::tip Примечание

Это легковесная кастомная система. Компоненты и службы внутри, такие как klipper, не являются официальной версией и не поддерживают пользовательские изменения и обновления.

:::

## Процесс скачивания образа системы

* Процесс скачивания образа FlyOS-Fast:

    1. Откройте браузер и перейдите по указанной выше ссылке на библиотеку制品 FlyOS-Fast.

    2. Выберите нужный формат сжатия и перейдите на страницу файла.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down1.webp').default} size="50%" align="center" />

    3. Выберите версию (по умолчанию последняя версия).

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down2.webp').default} size="50%" align="center" />

    4. Если кнопка скачивания не видна, переместите курсор мыши в поле со списком файлов, зажмите клавишу `Shift` и прокрутите колесико мыши, чтобы пролистать список файлов.
    5. Также можно нажать левую кнопку мыши на ползунке прокрутки внизу и, удерживая её, потянуть вправо, чтобы увидеть кнопку скачивания.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down3.webp').default} size="50%" align="center" />

    6. Нажмите кнопку скачивания, чтобы скачать соответствующий файл системы.
