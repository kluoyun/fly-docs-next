Connecting to the Network via SSH

  • Please ensure that the system has started and is connected to the host machine via SSH using the serial port, it is recommended to use MobaXterm_Personal
  • Please ensure that the host machine has installed M2WE or 2.4G_WiFi module, if it is Lite2 or MiniPad, no module installation is required
  • Please note that 2.4G_WiFi module, Lite2, and MiniPad can only use 2.4G WiFi and cannot use 5G WiFi
  • Please note that the network cannot be set to dual-band合一 (dual-band integration) otherwise it will not connect properly
  • Please ensure that the antenna is fully installed!!!!
  1. After entering the command nmtui in the SSH software and pressing enter, you will see the interface as shown below
  1. Press enter and then select Activate a connection and press enter again
  1. Select the network you need to connect to, press enter, input the WiFi password, and then select OK to save the connection
  1. After connecting to the WIFI, press the Esc key several times on the keyboard to exit the current interface, then enter ip a and press enter

  • The IP address of the host machine in the picture is 192.168.2.136
