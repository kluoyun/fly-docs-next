상위기기 SSH 연결

  • 시리얼 포트 또는 Wi-Fi를 사용하여 장치에 연결할 수 있습니다.
  • MobaXterm_Personal 또는 다른 SSH 터미널 도구를 미리 준비하십시오.
  • SD 카드나 M2WE가 상위기기에 맞는 시스템으로 번들되었는지 확인하십시오.
  • 상위기가 정상적으로 시작되지 않는 경우 드라이버, 리미터, 팬 등 외부 장치를 제거하십시오!
주의
  • 첫 번째로 상위기를 시작하는 경우 반드시 시리얼 포트를 통해 상위기기 SSH에 연결하십시오.
  • 펌웨어를 컴파일할 때는 반드시 Wi-Fi를 통해 상위기기 SSH에 연결하십시오. 이를 통해 펌웨어 컴파일 및 번들이 더 용이합니다!
  • FLYOS-Armbian 시스템과 FLYOS-FAST 시스템은 SSH 연결 방법이 다릅니다. 각각의 시스템에 맞는 가이드를 선택하십시오!!
  • FLYOS-Armbian 시스템 시리얼 포트 연결
  • FLYOS-FAST 시스템 시리얼 포트 연결

  • FLYOS-FAST 시스템 Wi-Fi 연결

FLYOS-Armbian 시리얼 포트를 통한 상위기기 SSH 연결​

  • USB Type-C 케이블을 사용하여 FLY 상위기기를 컴퓨터와 연결합니다.

  • 다른 상위기기는 연결 방법을 개별적으로 확인하십시오!!

  • FLY-Pi v2 포트

  • 메인보드와 컴퓨터가 정상적으로 연결된 경우, 장치 관리자에서 CH340 포트를 볼 수 있습니다.

  • MobaXterm을 열고, 앞에서 장치 관리자에서 본 포트를 선택하도록 설정합니다.
  • 문제가 없다면 부팅 화면을 볼 수 있습니다.
  • 잠시 기다리면 이 화면이 나타나면 부팅이 완료된 것입니다.
  • 몇 분 정도 기다렸는데도 화면에 아무런 표시가 없으면, 엔터 키를 몇 번 누르세요.
  • 엔터를 누른 후 아래와 같은 내용이 표시됩니다.
경고
  • 현재 root 계정으로 로그인 중입니다. 다음 명령어를 입력하여 일반 사용자로 전환하세요: su fly. 이후 작업을 진행할 수 있습니다!!!
  • IP를 확인하려면 SSH에서 다음 명령어를 입력하세요: ip a | grep inet. 반환된 내용에서 라우터와 동일한 세그먼트의 IP를 찾아 접속할 수 있습니다.

FLYOS-FAST 시스템 시리얼 포트를 통한 상위기기 SSH 연결​

  • USB Type-C 케이블을 사용하여 FLY 상위기기를 컴퓨터와 연결합니다.

  • 다른 상위기기는 연결 방법을 개별적으로 확인하십시오!!

  • FLY-Pi v2 포트

  • 메인보드와 컴퓨터가 정상적으로 연결된 경우, 장치 관리자에서 CH340 포트를 볼 수 있습니다.

  • MobaXterm을 열고, 앞에서 장치 관리자에서 본 포트를 선택하도록 설정합니다.
  • 문제가 없다면 부팅 화면을 볼 수 있습니다.
  • FLYOS-FAST는 부팅 속도가 빠르므로, 대체로 아래와 같이 깨진 문자열이 보이는 것이 정상적인 현상입니다. 걱정하지 마세요, 몇 번 엔터를 누르면 됩니다.
  • 엔터를 누른 후 아래와 같은 내용이 표시됩니다.
  • 계정을 입력하세요: root, 엔터, 비밀번호를 입력하세요: mellow, 엔터. 비밀번호는 표시되지 않으며, 이는 정상적인 현상입니다.
주의
  • FLYOS-FAST 시스템 계정은 root이고, 비밀번호는 mellow입니다.
  • FLYOS-FAST 시스템에는 유일한 계정이 root입니다!!
  • IP를 확인하려면 SSH에서 다음 명령어를 입력하세요: ip a | grep inet. 반환된 내용에서 라우터와 동일한 세그먼트의 IP를 찾아 접속할 수 있습니다.

FLYOS-Armbian Wi-Fi를 통한 상위기기 SSH 연결​

  • 상위기기 시스템이 이미 정상적으로 시작되었는지 확인하십시오.

  • 상위기기의 사용자 이름과 비밀번호를 명확히 알아야 합니다. FLY 시스템 기본 사용자는 fly, 기본 비밀번호는 mellow입니다.

  • Wi-Fi를 통해 상위기기 SSH에 연결하는 전제 조건은 상위기가 라우터에 연결되어 있으며, 상위기기의 IP를 알고 있는 것입니다.

  • MobaXterm을 열고, 아래 그림과 같이 설정하세요. Remote host에는 시리얼 연결 시 확인한 상위기기 IP를 입력하거나, 라우터 관리자 페이지에서 상위기기의 IP를 확인할 수 있습니다.

주의: 여기서 4단계에 상위기기의 사용자를 입력해야 합니다

팁
  • 4단계에서 상위기기의 사용자를 입력합니다. FLY 상위기기이며 FLYOS-Armbian 시스템을 사용하는 경우 fly를 입력하고, 다른 상위기기의 경우 실제 상황에 맞게 올바른 사용자를 입력하십시오.
  • 첫 연결 시 아래 창이 나타날 수 있습니다. Accept를 클릭하면 됩니다.
  • 비밀번호를 입력하라는 메시지가 나타나면 상위기기 비밀번호를 입력한 후 엔터를 누릅니다.
팁
  • 여기에서 입력하는 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르세요.

FLYOS-Armbian 시스템 기본 사용자는: fly, 비밀번호는: mellow

  • 아래와 같은 창이 나타나면, 이는 사용자 비밀번호를 저장할지 묻는 창입니다. No를 클릭하세요.
  • 아래와 같은 화면이 나타나면 로그인이 성공했습니다!
