# FLY U-IP
    本製品は、ラズベリーパイRP2040マイコンをベースにしたマルチファンクション状態モニタモジュールで、Type-AオスコネクタとType-Cメスポートを統合しており、プラグアンドプレイ（Windows/対応）をサポートしています。本製品は、ラズベリーパイRP2040マイコンをベースにした埋め込み型状態モニタモジュールで、1.14インチの全視野IPSディスプレイ（解像度240x135）、Type-AオスコネクタとType-Cメスポートを搭載し、プラグアンドプレイ（Windows/Linuxドライバ不要）をサポートしています。3Dプリンター、IoTデバイス、ネットワークデバッグシナリオ向けに設計され、画面にデバイスのネットワーク情報、システムリソースの使用率、サービスの実行状態を直接表示し、追加の端末ツールが不要で、直感的で便利な操作を提供します。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/uip.webp').default} size="100%" align="left" />

## 特徴 

    * ラズベリーパイRP2040コントローラを使用、クロック周波数は200MHz
    * USB Type-AオスポートとUSB Type-Cメスポートをサポート
    * 1.14インチLCD全視野IPSディスプレイを搭載、解像度は240x135
    * Windowsおよび一般的なLinuxディストリビューションでのドライバ不要使用
    * Moonraker経由でIPアドレスなどの情報を取得可能
    * SSH経由でIPアドレスなどの情報を取得可能
    * コンピュータからウェブページを通じてパラメーターを変更可能
    * スクリーンテーマカラーをカスタマイズ可能
    * スクリーン表示方向を90°と270°に回転可能
    * スクリーンバックライトの明るさを調整可能
    * 全ての表示情報の更新時間と表示時間をカスタマイズ可能
    * IPV6アドレスの表示をオン/オフ可能
    * RSTボタンを3回押して工場出荷時の設定に戻す機能あり

## 注意事項

    :::warning 注意事項

    * 本製品は当面Appleデバイスには対応していません
    * ECMまたはRndisのサポートが必要です。それがない場合、正常に使用できません
    * 本製品はKlipper専用に設計されており、Moonrakerサービスとの連携が必要です。IPやシステムリソースを表示するにはMoonrakerが必要です。それ以外の場合、バックグラウンドを起動してSSHモードを有効にする必要があります
    
    :::
