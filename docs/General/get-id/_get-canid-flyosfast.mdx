## 填写CAN ID

    1. 关闭`DEVICES`填写ID
    :::danger 须知
    * 请将配置文件中的:  
    ```
    [mcu] 
    serial: /tmp/klipper_host_mcu
    ```  
    修改为
    ```
    [mcu host] 
    serial: /tmp/klipper_host_mcu
    ```
    * 添加：
    ```
    [mcu]
    canbus_uuid: <这里替换为你刚刚查询到的ID>
    ```
    :::
