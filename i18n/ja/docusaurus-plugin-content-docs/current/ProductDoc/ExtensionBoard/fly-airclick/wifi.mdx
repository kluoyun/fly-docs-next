# WiFiに接続

    :::info 重要なヒント

    * 本製品のオンボードアンテナは受信にのみ使用でき、送信アンテナは自分で取り付ける必要があります。アンテナ端子は`ipex3`です
    * アンテナを正しく取り付けてください！！！そうしないと、信号が弱すぎて発熱がひどくなります！！！
    * 初めて使用する場合は、AirClickホットスポットに手動で接続した後、対応するWIFIに接続します

    :::


## AirClickでWIFIに接続

    * AirClickのホットスポットに接続します。ホットスポットの名前は`FLY-WBTN-XXXXXX`です
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
    * ホットスポットに接続後、ブラウザで[http://192.168.4.1/](http://192.168.4.1/#/setting)にアクセスします
    * この時、WIFI設定画面が表示されます。プリンタと同じルーターに手動で接続し、パスワードを入力して下の青いボックスをクリックしてWIFIに接続します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
    * WiFiに接続後、AirClickがWiFiに接続した後のIPアドレスが表示されます。下の青いボックスをクリックしてデバイスを再起動できます
        * 番号1で囲まれたのはAirClickのIPアドレスです
        * 番号2で囲まれたIPをコピーしてデバイスを再起動します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
    * 現在のネットワークを対応するルーターに手動で切り替える必要があります。そうしないとアクセスできません
    * 接続に成功した後、AirClickの電源を切って再起動することをお勧めします
