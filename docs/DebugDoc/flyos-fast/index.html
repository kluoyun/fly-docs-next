系统介绍和下载

简介​

  • FlyOS-Fast 是一个轻量化 Linux 系统，为3D打印深度定制，支持 Fly 全系列上位机平台。
  • FlyOS-Fast 轻量化，启动速度快。
  • 内核优化为Klipper实时性提供保障。
  • 文件系统优化减少直接断电导致无法开机的风险
  • 快速启动，30秒启动Klipperscreen
  • 预装了Klipper，RRF，Mainsail，Fluidd，Moonraker，KlipperScreen等服务。
  • 支持Kliiper和RRF系统切换
  • 支持mainsail和fluidd在线一键切换
  • 支持U盘打印
  • 支持断电续打
  • 支持串口屏幕
  • 支持开机自动烧录固件(fly-c8,fly-geminiv3)
  • 支持提前设置fluidd等前端UI的默认语言，默认打印机名称
  • 更多OEM定制功能：如开机LOGO,预设置配置文件，设置隐藏Mainsail标签页等功能请联系我们。

系统镜像文件名介绍​

  • 系统文件命名规则：FlyOS-Fast-[芯片系列]-[版本号].img，例如：FlyOS-Fast-H5-v1.0.0.img。
  • 源文件为.img格式，下载链接中只提供.exe和.7z格式的压缩文件，方便下载。
  • Windows系统下载.exe文件，此文件为7z自解压压缩包，下载完成后双击打开解压后即可得到.img文件（只能使用Windows系统解压）。
  • 其它系统如MacOS，Linux系统下载.7z文件，此文件为7z压缩包，下载完成后使用支持7z格式的工具解压后即可得到.img文件。
  • 因系统文件较大，暂时不提供其它压缩格式及源镜像的下载。
系统选择
  • fly-pi系列，fly-c8系列，fly-gemini系列等，所有使用全志H5芯片的Fly系列产品，选择FlyOS-Fast-H5.exe 或 FlyOS-Fast-H5.7z。
  • fly-pi-lite2系列，fly-miniPad系列等，所有使用全志H3芯片的Fly系列产品，选择FlyOS-Fast-H3.exe 或 FlyOS-Fast-H3.7z。
  • 二者的芯片型号不同，系统文件也不同，请根据实际情况选择。

系统镜像下载​

  • FlyOS-Fast 官方下载地址：FlyOS-Fast制品库
  • 默认用户名：root
  • 默认密码：mellow
须知

此系统为轻量化定制系统，内置的klipper等组件及服务非官方原版，不支持用户自行修改，升级。

系统镜像下载流程​

  • FlyOS-Fast 镜像下载流程：

    1. 浏览器访问上面的FlyOS-Fast制品库

    2. 选择需要的压缩格式，点击进入文件页面

    1. 选择版本（默认最新版本）
    1. 如果没有看到下载按钮，请将鼠标移动到文件列表里面，按住Shift键，然后滚动鼠标滚轮，即可将文件列表左右滚动。
    2. 也可以将鼠标放在下方滚动条上，左键长按然后向右拖动，即可看到下载按钮。
    1. 点击下载按钮，即可下载对应版本的系统文件
