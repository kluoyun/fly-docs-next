# 主板注意事项

## 电源指示灯

        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/img/power.png').default} size="80%" align="center" />
    * 只通过TYPE-C供电并不接其他主板电源时候至少亮两个灯`3.3V`与`5V`
    * 在接入主板电源后主板电源灯应该亮`3.3V`、`5V`、`12V`
    * 驱动电源0-2、驱动电源3-7、热床电源接入后，主板左下方电源指示灯才会正常亮起来

### 问题排查


    :::danger 须知

    * 在没有任何外接设备情况下（驱动、限位、12864等）
    * 只通过TYPE-C供电至少会常亮`3.3V`与`5V`两个LED

    :::

    * 如果`3.3V`与`5V`不亮，请检查主板5V跳线
    * 更换跳线后还是一样，并且确认typec正常供电，需要联系客服售后
    * 如果`3.3V`、`5V`与`12V`中有一个灯闪烁，代表有芯片短路，需要联系客服售后
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/img/5v.png').default} size="60%" align="center" />

## 驱动

    * 请注意驱动电源，驱动电源分两路，第一路给驱动3-7供电，第二路0-2
    * 主板支持外置驱动     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">接线与配置教程</Button>

## 限位口

    * 除了最后一个驱动口（Drive7）带无限位跳线，其他驱动口的无限位都是直连限位口
    * 如果发现限位口不能使用可以更换限位口或者剪去驱动的无限位脚
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/2209diag.png').default} size="80%" align="center" />




## 固件烧录

    * 主板使用TF卡进行烧录
    * 之前因教程问题使用DFU进行烧录，如果需要更新固件需要 <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/flash/bl">重新刷BL</Button>
