# USB Firmware Compilation


## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press the `Q` key, then you will see **Save configuration**, press `Y` key now
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * You should have saved the configuration and exited to the command line interface now
    * Enter `make -j4` to start compiling, it may take a while
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * The compilation is successful when the following output appears

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
