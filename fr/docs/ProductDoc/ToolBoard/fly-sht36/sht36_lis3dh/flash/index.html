  • Instructions pour la gravure du firmware
prompt
  • Le SHT36 LIS3DH est préchargé avec le firmware Katapult pour la connexion CAN, à une vitesse de 1M
  • Le firmware Katapult pour la connexion CAN du SHT36 LIS3DH est disponible à 500K et 1M, vous pouvez le graver vous-même si nécessaire
  • Gravure du firmware Katapult

Instructions pour la gravure du firmware

  • Le SHT36 LIS3DH ne peut se connecter à la machine hôte qu'en utilisant la méthode de connexion CAN
  • La CAN doit être connectée en utilisant XT30(2+2)-F
  • Ci-dessous se trouve le style du câble de données XT30(2+2)-F
À savoir
  • La gravure du firmware de la carte mère nécessite l'utilisation de Katapult pour graver Klipper
  • Si vous êtes certain que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin d'effectuer la dernière étape de gravure du firmware Katapult, cela n'est qu'une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été gravé avec succès
  • Vous pouvez appuyer rapidement deux fois sur la réinitialisation pour voir si un LED sur la carte mère clignote, si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult

Comment déterminer quel firmware vous avez besoin​

  • Le SHT36 LIS3DH peut se connecter à la machine hôte de la manière suivante
  • Gravure du firmware CAN
    • Avantages : Si le firmware Katapult est installé, vous pouvez mettre à jour le firmware sans toucher à la carte mère, et il supporte la connexion par bus
    • Inconvénients : Peut causer des erreurs comme le dépassement du délai de retour à l'origine lorsqu'il est utilisé avec KlipperScreen.
    • Conseils : Si vous avez besoin d'utiliser CAN, assurez-vous que la machine hôte supporte CAN, et vous aurez besoin de l'UTOC ou d'une carte mère avec un firmware de pont installé

Explication du firmware Katapult​

prompt

Le nom original de Katapult était CanBoot, puis il a été changé.

  • Le firmware Katapult est un type de bootloader qui peut être utilisé avec Klipper. Après avoir été flashé via DFU ou une autre méthode, il permet la mise à jour du firmware d'un simple clic de commande.
  • Avantages : Double-cliquez sur le bouton de réinitialisation ou coupez l'alimentation deux fois rapidement pour entrer dans le firmware Katapult, et les LED configurées à l'avance clignoteront. S'il n'y a pas de configuration, elles ne clignoteront pas.
  • Le firmware Katapult ne peut être utilisé que pour flasher le firmware et ne peut pas se connecter à Klipper !!!
  • Le firmware Katapult supporte trois méthodes pour se connecter à l'ordinateur hôte :
    • Connexion CAN
      • Avantages : Après avoir compilé le firmware KLIPPER, vous pouvez utiliser des commandes pour mettre à jour directement le firmware.
      • Inconvénients : Il faut connaître à l'avance le taux de compilation du firmware.
    • Connexion RS232
      • Avantages : Après avoir compilé le firmware KLIPPER, vous pouvez utiliser des commandes pour mettre à jour le firmware.
      • Inconvénients : Impossible d'utiliser des commandes pour mettre à jour le firmware en un clic, il faut entrer manuellement dans le firmware Katapult.
    • Connexion USB
      • Avantages : Après avoir compilé le firmware KLIPPER, vous pouvez utiliser des commandes pour mettre à jour directement le firmware.
      • Inconvénients : Une connexion trop longue peut provoquer des interférences.
