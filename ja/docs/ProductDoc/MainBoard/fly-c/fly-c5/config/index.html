システム設定の変更

必読
  • 設定ファイル内の部分の設定は、ファームウェア書き込み後に最初にシステムを起動する際にのみ有効です。
  • したがって、ファームウェア書き込み後にまず可動ディスクFlyOS-Confにある設定ファイルconfig.txtを編集してください。
  • その後、カードをボードに挿してシステムを起動してください。
注意
  • Windowsの標準のメモ帳を使用せずに、専用のテキストエディタ（例：VSCode）で設定ファイルを編集することをお勧めします。
  • 設定ファイルには日本語や特殊文字を入力しないでください。
  • 設定ファイルの変更後は、システムを再起動する必要があります。
危ない
  • 設定ファイル内の######### PLEASE DO NOT EDIT THE FOLLOWING #########以下の内容は編集しないでください。

設定ファイルの変更方法​

ファームウェア書き込み後の設定ファイルの変更​

  • 初回起動時にのみ有効となる設定項目は、ファームウェア書き込み後に、専用のテキストエディタで可動ディスクFlyOS-Conf内の設定ファイルconfig.txtを開いて変更してください。

システム起動後のオンラインでの設定変更​

  • 通常の設定項目は、システムのWeb設定ページで変更し、システムを再起動すると有効になります。

    • ブラウザでデバイスのIPアドレスにアクセスします。例：http://192.168.1.2/
    • Fluiddでは左図のように「チェックを外す」 -> 「隠しファイルとフォルダをフィルタリング」。
    • Mainsailでは右図のように「チェックを入れる」 -> 「隠しファイルを表示」。
    • これにより.flyos-configフォルダが見えます。このフォルダ内にはsys-config.confファイルがあります。
    • sys-config.confファイルは可動ディスクFlyOS-Conf内の設定ファイルconfig.txtのシンボリックリンクです。
    • sys-config.confファイルを開き、必要な設定項目を変更して「保存」 -> 「閉じる」してください。
    • 注意：sys-config.confファイルの設定形式はkey=valueであり、key: valueに変更しないでください。
    • 変更保存後、システムを再起動すると有効になります。
