## Referenzkonfiguration

    ```cfg
    [adxl345]
    cs_pin: PB12
    spi_bus: spi2 # Hardware-SPI, entweder Hardware- oder Software-SPI. Standardmäßig Hardware-SPI
    # spi_software_mosi_pin: PB15 # Software-SPI, entweder Hardware- oder Software-SPI. Standardmäßig Hardware-SPI
    # spi_software_miso_pin: PB14
    # spi_software_sclk_pin: PB13

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # In der Regel in der Mitte des Heizbetts, bitte an Ihre tatsächlichen Bedingungen anpassen
    ```
