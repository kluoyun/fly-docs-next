# FLY U-IP
    본 제품은 라즈베리 파이 RP2040 마이크로컨트롤러 기반의 다기능 상태 모니터링 모듈로, Type-A 메스 단자와 Type-C 페밀리 단자를 통합하고 있으며 즉시 사용이 가능합니다 (Windows/본 제품은 라즈베리 파이 RP2040 마이크로컨트롤러 기반의 임베디드 상태 모니터링 모듈로, 1.14인치 전방향 IPS 스크린(240x135 해상도)을 탑재하고 있으며 Type-A 메스 단자와 Type-C 피날리를 통합하고 있습니다. Windows/Linux 드라이버 설치 없이 즉시 사용이 가능하며, 3D 프린팅, 사물 인터넷 장비 및 네트워크 디버깅 시나리오에 특화되어 있습니다. 장치의 네트워크 정보, 시스템 자원 사용률 및 서비스 실행 상태를 화면에서 직접 확인할 수 있으며, 별도의 터미널 도구가 필요 없습니다. 직관적이고 간편한 조작이 가능합니다.

## 특징 

    * 라즈베리 파이 RP2040 컨트롤러 사용, 200MHz 클럭 속도
    * USB Type-A 메스 단자와 USB Type-C 피날리 지원
    * 1.14인치 LCD 전방향 IPS 스크린, 240x135 해상도 탑재
    * Windows 및 일반 Linux 배포판에서 드라이버 설치 없이 사용 가능
    * Moonraker를 통해 IP 주소 등의 정보 확인 가능
    * SSH를 통해 IP 주소 등의 정보 확인 가능
    * 컴퓨터에서 웹 페이지를 통해 매개변수 수정 가능
    * 화면 테마 색상을 사용자 정의 가능
    * 화면 방향을 90° 또는 270°로 회전 가능
    * 화면 백라이트 밝기를 조절 가능
    * 모든 정보의 갱신 시간과 표시 시간을 사용자 정의 가능
    * IPV6 주소 표시 여부 선택 가능
    * 충전 리셋 버튼을 세 번 누르면 출⼚ 설정으로 복구 가능

## 주의사항

    :::warning 주의사항

    * 본 제품은 Apple 기기에서는 현재 지원되지 않습니다.
    * ECM 또는 Rndis를 지원하는 시스템이 필요하며, 그렇지 않으면 정상적으로 작동하지 않을 수 있습니다.
    * 본 제품은 Klipper에 맞춰 설계되었으며 Moonraker 서비스와 함께 사용해야 IP 주소 및 시스템 자원을 표시할 수 있습니다. 그렇지 않으면 백그라운드를 실행하고 SSH 모드를 설정해야 합니다.
    
    :::
