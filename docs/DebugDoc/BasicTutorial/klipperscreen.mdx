# KlipperScreen扩展功能

## 打开KlipperScreen鼠标光标的方法

    * 打开打印机配置中的`KlipperScreen.conf`,此文件是安装好KlipperScreen后自动生成的。如果没有请自己创建
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.png').default} size="100%" align="center" />
    * 将下方代码添加到配置最上方
    ```cfg
    [main]
    show_cursor: True
    ```
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf-1.png').default} size="80%" align="center" />
    * 保存重启即可

## 远程连接其他上位机

    * 打开打印机配置中的`KlipperScreen.conf`,此文件是安装好KlipperScreen后自动生成的。如果没有请自己创建
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.png').default} size="100%" align="center" />
    * 将下方代码添加到配置最上方
    * 可以修改`VORON-300`打印机名称与`192.168.6.217`需要连接机器的IP地址。
    * 请注意第二个`[printer KlipperScreen]`是为了连接远程扩展的机器，可以连接此配置进行调试或者关机
    :::danger 须知

    * 请保证在一个局域网下

    :::

    ```cfg
    # Define printer and name. Name is anything after the first printer word
    [printer VORON-300]
    # Define the moonraker host/port if different from 127.0.0.1 and 7125
    moonraker_host: 192.168.6.217
    # ports 443 and 7130 will use https/wss
    moonraker_port: 7125
    # Moonraker API key if this is not connecting from a trusted client IP
    moonraker_api_key: False

    # Define printer and name. Name is anything after the first printer word
    [printer KlipperScreen]
    # Define the moonraker host/port if different from 127.0.0.1 and 7125
    moonraker_host: 127.0.0.1
    # ports 443 and 7130 will use https/wss
    moonraker_port: 7125
    # Moonraker API key if this is not connecting from a trusted client IP
    moonraker_api_key: False

    ```
