# 驱动欠压解决方法

## 方法一

    :::danger

    * 更换电源
    * 添加电源滤波器
    * 更换电机判断
    * 将电机外壳接到工具板的GND上
    * 在挤出机驱动配置添加`driver_TPFD: 0`

    :::

## 方法二
    * 通过SSH连接到上位机
    * 输入下方指令
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * 更新klipper后将可能还原此步骤
    :::
