Gravure du système

Guide de l'utilisateur pour les débutants avec le logiciel hôte​

Guide de l'utilisateur pour les débutants avec le logiciel hôte

Préparation avant la gravure du système​

  • Lecteur de carte
  • Une carte TF d'une capacité d'au moins 16 Go ou ne dépassant pas 128 Go, la vitesse de transfert de la carte TF doit être de classe 10 ou supérieure
  • Téléchargez l'image du système correspondant
  • Téléchargez l'outil de gravure nécessaire

Gravure du système​

Information

La gravure sur carte TF et la gravure sur M2WE via une carte TF ne nécessitent qu'une seule méthode de gravure

Information
  • Si la machine est équipée d'une M2WE et d'une carte mémoire, le système démarre par défaut sur la carte mémoire
  • Il est recommandé d'utiliser une carte TF pour flasher le système sur M2WE. L'utilisation de l'USB pour graver sur M2W peut échouer en raison de la compatibilité des pilotes. Il est recommandé d'utiliser directement la gravure sur M2WE via carte mémoire
  • Veuillez vérifier si l'image système téléchargée correspond, sinon elle ne pourra pas démarrer
  • Gravure sur carte TF
  • Gravure sur M2WE via carte TF
  • Gravure sur M2WE via USB
  • Gravure sur M2WE ou carte TF via M2SD Burner

Gravure du système sur une carte TF

Préparation​

  • Installez d'abord le logiciel Win32DiskImager
  • Décompressez le système, après décompression le nom de fichier se terminera par .img
  • Il n'est pas recommandé d'utiliser balenaetcher pour la gravure du système
  • Insérez la carte TF dans un lecteur de cartes, puis connectez le lecteur à un port USB de l'ordinateur
  • Si la carte TF contient des données, sauvegardez-les. Après avoir gravé le système, il n'est pas recommandé de retirer la carte TF sauf si vous devez graver sur M2WE, alors vous pouvez la retirer.

Formatage de la carte mémoire​

  • Utilisez le logiciel TFFormatter.exe pour formater la carte TF
  • Sélectionnez le disque que vous souhaitez formater
  • Ou utilisez la gestion des disques intégrée à Windows pour supprimer la partition, puis recréez une partition après la suppression.

Gravure sur la carte TF​

Test

  • Sélectionnez le fichier à flasher en rouge
  • Sélectionnez la carte TF à flasher en orange
  • Flashez le système sur la carte TF en bleu
  • Confirmez le flashage et attendez la fin du processus
  • Flashage terminé

Graver sur M2WE via une carte TF

Information

La migration du système permet de transférer intégralement l'image originale de la carte TF vers le M2WE. Une fois la migration terminée, le contenu du M2WE sera identique à celui de la carte TF d'origine.

  • Installez le M2WE puis mettez-le sous tension pour démarrer
  • Veuillez d'abord utiliser le port série pour vous connecter à SSH
  • Exécutez la commande ci-dessous, un mot de passe peut être demandé, entrez mellow et appuyez sur Entrée, notez que le mot de passe n'est pas visible
sudo nand-sata-install
  • Sélectionnez l'option Boot from eMMC - system on eMMC puis appuyez sur Entrée (l'option par défaut est généralement celle-ci, appuyez directement sur Entrée)
  • Lorsque This script will erase your eMMC. Continue? apparaît, sélectionnez Yes et appuyez sur Entrée (l'option par défaut est généralement celle-ci, appuyez directement sur Entrée)
  • Sélectionnez le type de système de fichiers ext4 et appuyez sur Entrée (l'option par défaut est généralement celle-ci, appuyez directement sur Entrée)
  • Attendez patiemment jusqu'à ce que la barre de progression apparaisse (si après 2 minutes la barre de progression ne s'affiche pas, appuyez sur CTRL+C pour interrompre l'opération, puis redémarrez l'appareil et réessayez)
  • Pendant le processus de migration du système, ne coupez pas l'alimentation et n'effectuez aucune autre opération, jusqu'à ce que la migration soit terminée
  • Une fois la migration terminée, appuyez directement sur Entrée, attendez que l'appareil s'éteigne complètement
  • Une fois l'appareil éteint, retirez la carte TF, puis rallumez l'appareil, vous pourrez maintenant démarrer à partir du système sur l'eMMC

Préparez la gravure USB vers M2WE

Préparation​

  • Installez d'abord le logiciel Win32DiskImager
  • Il n'est pas recommandé d'utiliser balenaetcher pour graver le système
  • Décompressez le système, le nom du fichier après décompression se termine par .img
  • Téléchargez le pilote M2WE BOOT

Installation du pilote​

  • Ouvrez le logiciel Zadig
  • À l'exception du premier Zadig qui peut être exécuté directement sans installation, les deux autres doivent être installés. Pendant l'installation, cliquez toujours sur Suivant ou Next
  • Ouvrez Zadig, dans le menu Options en haut, cochez List All Devices
prompt

Assurez-vous que l'appareil est connecté à l'ordinateur en appuyant sur la touche

  • Installez M2WE, maintenez la touche BOOT enfoncée et branchez le câble USB à double tête à l'ordinateur
  • FLY-C8
  • FLY-Pi v2
  • FLY-Pi(π)
  • FLY-Gemini
  • Port FLY-C8
  • Port FLY-Pi v2
  • Port FLY-Pi
  • Port FLY-Gemini
  • Ouvrez Zadig
  • Si tout se passe bien, un Unknow Device #x ou USB Device(VID_1f3a_PID_efe8) apparaîtra dans Zadig, sélectionnez-le
  • Après avoir sélectionné l'appareil, vérifiez que l'ID USB correspond à celui de l'image ci-dessus, sinon, sélectionnez un autre appareil
  • Dans l'image ci-dessus, à la deuxième place, vous ne pouvez choisir que WinUSB(vxxxxxxx), la version n'a pas d'importance
  • Cliquez sur Install Driver pour installer le pilote
  • Le pilote USB est installé avec succès (le pilote n'est installé qu'une fois, il n'est pas nécessaire de le réinstaller lors de futures gravures)
  • Ouvrez l'outil FLY-BOOT installé, cliquez sur Vérifier l'appareil
  • Si toutes les étapes précédentes ont été effectuées correctement, un message indiquant la découverte d'un appareil disponible apparaîtra
  • Cliquez pour démarrer BOOT, si tout va bien, un message indiquant que le BOOT a été activé avec succès apparaîtra

Formatage de M2WE​

  • Utilisez le logiciel TFFormatter.exe pour formater le disque BOOT
  • Ou utilisez la gestion de disque intégrée de Windows pour supprimer la partition, puis recréez la partition après suppression.
  • Maintenant, vous pouvez ouvrir l'explorateur de fichiers, une clé USB de 14,5 Go supplémentaire apparaîtra (la taille affichée de 16 Go eMMC est de 14,5 Go)
  • Vous pouvez maintenant graver directement l'image du système avec Win32diskimager, choisissez la lettre de lecteur de cette clé USB de 14,5 Go

Installer M2WE ou une carte SD sur M2WE​

Attention
  • Ce produit supporte USB3.0
  • Ce produit ne supporte pas l'utilisation simultanée de TF et M2WE
  • Après avoir installé M2WE ou une carte SD et connecté à un port USB de l'ordinateur, vous pouvez formater
  • Il est interdit d'installer des dispositifs NVME ou autres, sinon cela pourrait endommager l'appareil !!!

  • Insérez la carte TF dans un lecteur de carte, puis connectez le lecteur à l'interface USB de l'ordinateur
  • Si la carte TF contient des données, veuillez faire une sauvegarde. Après avoir installé le système, il n'est pas recommandé de retirer la carte TF sauf si vous devez graver sur M2WE

Configuration du système​

Utilisation de FLYOS_Env

Astuce
  • À partir du système 3.3.4, vous pouvez directement utiliser FLY-TOOLS pour configurer le système sans avoir à modifier la configuration dans le boot
  • Veuillez noter que les anciens systèmes ne supportent pas directement la configuration avec FLY-TOOLS
  • Tutoriel d'utilisation de FLY_TOOLS

Explication de la configuration WIFI​

À savoir
  • La méthode de configuration WIFI est complexe, il est nécessaire de se connecter via un port série à l'ordinateur hôte pour établir la connexion WIFI
À savoir
  • Gemini V3 doit entrer dans l'interface de configuration de FLY TOOLS pour configurer le modèle de la carte mère, sinon le firmware ne démarrera pas correctement

Installation de la carte TF ou M2WE sur le C8​

  • Installation de la carte TF sur le C8
  • Installation de la carte M2WE sur le C8
  • Après avoir gravé le système sur la carte TF, il est nécessaire de connecter la carte TF au C8 ici, sinon le système ne démarrera pas
  • Veuillez noter que si une carte TF est présente, le système démarre par défaut sur la carte TF
  • Veuillez noter que les vis de fixation doivent être fixées avec un tournevis hexagonal de 1,27 mm
