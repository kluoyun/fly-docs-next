# Инструкция по прошивке прошивки
    
    * Для компиляции и записи прошивки требуется ПК
    * Версия SHT36 V3 может подключаться к ПК через RS232 или CAN
    * Для подключения RS232 и CAN используется переходник `XT30(2+2)-F`
    * Ниже показан вид переходника `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Для прошивки основной платы необходимо использовать Katapult для записи Klipper
* Если основная плата уже прошита Katapult, то нет необходимости выполнять последующую прошивку Katapult — это просто мера на случай неудачного прошива основной платы
* Можно быстро дважды нажать кнопку сброса, чтобы проверить, загорается ли светодиод на плате. Если да, то это означает, что плата находится в режиме Katapult
:::

## Как определить, какая прошивка вам нужна

    * Версия SHT36 V3 может подключаться к ПК следующим образом
    * <Button variant="contained" disableElevation href="./can">CAN-прошивка</Button>
        * Преимущество: если установлена прошивка Katapult, прошивку можно обновлять без физического контакта с платой, а также поддерживаются подключения по шине
        * Недостаток: совместное использование с KlipperScreen может привести к ошибкам, таким как превышение времени ожидания при возврате домой
        * Примечание: если вы хотите использовать CAN, убедитесь, что ваш ПК поддерживает CAN, и используйте его с UTOC или основной платой с прошивкой шлюза
    * <Button variant="contained" disableElevation href="./rs232">RS232-прошивка</Button>
        * Преимущество: не требует специального программного обеспечения для использования, надежная и стабильная связь
        * Недостаток: для повторной прошивки требуется вход в DFU вручную или двойное нажатие кнопки RST для входа в Katapult
        * Примечание: для работы необходим G2T или UTOR
