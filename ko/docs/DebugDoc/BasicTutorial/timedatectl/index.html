시스템 시계 변경

FLY 상위기기에서 FLY OS 3.3.4 버전을 사용하고 있다면 FLY-Tools를 통해 시계를 변경할 수 있습니다. 다른 상위기기는 아래의 작업을 참고하세요.

sudo timedatectl set-timezone <TIMEZONE>

<TIMEZONE>을 원하는 시즈 코드로 바꾸세요. 시즈 코드는 iana.org 웹사이트에서 찾을 수 있습니다. 예를 들어, 시스템 시계를 베이징 시간대로 변경하려면 다음 명령어를 입력하면 됩니다:

sudo timedatectl set-timezone Asia/Shanghai

이제 시스템 시계는 상해 시간대로 성공적으로 변경되었습니다. 시계 설정을 확인하려면 다음 명령어를 입력하세요:

date
