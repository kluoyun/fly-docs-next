## 系统烧录前准备

    * 读卡器
    * TF卡一张，容量必须不小于16GB 或 不大于128G TF 卡，TF 卡的传输速度必须为 class10 级或 class10 级以上
    * 下载对应的 <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">系统镜像</Button>
    * 下载需要的 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">烧录工具</Button>

## 系统烧录

:::info
TF卡烧录与通过TF卡烧录到M2WE只需要使用其中一种烧录即可
:::

:::info
* 如果机器安装了M2WE与内存卡，系统默认从内存卡启动
* M2WE建议使用SD卡刷系统，使用通过USB烧录到M2W可能因为驱动兼容等原因烧录失败，建议直接使用[内存卡烧录到M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* 请注意下载的系统镜像是否对应，如果不对应是无法启动的
:::
