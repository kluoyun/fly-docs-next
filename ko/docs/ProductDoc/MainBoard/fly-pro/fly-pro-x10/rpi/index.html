라즈베리 파이 CM4 시스템 번인

다이얼 코드 설명​

  • 다이얼 코드 상태에 주의하세요
  • TF 카드 슬롯 사용 설명

주의 사항​

  1. USB 이외의 모든 장치는 전원이 켜진 상태에서 플러그를 뽑거나 꽂지 마십시오.
  2. Type C 인터페이스는 전원 공급과 USB SLAVE 인터페이스로 이미지를 구울 수 있습니다.
  3. CM4에 전원이 정상적으로 공급되도록 하기 위해, Type C 인터페이스로 이미지를 굽는 동안 다른 장치를 연결하지 마십시오.
  4. 모듈에는 보호 장치가 없으므로 전원을 단락시키지 마십시오.
  5. FLY 메인보드는 POE 기능을 지원하지 않습니다! POE를 사용하여 전원을 공급하지 마십시오.
주의
  • 주의: EMMC 버전은 SD 카드를 사용할 수 없습니다.
  • 주의: EMMC가 없는 버전은 MICRO SD 카드로 부팅해야 하며, 이미지를 굽기 위해 카드 리더기를 사용해야 합니다.
  • 메모리 카드는 전송 속도가 class10 등급 이상이어야 하며, 샌디스크 등의 브랜드 TF 카드 사용을 권장합니다.

라즈베리 파이에 사전 설치된 TFTV2 시스템 다운로드 링크​

  • 라즈베리 파이 3B/3B+/4B/CM4/5B에 적합한 시스템 이미지
  • 이 이미지는 klipper, moonraker, mainsail, fulidd, klipper-screen 등이 사전 설치되어 있습니다.
https://cdn.mellow.klipper.cn/IMG/Beta/RPI-TFT-IMG-new.img.xz

이미지 다운로드​

  • 최신 시스템 다운로드 여기를 클릭해서 다운로드하거나 그림과 같이 하세요.
  • 토렌트 파일을 다운로드한 경우, 썬더볼트 또는 코멧 등의 소프트웨어를 사용하여 다시 다운로드하세요.
  • 데스크탑 및 권장 소프트웨어가 포함된 이미지 (Raspberry Pi OS with desktop and recommended software)는 EMMC 16G 또는 CM4 lite 16G 메모리 카드 이상의 저장 공간을 사용하는 것이 좋습니다.
주의

EMMC 버전은 전원을 꺼야만 CM4 굽기 모드로 전환할 수 있으며, Lite 버전은 시스템 카드에 직접 이미지를 굽기만 하면 됩니다.

주의
  • EMMC 버전은 굽기 모드로 들어가야만 가능하며, 아래는 굽기 모드로 들어가는 방법입니다.
  • Lite 버전은 굽기 모드로 들어갈 필요가 없으며, 직접 포맷 후 이미지를 굽기만 하면 됩니다.

rpiboot（부트 로더） 다운로드​

  • 관리자 권한으로 rpiboot 소프트웨어 다운로드 및 실행 여기를 클릭
  • 드라이버 설치 및 시작 도구로 설치합니다. 설치가 성공하면 설치 디렉토리에 rpiboot.exe 애플리케이션이 있습니다.

굽기 모드로 들어가기​

  • 이 때 컴퓨터에 연결하고 메인보드에 전원을 공급하면, 컴퓨터 장치 관리자에서 BCMxxx 장치를 인식합니다.

  • 그 다음에 rpiboot를 실행합니다.

  • 실행이 완료될 때까지 기다리면, 내 컴퓨터에 USB 드라이브가 나타납니다.

주의

주의: 만약 컴퓨터가 BCMxxx 장치를 인식하지 못하면, 위의 단계 중 하나가 잘못되었을 수 있습니다. BCMxxx 장치를 인식해야만 rpiboot 실행이 유효하므로, 그렇지 않으면 무효합니다.

EMMC 또는 SD 카드 포맷​

  • SDFormatter.exe 여기를 클릭하여 다운로드를 사용하여 SD 카드를 포맷합니다.

  • 또는 WIN 내장 디스크 관리자를 사용하여 파티션을 삭제하세요. 파티션을 삭제한 후 다시 파티션을 생성합니다.

  • 이 컴퓨터에서 오른쪽 클릭 -> 관리 -> 디스크 관리 -> 새로 추가된 드라이브 찾기 (잘못 선택하면 큰일 납니다) -> 모든 파티션 삭제 -> 임의의 파일 시스템으로 파티션 생성 (시스템이 이미 설치된 경우 두 개의 드라이브가 있을 수 있습니다. 이 경우 무시하고 포맷하거나 파티션을 삭제하면 됩니다.)

이미지 굽기​

  1. Raspberry Pi Imager 여기를 클릭하여 다운로드를 사용하여 이미지를 굽습니다.

  2. 굽기 전에 계정 및 비밀번호를 설정하고 SSH를 활성화해야 합니다.

  3. 굽기가 완료되면, 내 컴퓨터에 USB 드라이브가 나타납니다.

굽기 완료 후 재부팅​

  • 굽기가 완료되면 전원을 끄고, 컴퓨터와의 연결선을 끊고, BOOT 스위치를 원래 위치로 되돌립니다. 다시 전원을 켜면 됩니다.

라즈베리 파이 zero2 및 zero2W 주의 사항​

  • 이 주의 사항은 FLY-TFT 화면과 G2T를 함께 사용할 때 설명하는 것입니다.
  • 라즈베리 파이 zero2 및 zero2w는 기본 디스플레이 장치를 수정해야 하며, 아래 코드를 실행하여 직접 수정할 수 있습니다.\
  • 수정이 완료되면 재부팅해야 사용할 수 있습니다.
sudo sed -i 's/\/dev\/fb0/\/dev\/fb1/g' /etc/X11/xorg.conf.d/99-fbdev.conf
