## 첫 번째 화면
    :::warning 주의사항

    * ECM 또는 Rndis를 지원하는 시스템이어야 정상적으로 사용할 수 있습니다.
    * 본 제품은 Klipper용으로 설계되었으며 Moonraker 서비스와 함께 사용해야 IP 및 자원을 표시할 수 있습니다. 그렇지 않으면 백그라운드를 열고 SSH 모드를 설정해야 사용 가능합니다.
    
    :::
    * IP 표시는 연결된 기기에서 읽어온 정보를 기반으로 표시되며, IP를 읽을 수 없거나 읽지 못한 경우 아래와 같은 메시지가 표시됩니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ip.webp').default} size="100%" align="left" />
    * 정상적으로 읽어오면 아래와 같은 화면이 나타납니다 (IP는 상위 기기에서 표시됨)
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ip1.webp').default} size="100%" align="left" />

## 두 번째 화면

    * 연결된 기기의 CPU 및 메모리 사용량을 표시합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/cpu.webp').default} size="100%" align="left" />

## 세 번째 화면

    * 상위 기기에서 실행 중인 서비스를 표시합니다. 주의: 표시되는 서비스는 Klipper와 관련된 서비스입니다.
    * 만약 서비스가 실행되지 않았다면 해당 서비스 앞에 빨간 점이 표시됩니다.
    * 만약 서비스가 실행 중이라면 해당 서비스 앞에 파란 점이 표시됩니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/service.webp').default} size="100%" align="left" />
