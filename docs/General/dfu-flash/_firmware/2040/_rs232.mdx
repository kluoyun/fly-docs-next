

    下面介绍如何编译固件：
    * 连接到SSH后输入下面的命令并回车：

        ```bash
        cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
        ```

    * 其中` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`是删除之前编译数据与固件`make menuconfig`是编译固件,执行后应该出现下方界面

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

    * 选择**Enable extra low-level configuration options**并且回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

    * 进入菜单**Micro-controller Architecture**然后选择**Raspberry Pi RP2040**回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="Center" />

    * 选择**Bootloader offset**，选择：**NO bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.png').default} size="100%" align="Center" />

    * 选择**Communication interface**，选择：**UART0 on GPIO0/GPIO1**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/rs232.png').default} size="100%" align="Center" />
