## Особенности продукта

- 32-битный ARM Cortex-M4, 550 МГц, чип STM32H723VGT6
- Поддержка режимов работы SPI, UART, STEP/DIR с драйвером TMC
- Функция DIAG не занимает место для концевиков, после настройки может обеспечить бесконечную прокрутку
- Поддержка числового управления вентиляторами с напряжением 24В, 12В и 5В без необходимости дополнительного модуля понижения напряжения
- Поддержка автовыключения при отключении питания и продолжения печати после восстановления (автовыключение по умолчанию включено)
- Предустановленное ПО USB Klipper при выпуске
- Многоканальное управление питанием, короткое замыкание портов, концевиков или драйверов не влияет на работу основной системы
- Зарезервировано подключение RGB с напряжением 5В
- Поддержка передачи данных по одному кабелю для TFT и HDMI-экранов
- Предустановленное ПО klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, компенсационные плагины numpy для резонансов, не требует замены источника, простое в использовании (требуется покупка карты памяти TF или M2WE)

## Технические параметры

### Верхний контроллер

- **CPU**: Allwinner H618, 4-ядерный 64-битный Cortex-A53
- **GPU**: Arm Mali-G31
- **ОЗУ**: 1 ГБ DDR4
- **ПЗУ**: максимальная поддержка 128 ГБ microSD
- **Периферия**:
    - TFT x1
    - USB 2.0 x2
    - Micro HDMI x1
    - Сетевой разъем (Eth) x1 (100Мб/с)
    - FPC-HDMI x1
    - UART x1
- **Интерфейсы**: интегрированный M.2, расширение возможностей Wi-Fi, eMMC и т.д. (частная спецификация, подключать только устройства FLY)
- **Поддержка**: передача данных по одному кабелю для экранов TFT и HDMI
- **Предустановленное ПО**: klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, компенсационные плагины numpy для резонансов, не требует замены источника, простое в использовании (требуется покупка карты памяти TF или M2WE)

### Нижний контроллер

- **MCU**: STM32H723VGT6
- **Фirmware**: KLIPPER/RRF/Marlin
- **Питание**: поддерживает питание 12-24 В постоянного тока, отсутствует необходимость в модуле понижения напряжения
- **Максимальный ток для нагревательного элемента термоблока**: 15 А
- **Максимальный ток для нагревательных элементов**: 6 А
- **Максимальный ток для вентиляторов**: 1 А
- **Режим работы драйверов**: SPI, UART, STEP/DIR
- **Поддерживаемые драйверы**: TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и другие
- **Расширяемые интерфейсы**: BLTouch (Servos, Probe), RGB x1, UART x1, CAN x1
- **Вентиляторы**: двухканальные программируемые вентиляторы x4 (поддерживают 24В, 12В и 5В)
- **Датчики температуры**: 3 канала NTC 100К
- **Поддерживаемые конструкции**: Cartesian, Delta, Kossel, Ultimaker, CoreXY и другие
