시스템 소개 및 다운로드

소개​

  • FlyOS-Fast는 3D 프린팅을 위한 경량화된 Linux 시스템으로, Fly 전체 시리즈의 상위 플랫폼을 지원합니다.
  • FlyOS-Fast는 경량화되어 빠르게 부팅됩니다.
  • 커널은 Klipper 실시간성을 보장하기 위해 최적화되었습니다.
  • 파일 시스템이 최적화되어 직접 전원이 꺼져도 부팅할 수 없는 위험을 줄입니다.
  • 빠른 부팅, 30초 이내에 KlipperScreen 시작
  • Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen 등의 서비스가 미리 설치되어 있습니다.
  • Klipper와 RRF 시스템 간의 스위치를 지원합니다.
  • mainsail과 fluidd 온라인 한 번 클릭으로 스위치 가능
  • U디스크 프린트를 지원합니다.
  • 전력 차단 후 재개 인쇄를 지원합니다.
  • 시리얼 포트 스크린을 지원합니다.
  • fly-c8, fly-geminiv3를 사용하여 부트시 자동으로 펌웨어 업데이트를 지원합니다.
  • Fluidd 등 프론트엔드 UI의 기본 언어 및 기본 프린터 이름을 사전에 설정하는 것을 지원합니다.
  • 추가 OEM 맞춤 기능: 부트 로고, 미리 설정된 구성 파일, Mainsail 탭 숨기기 기능 등은 저희에게 문의하세요.

시스템 이미지 파일명 소개​

  • 시스템 파일 명명 규칙: FlyOS-Fast-[칩 시리즈]-[버전 번호].img, 예: FlyOS-Fast-H5-v1.0.0.img.
  • 원본 파일은 .img 형식이며, 다운로드 링크에서는 .exe 및 .7z 형식의 압축 파일만 제공하여 다운로드를 편리하게 합니다.
  • Windows 시스템에서 .exe 파일을 다운로드합니다. 이 파일은 7z 자동 해제 압축 패키지로, 다운로드가 완료되면 더블클릭하여 해제하면 .img 파일을 얻을 수 있습니다 (이미지만 윈도우 시스템에서 압축을 풀 수 있습니다).
  • MacOS, Linux 시스템 등 다른 시스템에서 .7z 파일을 다운로드합니다. 이 파일은 7z 압축 패키지로, 다운로드가 완료되면 7z 포맷을 지원하는 도구로 압축을 풀면 .img 파일을 얻을 수 있습니다.
  • 시스템 파일이 크기 때문에, 현재는 다른 압축 형식 및 원본 이미지 다운로드를 제공하지 않습니다.
시스템 선택
  • fly-pi 시리즈, fly-c8 시리즈, fly-gemini 시리즈 등, 모든 Allwinner H5 칩을 사용하는 Fly 제품 시리즈는 FlyOS-Fast-H5.exe 또는 FlyOS-Fast-H5.7z를 선택하세요.
  • fly-pi-lite2 시리즈, fly-miniPad 시리즈 등, 모든 Allwinner H3 칩을 사용하는 Fly 제품 시리즈는 FlyOS-Fast-H3.exe 또는 FlyOS-Fast-H3.7z를 선택하세요.
  • 두 가지 칩 모델이 다르므로 시스템 파일도 다르므로, 실제 상황에 따라 선택해야 합니다.

시스템 이미지 다운로드​

  • FlyOS-Fast 공식 다운로드 주소: FlyOS-Fast制品库
  • 기본 사용자 이름: root
  • 기본 비밀번호: mellow
주의사항

이 시스템은 경량화된 맞춤형 시스템으로, 내장된 klipper 등 구성 요소 및 서비스는 공식 원본이 아니며 사용자가 임의로 수정하거나 업그레이드할 수 없습니다.

시스템 이미지 다운로드 프로세스​

  • FlyOS-Fast 이미지 다운로드 프로세스:

    1. 브라우저를 열어 위의 FlyOS-Fast制品库를 방문합니다.

    2. 필요한 압축 형식을 선택하고 파일 페이지로 들어갑니다.

    1. 버전을 선택합니다 (기본값은 최신 버전).
    1. 다운로드 버튼을 볼 수 없다면, 마우스를 파일 목록 안으로 이동하고 Shift 키를 누른 상태에서 마우스 휠을 스크롤하여 파일 목록을 좌우로 스크롤할 수 있습니다.
    2. 또한 아래쪽 스크롤 바를 마우스 왼쪽 버튼으로 길게 누른 상태에서 오른쪽으로 드래그하여 다운로드 버튼을 볼 수 있습니다.
    1. 다운로드 버튼을 클릭하여 해당 버전의 시스템 파일을 다운로드할 수 있습니다.
