Инкрементальное обновление (OTA)​

  • На странице информации устройства Halo, рядом с номером версии, можно нажать для проверки обновлений. Если есть доступное обновление, появится уведомление. Нажмите "Update", чтобы начать обновление. В процессе не отключайте питание, дождитесь 100% прогресса, после чего устройство перезагрузится автоматически.

Полное обновление (USB)​

  • Если выпущена полная версия прошивки, вам нужно подключить Halo к компьютеру через USB для обновления
подсказывать
  • При полном обновлении все ваши настройки будут сброшены на значения по умолчанию
  1. Скачайте и распакуйте инструмент для записи USB flash_download_tool_3.9.4.zip(скачать)
  2. Скачайте прошивку Halo прошивка
  3. Откройте flash_download_tool_3.9.4.exe, обратите внимание: после открытия инструмента для записи сначала появится окно CMD, затем откроется интерфейс инструмента. Не закрывайте окно CMD
  4. В инструменте записи выберите ChipType: как ESP32-S3
  5. В инструменте записи выберите WorkMode: как Develop
  6. В инструменте записи выберите LoadMode: как USB
  7. Затем нажмите OK, смотрите рисунок 1 ниже
подсказывать
  • Область, обозначенная зеленой рамкой на рисунке 2, должна соответствовать изображению, не изменяйте ее
  1. Подключите Halo к компьютеру через USB-Typec кабель
  2. Нажмите кнопку в области 1 синего цвета на рисунке 2, чтобы выбрать загруженную прошивку
  3. Нажмите кнопку в области 2 синего цвета на рисунке 2, чтобы выбрать порт для Halo (если он не отображается, удерживайте кнопку B на задней панели Halo и подключите его к компьютеру)
  4. Нажмите кнопку в области 3 синего цвета на рисунке 2, чтобы начать запись. После завершения записи перезагрузите Halo для завершения полного обновления (не отключайте Halo, пока не появится сообщение о завершении записи)
