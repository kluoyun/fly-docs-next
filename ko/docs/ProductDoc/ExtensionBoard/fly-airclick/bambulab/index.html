BambuLab에 연결

중요 팁
  • AirClick은 대상 프린터와 동일한 네트워크에 연결해야 합니다.
  • LAN Only Mode로 BambuLab 프린터에 연결을 지원합니다.

  1. BambuLab 프린터의 LAN Only Mode를 엽니다.

    • BambuLab 프린터 화면을 엽니다.
    • 설정 -> LAN Only Mode -> 활성화

  2. 왼쪽 시스템 설정을 클릭합니다.

  3. Printer Type을 BambuLab으로 선택합니다.

  4. 장치 목록 새로고침 버튼을 클릭합니다.

  5. 프린터를 선택합니다.

    • 프린터가 항상 새로고침되지 않는 경우 사용자 정의 입력을 선택할 수 있습니다. 프린터의 IP 주소와 시리얼 번호를 수동으로 입력합니다.

  6. 액세스 코드를 입력합니다.

    • 액세스 코드는 프린터의 LAN Only Mode 액세스 코드이며, 단계 1에서 프린터 설정에서 확인할 수 있습니다.

  7. 위의 구성이 완료되면 오른쪽 하단의 제출을 클릭하여 구성 저장합니다.

  8. 올바르게 연결되면 웹 페이지 상단에 메시지가 표시됩니다.

  9. 연결 실패 시

    • IP 주소가 올바른지 확인합니다.
    • 프린터와 AirClick이 동일한 네트워크에 연결되어 있는지 확인합니다.
    • 프린터가 LAN Only Mode가 활성화되었는지 확인합니다.
    • 액세스 코드가 잘못 입력되었는지 확인합니다.
