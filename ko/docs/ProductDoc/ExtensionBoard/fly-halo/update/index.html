증분 업그레이드(OTA)​

  • Halo 장치 정보 페이지에서 버전 번호 위치에 업데이트를 확인할 수 있는 버튼이 있습니다. 업데이트가 가능하면 정보가 팝업되고, 이 때 Update를 클릭하면 업그레이드가 시작됩니다. 업그레이드 중에는 전원을 끊지 말고, 진행률이 100%가 되면 자동으로 재부팅됩니다.

전체 업그레이드(USB)​

  • 전체 업그레이드 펌웨어가 배포되면 Halo를 USB로 컴퓨터에 연결하여 업그레이드해야 합니다.
팁
  • 전체 업그레이드 시 기존 설정은 유지되지 않으며, 모든 설정이 기본값으로 복원됩니다.
  1. USB 번인 도구를 다운로드 및 압축 해제 flash_download_tool_3.9.4.zip(다운로드 클릭)
  2. 펌웨어 다운로드 Halo 펌웨어 다운로드
  3. flash_download_tool_3.9.4.exe를 실행합니다. 주의: 번인 도구를 실행하면 먼저 CMD 창이 열리고, 그 다음 번인 도구의 인터페이스가 열립니다. CMD 창을 닫지 마십시오.
  4. 번인 도구에서 ChipType:를 ESP32-S3로 선택합니다.
  5. 번인 도구에서 WorkMode:를 Develop로 선택합니다.
  6. 번인 도구에서 LoadMode:를 USB로 선택합니다.
  7. 마지막으로 OK를 클릭하면 됩니다. 참조할 수 있는 그림1을 아래에 표시합니다.
팁
  • 그림2의 초록색 박스 선택 영역은 그림과 동일해야 하며, 임의로 수정할 수 없습니다.
  1. Halo를 USB-Typec 데이터 케이블로 컴퓨터에 연결합니다.
  2. 그림2의 파란색 영역1 버튼을 클릭하여 다운로드한 펌웨어를 선택합니다.
  3. 그림2의 파란색 영역2에서 Halo의 시리얼 포트를 선택합니다 (시리얼 포트가 없으면, Halo 뒷면의 B 버튼을 누른 상태로 컴퓨터에 연결합니다).
  4. 그림2의 파란색 영역3을 클릭하여 번인을 시작합니다. 번인이 완료된 후 Halo를 재부팅하면 전체 업그레이드가 완료됩니다 (반드시 번인이 완료되었다는 메시지가 나타난 후에 Halo의 케이블을 분리하십시오).
