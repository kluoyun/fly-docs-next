WIFI 연결

LITE 2.1에 안테나 설치​

  • 빨간 화살표가 가리키는 곳이 WIFI 안테나 접속부입니다.
  • WIFI 안테나 포트는 ipex1 입니다.
  • 안테나를 LITE 2.1에 설치하지 않으면 신호가 약해져 네트워크에 연결할 수 없을 수 있습니다.
  • WIFI가 정상적으로 연결되지 않는다면 안테나가 제대로 설치되어 있는지 확인한 후 아래 방법으로 WIFI를 연결하세요.

FLYOS-Armbian 시스템 WiFi 연결​

  1. 먼저 시리얼 포트를 사용하여 상위 기계에 연결하세요: FLYOS-FAST 시리얼 포트를 사용하여 상위 기계 연결

  2. 명령어 sudo nmtui를 입력하면 비밀번호를 입력하라는 메시지가 표시될 수 있습니다. 비밀번호 mellow를 입력하고 Enter 키를 누르면 아래의 그림과 같은 화면이 나타납니다 (참고: 입력된 비밀번호는 보이지 않습니다)

  1. Activate a connection 옵션을 선택한 후 Enter 키를 누르면 다음 단계로 들어갑니다.

::: tip WiFi 설정 페이지 사용 설명서

  • 반각 모드, 즉 영문 모드로 키보드 입력기를 설정하시기 바랍니다.
  • 키보드의 ↑ 키, ↓ 키는 커서를 이동하여 메뉴 항목을 선택하는 데 사용됩니다.
  • 확인 키 Enter는 메뉴를 선택하거나 하위 메뉴로 진입하는 데 사용됩니다.
  • 종료 키 ESC는 상위 메뉴로 돌아가는데 사용됩니다. :::
  1. 라우터의 WiFi가 정상 작동하는 경우 아래 그림과 같이 WiFi 목록을 볼 수 있습니다. WiFi 이름 뒤의 *는 신호 강도를 나타냅니다
팁
  • 한글 네트워크 이름은 지원되지 않습니다!!
  • 2.4G WiFi는 지원하지만, 5G WiFi는 구입한 WiFi 모듈이 5G WiFi를 지원하는지 여부에 따라 다릅니다.
  • M2WE는 5G WiFi를 지원합니다.
  1. 연결할 WiFi를 선택한 후 Enter 키를 누르면 다음 단계로 들어갑니다.
  1. 다음 화면이 나타나므로 올바른 WiFi 비밀번호를 입력하고 Enter 키를 누르면 WiFi에 연결됩니다.
  1. 비밀번호가 맞다면 해당 WiFi 앞에 * 기호가 표시되어 연결 성공을 나타냅니다.
  1. 다음으로 두 번의 Esc 키를 눌러 명령줄 인터페이스로 돌아갑니다.
  1. SSH에서 ip a | grep inet 명령어를 입력하고 Enter 키를 누릅니다. 반환된 내용에서 라우터와 같은 세그먼트의 IP를 찾으면 접속할 수 있습니다.
  1. 브라우저에서 검색창에 방금 확인한 상위 기계의 IP를 입력하고 Enter 키를 누르면 fluidd/mainsail의 웹 컨트롤 패널에 접속할 수 있습니다.
  1. 브라우저의 검색창에 IP:9999를 입력하고 Enter 키를 누르면 fly tools 인터페이스에 접속할 수 있으며, 여기서 상위 기계의 다양한 기능을 활성화할 수 있습니다.
팁
  • fly tools는 FLYOS-Armbian 시스템에만 있습니다. FLYOS-FAST에는 fly tools가 없습니다.
