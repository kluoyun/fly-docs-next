시스템 시간대 변경

FLY 상위 기기에서 FLY OS 3.3.4 버전을 사용하는 경우 FLY-Tools를 사용하여 시간대를 직접 변경할 수 있습니다. 다른 상위 기기는 아래 작업을 참조하세요.

sudo timedatectl set-timezone <TIMEZONE>

<TIMEZONE>을 사용하고자 하는 시간대 코드로 교체하십시오. 시간대 코드는 iana.org 웹사이트에서 찾을 수 있습니다. 예를 들어, 시스템 시간대를 베이징 시간대(Asia/Shanghai)로 변경하려면 다음 명령을 입력하십시오:

sudo timedatectl set-timezone Asia/Shanghai

이제 시스템 시간대가 성공적으로 상하이 시간으로 변경되었습니다. 시간대 설정을 확인하려면 다음 명령을 사용하십시오:

date
