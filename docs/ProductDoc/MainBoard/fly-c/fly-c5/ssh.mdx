# 连接上位机SSH

:::danger 注意
- 如果您的上位机为烧录镜像后第一次启动，请务必使用`串口连接到上位机SSH`
- 第一次启动时，禁止连接除供电和Type-C之外的任何线！！！
- 编译固件时，请务必使用`WiFi连接到上位机SSH`，以方便固件编译和烧录！
:::

* 连接开始前，请下载： <Button variant="contained" disableElevation component="a" rel="noopener" href="https://mobaxterm.mobatek.net/download-home-edition.html ">SSH软件</Button>    选择右边的`Installer edition`即可下载

        ## 使用WiFi连接到上位机SSH
       :::tip 注意
        - 请确保上位机系统已经**正常启动**
        - FLYOS-FAST系统默认用户名`root`, 默认密码`mellow`
        - 通过WiFi连接到上位机SSH的**前提**是：您的**上位机已经连接到路由器，并且知道上位机的IP**
       :::

        * 使用Type-c数据线将FLY-C5与电脑连接

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * 打开MobaXterm，设置如下图所示，`Remote host`处则填写您在**串口连接时查询到的上位机IP**，也可以通过路由器后台查询上位机**IP**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **注意：这里第4步输入的上位机的用户名`root`**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_2.webp').default} size="60%" align="center" />

        * 第一次连接回弹出这个窗口，点击`Accept`即可

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * 提示输入密码，输入上位机密码后回车
        :::tip
        - 这里输入的密码是看不到的，输完后直接回车即可
        :::
        
        **FLYOS-FAST** 系统默认用户名为：`root` ，密码为：`mellow`

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_3.webp').default} size="60%" align="center" />

        **如果弹出下图所示界面，此弹窗提示是否保存用户密码，点击`No`即可**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * 出现下图所示界面，则登录成功！

        :::warning 注意 
        - FLYOS-FAST 系统账户为`root`，密码为`mellow`
        - FLYOS-FAST 系统有且仅有`root`账户！！！
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_5.webp').default} size="60%" align="center" />



    </TabItem>
