# 시스템 소개 및 다운로드

## 개요

* FlyOS-Fast는 3D 프린팅에 최적화된 가벼운 Linux 시스템으로, Fly 전체 시리즈의 상위 플랫폼을 지원합니다.
* FlyOS-Fast는 가볍고 부팅 속도가 빠릅니다.
* 커널은 Klipper의 실시간성을 보장하기 위해 최적화되어 있습니다.
* 파일 시스템이 최적화되어 직접 전원이 꺼져도 시스템이 시작되지 않는 문제를 줄입니다.
* 30초 내에 KlipperScreen을 시작할 수 있습니다.
* Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen 등의 서비스를 미리 설치했습니다.
* Klipper와 RRF 시스템 간의 전환을 지원합니다.
* Mainsail과 Fluidd 온라인에서 한 번의 클릭으로 전환 가능.
* U디스크 인쇄를 지원합니다.
* 전력이 끊겼을 때 다시 인쇄를 이어갈 수 있습니다.
* 시리얼 스크린을 지원합니다.
* Fly-C8, Fly-Geminiv3에서 부팅 시 자동 펌웨어 쓰기를 지원합니다.
* Fluidd 등 프론트엔드 UI의 기본 언어 및 기본 프린터 이름을 사전 설정할 수 있습니다.
* 추가 OEM 맞춤 기능: 로고 설정, 기본 구성 파일 지정, Mainsail 탭 숨기기 설정 등은 저희에게 문의하세요.

## 시스템 이미지 파일명 설명

* 시스템 파일 명명 규칙: `FlyOS-Fast-[칩 시리즈]-[버전 번호].img`, 예: `FlyOS-Fast-H5-v1.0.0.img`.
* 원본 파일 형식은 `.img`이며, 다운로드 링크에서는 `.exe`와 `.7z` 형식의 압축 파일만 제공하여 다운로드가 편리하도록 합니다.
* Windows 시스템에서는 `.exe` 파일을 다운로드하고, 이 파일은 7z 자동 압축 패키지입니다. 다운로드 후 더블 클릭하여 압축을 해제하면 `.img` 파일을 얻을 수 있습니다 (Windows 시스템에서만 압축 가능).
* MacOS 또는 Linux 시스템에서는 `.7z` 파일을 다운로드하고, 이 파일은 7z 압축 패키지입니다. 다운로드 후 7z 형식을 지원하는 도구를 사용하여 압축을 해제하면 `.img` 파일을 얻을 수 있습니다.
* 시스템 파일 크기가 큰 관계로 현재 다른 압축 형식 및 원본 이미지 파일은 제공하지 않습니다.

:::tip 시스템 선택

* fly-C5: Allwinner H618 칩을 사용하는 Fly 제품군은 `FlyOS-Fast-H618.exe` 또는 `FlyOS-Fast-H618.7z`를 선택하세요.
* fly-pi 시리즈, fly-c8 시리즈, fly-gemini 시리즈 등: Allwinner H5 칩을 사용하는 모든 Fly 제품군은 `FlyOS-Fast-H5.exe` 또는 `FlyOS-Fast-H5.7z`를 선택하세요.
* fly-pi-lite2 시리즈, fly-miniPad 시리즈 등: Allwinner H3 칩을 사용하는 모든 Fly 제품군은 `FlyOS-Fast-H3.exe` 또는 `FlyOS-Fast-H3.7z`를 선택하세요.
* 칩 모델이 다르므로 시스템 파일도 다릅니다. 실제 상황에 따라 선택하시기 바랍니다.

:::

## 시스템 이미지 다운로드

* FlyOS-Fast 공식 다운로드 주소: [FlyOS-Fast 제품 라이브러리](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/packages)

- 기본 사용자 이름: `root`
- 기본 비밀번호: `mellow`

:::tip 유의사항

이 시스템은 가벼운 사용자 정의 시스템이며, 내장된 klipper 등 구성 요소 및 서비스는 공식 버전이 아닙니다. 사용자가 임의로 수정하거나 업그레이드하는 것을 지원하지 않습니다.

:::

## 시스템 이미지 다운로드 절차

* FlyOS-Fast 이미지 다운로드 절차:

    1. 브라우저로 위의 FlyOS-Fast 제품 라이브러리를 방문합니다.

    2. 필요한 압축 형식을 선택하고 파일 페이지로 이동합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down1.webp').default} size="50%" align="center" />

    3. 버전을 선택합니다 (기본값은 최신 버전).

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down2.webp').default} size="50%" align="center" />

    4. 다운로드 버튼이 보이지 않으면 마우스를 파일 목록 안으로 이동한 후 Shift 키를 누른 상태에서 마우스 휠을 스크롤하여 파일 목록을 좌우로 스크롤할 수 있습니다.
    5. 또한 아래쪽 스크롤 막대를 클릭한 후 왼쪽 마우스 버튼을 누른 상태에서 오른쪽으로 드래그하면 다운로드 버튼을 볼 수 있습니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down3.webp').default} size="50%" align="center" />

    6. 다운로드 버튼을 클릭하여 해당 버전의 시스템 파일을 다운로드합니다.
