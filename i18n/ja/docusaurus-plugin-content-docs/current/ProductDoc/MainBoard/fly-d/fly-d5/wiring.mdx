## メインボードの配線

:::warning 注意事項
画像の解像度が高いため、ウェブページ上で見ると鮮明に見えません。マウスカーソルを接線図の上に置き、右クリックして「画像を別名で保存」を選択し、デスクトップに保存してからご覧ください！
:::

:::warning 警告
購入した完成品の「3本線リミットスイッチ」は使用しないでください。このようなリミットスイッチは、**メインボードのショート**や**メインボードの損傷**の原因となる可能性があります！！！接線図に示されている接線方法に従って必ず接続してください！！！
:::
