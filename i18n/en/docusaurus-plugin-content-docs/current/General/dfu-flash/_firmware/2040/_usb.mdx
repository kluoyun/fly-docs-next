

    Below is an introduction on how to compile the firmware:
    * After connecting to SSH, enter the following command and press Enter:

        ```bash
        cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
        ```

    * Where `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` is to delete previous compilation data and firmware. `make menuconfig` is for compiling the firmware, after which the following interface should appear:

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

    * Select **Enable extra low-level configuration options** and press Enter

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

    * Enter the menu **Micro-controller Architecture** then select **Raspberry Pi RP2040** and press Enter

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="Center" />

    * Select **Bootloader offset**, choose: **16KiB bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.png').default} size="100%" align="Center" />

    * **Communication interface**, choose: **USBSERIAL**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb.png').default} size="100%" align="Center" />
