连接WIFI

将天线安装到LITE 2上​

  • 红色箭头所指的地方为WIFI天线接口
  • WIFI天线口是ipex1
  • 将天线安装到LITE 2上，否则可能因为信号差导致无法联网
  • 如果发现没有正常连接WIFI可以先检查天线有没有接上，然后在使用下方方法连接WiFi
须知
  • FLY上位机需要使用M2WE或者2.4G WIFI 模块否则使用WiFi
  • 请注意不支持中文名字的WIFI与2.4G和5G双频合一的WIFI
  • 如果是mini pad或者lite2请无视此警告

连接上位机​

  • 如果在BOOT盘下配置WIFI失败可以参考此配置进行WIFI连接，请注意需要使用串口连接上位机
  • 请先使用串口连接到SSH
  • 执行下面的命令, 可能会提示输入密码，输入密码mellow回车，请注意密码不可见
sudo nmtui

  • 选择Activate a connection后回车

  • 进入如下界面，通过键盘上的↑ ↓键选择要连接的WiFi后，回车，输入密码后等待连接成功

  • 连接成功后如图所示

  • 按ESC键退出，返回命令行界面，输入ip a，图中192.168.2.126即为上位机的IP地址
