# 固件烧录说明

    
## 如何判断你需要的哪一种固件

    :::warning
    - 注意，由于C5为上下位机一体的主板，只要不使用CAN工具板，请务必选择`USB固件`！！！
    - 在使用CAN工具板时，请务必选择`CAN桥接固件`
    :::
    


    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB固件</Button>
        * 优点：3D打印机中**最常用的连接方式**，通信可靠，稳定
        * 缺点：如果多个USB设备可能会影响数据传输  

    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB桥接固件</Button>
        * 优点：可以同时接多个设备，并且不占用USB口。**只有在使用CAN工具板时，才需使用此固件**
        * 缺点：第一次编译烧录固件会麻烦些。
