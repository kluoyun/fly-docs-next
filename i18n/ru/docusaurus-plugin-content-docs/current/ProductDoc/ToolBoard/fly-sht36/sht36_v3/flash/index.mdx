# Инструкция по прошивке прошивки
    
    * Для компиляции и прошивки прошивки требуется ПО верхнего уровня
    * SHT36 V3 может подключаться к ПО верхнего уровня через RS232 или CAN
    * RS232 и CAN требуют использования `XT30(2+2)-F` для подключения
    * Ниже приведен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Для прошивки материнской платы через Katapult необходимо прошить Klipper
* Если вы уверены, что материнская плата уже прошита Katapult, то нет необходимости выполнять последний шаг прошивки прошивки Katapult — это просто мера предосторожности на случай неудачной прошивки
* Можно быстро дважды нажать кнопку Reset и проверить, загорается ли один из светодиодов на материнской плате; если да, то это означает, что материнская плата находится в режиме Katapult
:::

## Как определить нужную прошивку

    * SHT36 V3 может подключаться к ПО верхнего уровня следующими способами
    * <Button variant="contained" disableElevation href="./can">CAN-прошивка</Button>
        * Преимущество: если установлены прошивки Katapult, можно обновить прошивку без физического контакта с материнской платой и поддерживаются шины подключения
        * Недостаток: возможны ошибки типа превышения времени сброса при совместном использовании с KlipperScreen
        * Примечание: если требуется использовать CAN, убедитесь, что ПО верхнего уровня поддерживает CAN, и используйте его вместе с UTOC или материнской платой с прошивкой моста
    * <Button variant="contained" disableElevation href="./rs232">RS232-прошивка</Button>
        * Преимущество: не требуется специальное программное обеспечение для использования, надежная и стабильная связь
        * Недостаток: для повторной прошивки требуется ручной переход в DFU или двойное нажатие RST для входа в Katapult для обновления прошивки
        * Примечание: необходима совместная работа с G2T или UTOR
