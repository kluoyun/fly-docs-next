FLY_halo

1. ホームページ​

ヒートベッドまたはホットエンドをクリックし、画面上部から時計回りにスライドして目標温度に達したら、再度ヒートベッドまたはホットエンドをクリックして温度を設定します。

MCU温度とホスト温度を表示する必要がある場合は、以下の設定を追加します。

[temperature_sensor mcu]
sensor_type: temperature_mcu      # MCUに関連付け（デフォルト）

[temperature_sensor host]        # ホスト温度
sensor_type: temperature_host     # ホストに関連付け
注意事項
  • 最高温度は設定ファイルの値を読み取ってください。

2. 押出し​

  • アイコンを長押ししてホットエンド温度、押出し長さ、押出し速度を設定できます。
注意事項
  • haloは単一のホットエンドの制御のみをサポートします。

3. システムサービス​

サービスをクリックすると、停止、再起動、起動ができます。

4. 設定​

  1. WIFIのリセット
  2. KLIPPERのIPのリセット
  3. 言語選択
  4. スクリーンセーバー時間、スクリーンオフ時間、バックライト、画面の回転などの機能
