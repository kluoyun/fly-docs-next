# Firmware Update
    :::danger Notice

    * A WINDOWS system is required to use it.
    * Only the following two connection methods can be used, and only one of them can be selected. There are no other connection methods!!!

    :::
## Download Firmware

    <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/usblcd/fly-uip.uf2/version/52147655/files">Firmware Download Address</Button>
    * Click the file list
    * Then click and hold the left mouse button on the scrollbar and drag it to the right
    * Then click the download button
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/firmware.webp').default} size="100%" align="left" />


## Update Method
    * After connecting U-IP to the computer, enter ``192.168.4.1`` in the address bar of the browser
    * Find System Control and click Restart into Bootloader Mode
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

### Update Firmware
    
    * After clicking Restart into Bootloader Mode, a `RPI-RP2` disk will pop up on the computer
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf2.webp').default} size="100%" align="center" />
    * Copy the firmware to be flashed into it
    * Reference
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf21.webp').default} size="100%" align="center" />
    * After the firmware is copied in, wait for about ``ten seconds``, and the `RPI-RP2` disk will automatically disappear. At this point, the firmware flashing is complete!
