WiFi 연결

중요 안내
  • 본 제품의 내장 안테나는 수신만 가능하며, 송신용 안테나는 별도로 설치해야 합니다. 안테나 단자는 ipex3입니다.
  • 안테나를 제대로 설치해 주십시오. 그렇지 않으면 신호가 너무 약해져서 발열이 심각해질 수 있습니다.
  • 첫 사용 시 AirClick 핫스팟에 수동으로 연결한 후, 해당 WiFi에 연결해야 합니다.

AirClick WiFi 연결​

  • AirClick의 핫스팟에 연결합니다. 핫스팟 이름은 FLY-WBTN-XXXXXX입니다.
  • 핫스팟에 연결한 후 브라우저를 열고 http://192.168.4.1/에 액세스합니다.
  • 이때 Wi-Fi 설정 화면이 나타납니다. 프린터와 동일한 라우터에 수동으로 연결하고 비밀번호를 입력한 후 아래쪽 파란 상자를 클릭하여 Wi-Fi에 연결하세요.
  • Wi-Fi에 연결한 후 AirClick의 연결된 Wi-Fi IP 주소가 표시됩니다. 아래쪽 파란 상자를 클릭하여 장치를 재시작할 수 있습니다.
    • 번호 1이 둘러싸인 부분은 AirClick의 IP 주소입니다.
    • 번호 2가 둘러싸인 부분을 복사하고 장치를 재시작하세요.
  • 현재 네트워크를 해당 라우터로 수동으로 전환해야 합니다. 그렇지 않으면 액세스할 수 없습니다.
  • 성공적으로 연결한 후에는 AirClick을 재시작하기 위해 전원을 차단하고 다시 켜는 것을 권장합니다.

언어 변경​

  • 오른쪽 위에 둘러싸인 아이콘을 클릭하고 원하는 언어를 선택하세요.
