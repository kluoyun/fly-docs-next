# ファームウェア更新
    :::danger 注意事項

    * 使用するにはWINDOWSシステムが必要です
    * 接続方法は以下の2つの方法のみを使用でき、どちらか一方を選択してください。他の接続方法はありません！！！

    :::
## ファームウェアのダウンロード

    <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/usblcd/fly-uip.uf2/version/52147655/files">ファームウェアのダウンロード先</Button>
    * ファイルリストをクリック
    * 次にスクロールバーを左クリックして長押しし、右にドラッグします
    * その後、ダウンロードボタンをクリックします
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/firmware.webp').default} size="100%" align="left" />


## 更新方法
    * U-IPをコンピューターに接続した後、ブラウザのURLバーに``192.168.4.1``と入力します
    * システムコントロールを見つけてクリックし、「リブートしてブートモードにする」をクリックします
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

### ファームウェアの更新
    
    * 「リブートしてブートモードにする」をクリックすると、コンピューターに`RPI-RP2`ドライブが表示されます
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf2.webp').default} size="100%" align="center" />
    * 更新するファームウェアをコピーします
    * 参照
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf21.webp').default} size="100%" align="center" />
    * ファームウェアをコピーした後、約``十秒後に`` `RPI-RP2` ドライブが自動的に消えます。この時点でファームウェアの更新が完了しています！
