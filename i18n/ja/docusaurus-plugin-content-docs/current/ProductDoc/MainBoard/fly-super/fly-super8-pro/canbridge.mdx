# CANブリッジ

CANバスは、リアルタイムアプリケーション向けのシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができます。これは、世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまなコンポーネント間の通信に使用され、高価でかさばる配線ハーネスを置き換えます。このプロトコルの堅牢性により、他の自動化および産業用途にも拡張されています。CANプロトコルの特性には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシングおよびエラーチェック機能が含まれます。3Dプリンターでは、CANバスを使用してプリントヘッドからマザーボードへの配線の数を減らします。以前は十数本のケーブルが必要でしたが、CANを使用すると4本のケーブルで済み、配線の数が大幅に減少し、配線の難易度が低くなります。本章では、CANのブリッジの使用について簡単に概説します。

## ツールヘッドボードはCANを使用してマザーボードに接続します

```
*ツールヘッドボードの電源線はCANポートに接続するのではなく、12V-24Vの電源線に接続してください。
```
