# Klipperに接続する

    :::info 重要なヒント
    * AirClickはターゲットプリンタと同じネットワークに接続する必要があります
    :::

1. 左側のシステム設定をクリックする

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. `Printer Type` で `Klipper` を選択する
3. `Moonraker IP` に klipperマシンのIPアドレスを入力する
4. `Moonraker Port` に klipper上位機のポートを入力する。デフォルトは `7125` で、上位機が他のポートを使用している場合は、対応するポートを自分で入力する必要があります
5. `Apikey` は無視してください。現在、Apikeyはサポートされていません

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. 上記のように設定した後、右下の `提出` をクリックして設定を保存する
7. 正しく接続されると、ウェブページの上部にメッセージが表示されます

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. 接続に失敗した場合
    - IPアドレスとポートが正しいか確認する
    - プリンタとAirClickが同じネットワークに接続されているか確認する
    - klipper上位機が正常に動作しているか確認する
    - ブラウザでklipper上位機のウェブページを開けるか確認する
