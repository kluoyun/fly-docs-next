


## CSI Usage

    * After the mirror image is burned, enter /boot/config.txt to find `camera_auto_detect=1` and change it to `camera_auto_detect=0`, then add configuration according to the camera model.
    * Below is a reference configuration.

    ```bash
    dtoverlay=imx219,cam1
    ```
