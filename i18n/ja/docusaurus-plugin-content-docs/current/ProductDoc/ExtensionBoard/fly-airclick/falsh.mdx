# ファームウェアのアップデートと書き込み

    :::danger 注意事項

    * 工場出荷時にはすでにファームウェアが予め書き込まれています

    :::

## ファームウェアのダウンロード

    * ファームウェアの説明
    * `wbtn-vX.X.X-X-XXXXXX-full.bin` は完全ファームウェアで、`0x0` アドレスから書き込む必要があり、書き込んだ後はすべての設定が保持されません
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin` はインクリメンタルファームウェアで、`0x10000` アドレスから書き込む必要があり、ファームウェアのみが更新され、既存の設定に影響を与えません
    * `wbtn-vX.X.X-X-XXXXXX-www.bin` は UI ファームウェアで、`0x290000` アドレスから書き込む必要があり、Web ページのフロントエンド UI のみが更新され、既存のファームウェアと設定に影響を与えません
    * <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/airclick.zip">ファームウェアパッケージのダウンロード</Button>

## ファームウェアのアップデート
    * `AirClick`の電源を切った状態で、BOOTに従って`AirClick`のESP32書き込みポートをコンピュータに接続します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.webp').default} size="80%" align="Center" />

    1. USB書き込みツールをダウンロードして解凍します <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">ダウンロードをクリック</Button>
    2. ``flash_download_tool_3.9.4.exe``を開きます。注意：書き込みツールを開くと、最初にCMDウィンドウがポップアップし、その後に書き込みツールのインターフェースがポップアップします。CMDウィンドウを閉じないでください
    3. 書き込みツールの``ChipType:``で``ESP32-C3``を選択します
    4. 書き込みツールの``WorkMode:``で``Develop``を選択します
    5. 書き込みツールの``LoadMode:``で``USB``を選択します
    6. 最後に``OK``をクリックします、下図を参考にしてください
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.webp').default} size="50%" align="Center" />
    7. クリックすると、下の画像が表示されます
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.webp').default} size="50%" align="Center" />

    :::tip

    * 上記の画像の緑色の枠で選択された領域は、画像と一致する必要があり、独自に変更してはいけません

    :::

    8. AirClickをUSB-TypeCデータケーブルでコンピュータに接続します
    9. 上記の画像の青い領域1のボタンをクリックして、書き込むファームウェアを選択します
    10. 上記の画像の青い領域2をクリックして、書き込みアドレスを入力します
    11. 上記の画像の青い領域3をクリックして、AirClickのシリアルポートを選択します（ない場合は、AirClickの書き込みボタンを押したまま、コンピュータに接続してください）
    12. 上記の画像の青い領域4をクリックして、ボーレートを最大`921600`に選択します
    13. 上記の画像の青い領域5をクリックして、書き込みを開始します。書き込みが完了したら、AirClickはファームウェアのアップグレードが完了します（書き込みが完了するまで待ってから、AirClickのデータケーブルを切断してください）
