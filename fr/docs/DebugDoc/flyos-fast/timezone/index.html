Tableau comparatif des fuseaux horaires mondiaux

À lire attentivement
  • Pour configurer les fuseaux horaires, entrez l'identifiant du fuseau horaire dans l'élément de configuration localtime, par exemple : Asia/Shanghai.
  • Le redémarrage du système est nécessaire pour que les modifications prennent effet.
  • Le temps du système est incorrect tant qu'il n'est pas connecté à Internet ; une fois connecté, le temps sera synchronisé automatiquement.
