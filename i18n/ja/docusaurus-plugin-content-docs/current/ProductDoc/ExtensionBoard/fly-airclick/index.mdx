# AirClick ワイヤレスボタン
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.webp').default} size="100%" align="Center" />



## 製品の特徴

    * 高い互換性：Klipper、RRF、BambuLabをサポートし、マシンに直接接続して予め設定されたコマンドを送信でき、シームレスな互換性と操作体験を提供します。
    * カスタムマクロコマンド：ユーザーは自身のニーズに応じてコマンドをカスタム設定でき、マシン操作をより柔軟かつ便利にします。
    * Klipper、RRF、BambuLabにワイヤレス接続をサポートします。有線接続はKlipperのみをサポートします。
    * ワイヤレス接続：ESP32の設計により、klipperに直接カスタムコマンドまたはマクロを送信でき、ユーザーに高い柔軟性と拡張性を提供します。
    * 有線接続：STM32F072チップを通じて、ネットワークでKlipperに接続できない場合でも、簡単な操作ができます。合計10個のカスタムコマンドまたはマクロをサポートします。
    * 使いやすさ：直感的なユーザーインターフェースを備えており、ユーザーはワイヤレスボタンを簡単に追加および管理でき、操作の利便性が大幅に向上します。

## 製品仕様

    * メインコントロールチップ：ESP32
    * 追加チップ：STM32F072
    * ESP32-WIFIを通じてコマンドを送信：制限なし
    * STM32F072-IOを通じてコマンドを送信：10個
    * 互換性：Klipper、RRF、BambuLabをサポート
    * 電源：5V

## 注意事項

    :::danger 注意事項

    * 本製品は5V電源で動作し、`type c`または`PH2.0 - 2P`端子を通じて給電できます。
    * 本製品のボード搭載アンテナはボタン信号の受信のみに使用できます。WIFIに接続する場合は、アンテナを正しく取り付ける必要があります。アンテナ端子は`ipex3`です。
    * アンテナを正しく取り付けてください！！！そうでないと、信号が弱すぎて発熱がひどくなります！！！
    * 注意してください。誤作動を防ぐため、ボタンを長押しして15秒後、システムは自動的にそれをオフにします。

    :::
