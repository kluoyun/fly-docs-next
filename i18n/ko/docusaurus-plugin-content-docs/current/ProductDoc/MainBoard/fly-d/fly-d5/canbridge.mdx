# CAN 브리지

CAN 버스는 실시간 응용 프로그램을 위한 시리얼 통신 프로토콜 버스로, 두 개의 트위스트드 페어 케이블을 사용하여 신호를 전송할 수 있으며, 세계적으로 가장 널리 사용되는 필드 버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 요소 간의 통신을 위해 사용되며, 비싸고 무거운 배선 하arness를 대체합니다. 이 프로토콜의 강력한 견고성 덕분에 자동화 및 산업 응용 분야에도 사용 범위가 확장되었습니다. CAN 프로토콜의 특징에는 완전한 시리얼 데이터 통신, 실시간 지원, 최대 1Mbps의 전송 속도, 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린팅 헤드가 메인보드에 연결되는 배선 수를 줄일 수 있습니다. 원래 십여 개의 선이 필요했지만, CAN을 사용하면 단지 네 개의 선만 사용하면 되어 배선 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리지를 사용하는 방법에 대해 간단히 설명합니다.

## 도구 보드가 메인보드와 CAN으로 연결됨

    * 도구 보드 전원선은 CAN 포트에 연결되지 않고 12V-24V 전원선에 연결된다는 점에 유의하십시오.
    * 120Ω 점퍼를 연결해야 한다는 점에 유의하십시오.

## 메인보드 펌웨어 업데이트

* D5 메인보드에 번들링된 펌웨어가 반드시 `USB 브리지 CAN 펌웨어 구성`인지 확인하십시오.
