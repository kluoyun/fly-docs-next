系统烧录

上位机新手操作指南​

  • 上位机新手操作指南

系统烧录前准备​

  • 读卡器
  • TF卡一张，容量必须不小于16GB 或 不大于128G TF 卡，TF 卡的传输速度必须为 class10 级或 class10 级以上
  • 下载对应的系统镜像
  • 下载需要的烧录工具

系统烧录​

備考
  • Lite 2 只能将系统烧录到TF卡,无法使用M2WE
  • 系统烧录完成后要将TF卡安装到Lite 2上

系统烧录TF卡

准备工作​

  • 先安装好软件Win32DiskImager
  • 将系统解压出来，解压后文件名后缀为.img
  • TF卡插入读卡器，将读卡器插到电脑USB接口
  • 如果TF卡有数据请备份刷完系统后系统卡不建议拆下来，除非需要烧录到M2WE后从可以拆下TF卡

格式化内存卡​

  • 使用TFFormatter.exe软件格式化TF 卡
  • 选择你需要格式化的磁盘
  • 或者使用WIN自带的磁盘管理删除分区,删除分区之后重新建立分区即可。

烧录到TF卡​

测试

  • 红色选择需要烧录的文件
  • 橙色选择需要烧录的TF卡
  • 蓝色将系统烧录到TF卡中
  • 确认烧录然后等待烧录完成
  • 烧录完成

系统配置​

ヒント

此文档正在更新

FLY_Config 的使用

  • FLY-Config用来简化系统配置等，适用于Klipper全家桶，免去进入ssh写命令的麻烦
  • 只能在FLY定制Armbian系统运行
  • 对初学者友好
  • 直接配置Klipper及周边

打开 FLY_Config.conf​

  • 如果您已经制作好TF卡，请使用读卡器连接到电脑。此时电脑应该会弹出资源管理器并且有一个“可移动磁盘BOOT” 如果未出现，请重新拔下TF卡，再插入电脑
  • 打开BOOT盘下的FLY_Config.conf

WIFI配置​

  • 找到WIFI配置，将WIFI处的flase改成true，并将WiFi名称和密码填写上，保存，插回上位机。等待几分钟后，进入路由器后台管理界面查看IP地址。
注意事项

系统启动后会将此wifi保存到系统，并且会自动将WiFi=true修改为WiFi=flase

屏幕配置​

注意事项

没有屏幕则不需要配置

打开KlipperScreen​

  • 需要将箭头所指的false改成true

配置屏幕​

  • LCD HDMI屏幕使用
  • TFT屏幕使用
  • 将下方箭头所指的NONE改成HDMI
  • 屏幕接线
  • 将下方箭头所指的NONE改成FBTFT
  • 此配置无需修改保存即可
  • 下方是两个TFT屏幕的接线，请注意屏线方向！！！

将TF卡安装到LITE 2上​

  • 在将系统烧录到TF卡后，需要将TF卡接到LITE 2此处，否则系统无法启动
