# BL固件

    :::tip

    * 请注意出厂已经预刷了`bootloader`无需在刷

    :::

## 进入烧录模式

    :::tip

    * 左边是主板的RST,右边是主板的BOOT

    :::

    * 进入烧录模式有两种方法
    * 第一种：主板完全断电情况下，长按BOOT然后将带有数据传输的数据线一头接主板一头接你设备，合适WINDOWS
    * 第二种：在将带有数据传输的数据线一头接主板一头接你设备后，长按BOOT然后输入指令，合适上位机
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/boot.webp').default} size="60%" align="center" />
