# Firmware Burning Instructions

    :::warning Notice

    * PRO X10 comes pre-flashed with Katapult
    * Once BL has been flashed, there is no need to flash BL again
    * PRO X10 has two types of BL, one is the Katapult firmware and the other is the BL firmware flashed via TF card
    * If needed, you can choose to re-flash the BL according to your preference <Button variant="contained" disableElevation href="./bl">BL firmware</Button>

    :::
    
    :::danger Important Notice
    * The motherboard firmware burn needs to be done through Katapult to flash Klipper
    * If it is confirmed that the motherboard has already been flashed with Katapult, there is no need to perform the final Katapult firmware flashing, which is just a precaution in case the Katapult firmware flashing was unsuccessful
    * You can quickly press the reset button twice to see if an LED on the motherboard is blinking, if so, it indicates that the motherboard has entered the Katapult firmware
    :::

## How to Determine Which Firmware You Need

    :::tip

    * Please note that you only need to flash one type of firmware

    :::
    
    * Pro X10 can use the following methods to connect to the host computer
    * <Button variant="contained" disableElevation href="./usb">USB firmware</Button>
        * Advantage: Reliable and stable communication
        * Disadvantage: May affect data transmission if multiple USB devices are used
    * <Button variant="contained" disableElevation href="./can">USB bridge firmware</Button>
        * Advantage: Can connect multiple devices without occupying USB ports
        * Disadvantage: May cause errors such as homing timeout when used together with `KlipperScreen`
