    下面介绍如何编译固件：
    * 连接到SSH后输入下面的命令并回车：

        ```bash
        cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
        ```

    * 其中` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`是删除之前编译数据与固件`make menuconfig`是编译固件,执行后应该出现下方界面

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="left" />

    * 选择**Enable extra low-level configuration options**并且回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="left" />

    * 进入菜单**Micro-controller Architecture**然后选择**Raspberry Pi RP2040**回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="left" />

    * 选择**Bootloader offset**，选择：**16KiB bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.png').default} size="100%" align="left" />

    * 选择**Communication interface**，选择：**CAN bus**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/can.png').default} size="100%" align="left" />

    * 选择`(4) CAN RX gpio number`，删除4输入1
    * 选择`(5) CAN RX gpio number`，删除5输入0

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/can1.png').default} size="100%" align="left" />
