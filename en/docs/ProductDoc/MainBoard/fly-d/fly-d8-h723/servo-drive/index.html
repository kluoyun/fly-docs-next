External Drive Usage Guide

Precautions‚Äã

Notice
  • This tutorial is based on Klipper control
  • The module uses common anode wiring

Drive Wiring‚Äã

info
  • Only one of the following two methods is needed
  • First wiring method
  • Second wiring method

Drive Microstepping Jumper‚Äã

  • No drive jumpers are needed!!!

Built-in Drive DIP Switch‚Äã

  • It is recommended to use 3200 for the built-in drive DIP switch

Klipper Reference Configuration‚Äã

  • The corresponding drive configuration needs to be disabled, such as [tmc5160 stepper_x] or [tmc2209 stepper_x]
  • Set microsteps: to 16
  • Add step_pulse_duration: 0.000004
  • Motor enable: Remove the ! in enable_pin:!

Drive Pulse Explanation‚Äã

info
  • Model misalignment may occur during printing
  • Motor vibration, unusual noises, various strange issues
  • You can modify step_pulse_duration: 0.000004
  • Increase this value to 0.000009, for example, 0.00001 or 0.00002, etc.
