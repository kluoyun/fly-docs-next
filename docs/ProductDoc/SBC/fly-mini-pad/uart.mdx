# 配置方法

:::danger 须知
请注意使用此方法后无法在使用mini pad的加速度接口外接加速度计
:::

    * 打开BOOT盘找到`armbianEnv.txt`并且打开
    * 将`overlays=usbhost2 usbhost3 uart2`中的**uart2**修改成**uart3**保存即可

    <ImageView image={require('./img/uart.png').default} size="80%" align="center" />

## 接线方法

    * 请注意mini-pad的TX要接刷了串口固件主板的RX
    * 请注意mini-pad的RX要接刷了串口固件主板的TX
    * 请注意需要共地
    * 否则无法连接刷了串口固件的主板

    <ImageView image={require('./img/uart1.png').default} size="80%" align="center" />

## Klipper参考配置

    ```cfg
    [mcu]
    serial: /dev/ttyS3
    baud：250000
    #   使用的波特率。默认是250000
    restart_method:command
    ```
