树莓派镜像

warning

SD卡注意事项:

  • 容量必须不小于16GB或不大于128GTF卡
  • TF卡的传输速度必须为class10级或class10级以上，建议使用闪迪等品牌的TF卡
须知
  • 因服务器问题请不要随意更新系统或者更新klipper
  • 适用于树莓派3B/3B+/4B/CM4/5B的系统镜像
  • 本镜像基于bookworm安装各种服务
  • 本镜像默认安装并启用了TFT屏幕，如果不需要使用TFT屏幕需要手动进去/boot/config.txt修改
重要提示
  • 请注意下载好的系统是需要解压，而不是直接改后缀或者直接烧录！！！！
  • 不解压后烧录是无法启动！！！
