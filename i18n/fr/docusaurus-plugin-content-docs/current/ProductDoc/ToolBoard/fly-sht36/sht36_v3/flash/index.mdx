# Notice sur la programmation du firmware
    
    * La compilation et la programmation du firmware nécessitent l’utilisation d’un logiciel tiers.
    * Les SHT36 V3 peuvent être connectés à un ordinateur de contrôle via RS232 ou CAN.
    * RS232 et CAN doivent être connectés via le câble `XT30(2+2)-F`.
    * Voici le modèle du câble `XT30(2+2)-F` :
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Remarque importante
* La mise à jour du firmware de la carte mère doit être effectuée en programmant Katapult pour Klipper.
* Si la carte mère est déjà programmée avec Katapult, il n’est pas nécessaire de programmer le firmware Katapult à la fin, cela ne sert qu’à prévenir une éventuelle erreur de programmation précédente.
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour vérifier si une LED clignote sur la carte mère. Si c’est le cas, cela signifie que la carte mère est déjà dans le firmware Katapult.
:::

## Comment savoir quel firmware vous avez besoin

    * Les SHT36 V3 peuvent être connectés à un ordinateur de contrôle de la manière suivante :
    * <Button variant="contained" disableElevation href="./can">Firmware CAN</Button>
        * Avantages : Si vous installez le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher à la carte mère, et il prend en charge la connexion en bus.
        * Inconvénients : L’utilisation conjointe de KlipperScreen peut entraîner des erreurs telles que le dépassement de délai lors du retour à la position nulle.
        * Conseil : Si vous utilisez CAN, assurez-vous que votre ordinateur de contrôle supporte CAN et qu’il est compatible avec UTOC ou une carte mère flashée avec un firmware pont.
    * <Button variant="contained" disableElevation href="./rs232">Firmware RS232</Button>
        * Avantages : Pas besoin de logiciel spécial pour l’utiliser, communication fiable et stable.
        * Inconvénients : Pour mettre à jour le firmware, vous devez entrer en mode DFU manuel ou appuyer deux fois sur RST pour entrer dans Katapult pour mettre à jour le firmware.
        * Conseil : Vous devez utiliser G2T ou UTOR pour l’utiliser.
