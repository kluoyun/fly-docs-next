システムビルトインコンポーネントの紹介

危ない
  • FlyOS-Fastシステムビルトインのすべてのコンポーネントとサービスは、ユーザーによる変更やアップグレードをサポートしていません。
  • ユーザーによる変更によってウェブページにアクセスできなくなる、システムが起動しなくなるなどの問題が発生する可能性があります。
  • システムを再フラッシュしてください。

Klipper​

手記
  • 既存のKlipperにサードパーティのextrasを追加することは試みることができます。ただし、既存のすべてのextrasモジュールを変更しないでください。

異常自動一時停止​

  • プリント中にKlipperに温度異常やドライバ異常などのエラーが発生した場合、プリントは自動的に一時停止します。装置は直ちに停止しません。

実時間性の最適化​

  • Klipperの実時間性の最適化は、カーネルとKlipperコードの変更により実現されます。Klipperの実時間性を最適化します。
  • Timer too closeエラーを効果的に削減します。

停電時の印刷再開、異常時の印刷再開のサポート​

  • 停電時の印刷再開をサポートしています。
  • エラーによる停止からの印刷再開をサポートしています。

TCPソケットのサポート​

  • UnixソケットとTCPソケットを同時に有効にすることができます。
  • -pパラメータを使用してTCPポートを指定することで、TCPソケットを有効にできます。

USBドライブファイルの自動処理​

  • USBドライブファイルを自動的に処理し、プリントが中断することを防ぎます。

Moonraker​

TCPソケットのサポート​

  • KlippyにTCPソケットを使用して接続することができます。
  • klippy_tcp_addressパラメータを使用してTCPアドレスを指定します。
  • klippy_tcp_portパラメータを使用してTCPポートを指定します。

大ファイルアップロードの最適化​

  • 連続して大ファイルをアップロードすると発生するエラーの問題を解決します。

ネットワーク管理 - FLY5,7スクリーン向け​

  • FLY5,7スクリーン向けの無線ネットワーク管理をサポートしています。

FLYスクリーンのサポート​

  • FLYスクリーンをサポートしています。
  • モデルの初層グラフィックス生成をサポートしています。

KlipperScreen​

停電時の印刷再開​

  • 停電時の印刷再開をサポートしています。
  • 続けて印刷を検出すると自動的にポップアップメッセージを表示します。

Fluidd​

停電時の印刷再開​

  • 停電時の印刷再開をサポートしています。
  • 続けて印刷を検出すると自動的にポップアップメッセージを表示します。

Mainsail​

停電時の印刷再開​

  • 停電時の印刷再開をサポートしています。
  • 続けて印刷を検出すると自動的にポップアップメッセージを表示します。
