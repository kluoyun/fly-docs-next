# Подключение к ПК по SSH

:::danger Внимание
- Если ваш ПК запущен впервые после записи образа, обязательно используйте `Подключение через последовательный порт к SSH на ПК`.
- Во время первого запуска подключайте только питание и Type-C.
- При компиляции прошивки используйте `Подключение через Wi-Fi к SSH на ПК`, чтобы упростить процесс компиляции и записи прошивки!
:::

* Перед началом подключения скачайте: <Button variant="contained" disableElevation component="a" rel="noopener" href="https://mobaxterm.mobatek.net/download-home-edition.html ">SSH-программное обеспечение</Button>. Выберите «Installer edition» справа для загрузки.

        ## Использование последовательного порта для подключения к SSH на ПК

       :::tip Важно
        - Убедитесь, что карта TF или M2WE записаны правильно и установлены на FLY-C5.
        - По умолчанию для системы FLYOS-FAST имя пользователя — `root`, пароль — `mellow`.
       :::

        * Используйте кабель Type-C для подключения FLY-C5 к компьютеру.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * Если материнская плата подключена нормально, откройте диспетчер устройств, и вы увидите порт CH340.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/pc-device-mgr.webp').default} size="50%" align="center" />

        * Откройте «MobaXterm» и выполните следующие настройки, выбрав ранее виденный порт в диспетчере устройств.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-2.webp').default} size="60%" align="center" />

        * Если всё сделано правильно, вы увидите экран загрузки.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial1.webp').default} size="60%" align="center" />  
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial2.webp').default} size="60%" align="center" />


        * Так как система FLYOS-FAST запускается быстро, скорее всего вы увидите мусор на экране, как показано ниже. Это нормально, не беспокойтесь, **нажмите несколько раз Enter**.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial3.webp').default} size="60%" align="center" />


        * После нажатия Enter вы увидите следующее содержимое.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial4.webp').default} size="60%" align="center" />

        * Введите логин: `root`, нажмите Enter, затем введите пароль: `mellow`, нажмите Enter. При вводе пароля ничего не будет отображаться — это нормально.

        :::warning Важно 
        - Для системы FLYOS-FAST логин — `root`, пароль — `mellow`.
        - В системе FLYOS-FAST существует только один аккаунт — `root`.
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial5.webp').default} size="60%" align="center" />

        * Если нужно узнать IP, введите в ssh: `ip a | grep inet`, затем найдите IP в том же диапазоне, что и ваш роутер, чтобы получить доступ.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial6.webp').default} size="60%" align="center" />


    </TabItem>
    <TabItem value="net" label="Подключение через Wi-Fi к SSH на ПК" default>
        ## Использование Wi-Fi для подключения к SSH на ПК
       :::tip Важно
        - Убедитесь, что система ПК уже запущена нормально.
        - По умолчанию для системы FLYOS-FAST логин — `root`, пароль — `mellow`.
        - Предварительным условием для подключения через Wi-Fi является подключение ПК к роутеру и знание его IP-адреса.
       :::

        * Используйте кабель Type-C для подключения FLY-C5 к компьютеру.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * Откройте MobaXterm и установите настройки, как показано на картинке. В поле «Remote host» укажите IP-адрес ПК, который вы определили при подключении через последовательный порт, либо получите IP через интерфейс роутера.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Важно: здесь в четвертом шаге введите логин ПК — `root`**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_2.webp').default} size="60%" align="center" />

        * При первом подключении появится окно, нажмите «Accept».

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Введите пароль, затем нажмите Enter.
        :::tip
        - Здесь вводимый пароль не отображается, просто нажмите Enter после ввода.
        :::
        
        По умолчанию для системы FLYOS-FAST логин — `root`, пароль — `mellow`.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_3.webp').default} size="60%" align="center" />

        **Если появится окно с запросом сохранить пароль пользователя, нажмите «No».**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Если появится следующий экран, подключение успешно завершено!

        :::warning Важно 
        - Для системы FLYOS-FAST логин — `root`, пароль — `mellow`.
        - В системе FLYOS-FAST существует только один аккаунт — `root`.
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_5.webp').default} size="60%" align="center" />



    </TabItem>
</Tabs>
