펌웨어 컴파일 및 플래싱

팁
  • 407ZG는 기본적으로 TF 카드를 사용하여 펌웨어를 플래싱합니다.
  • 용량이 32G를 초과하지 않는 TF 카드를 준비하십시오.
  • 브랜드가 명확하지 않은 메모리 카드는 플래싱 실패를 유발할 수 있으니, 가능한 한 아래 추천 메모리 카드를 사용하여 펌웨어를 업데이트하십시오.
  • TF 카드의 전송 속도는 class10 이상인 것이 좋으며, 샌디스크 등의 브랜드를 사용하는 것이 좋습니다.

준비 작업​

  • klipper 서비스가 설치된 상위 기기를 사용해야 합니다 네트워크를 통해 SSH 연결

    • 그리고 장치를 상위 기기와 연결해야 합니다
    • 주의: 일반적인 상위 기기를 사용하세요, 예를 들어 라즈베리 파이, FLY파이 등, 만약 커스텀 상위 기기라면 문제가 너무 많아서 여기서는 많은 도움을 제공할 수 없습니다
    주의사항
    • ROOT 사용자를 사용하지 마세요

Klipper 펌웨어 컴파일​

팁
  • 请确保上位机可以正常联网
  • 固件编译前需要使用SSH通过网络登录上位机
  • 请不要使用串口工具登录上位机

Klipper固件编译配置页面操作说明​

팁

请确保键盘输入法为半角模式，即英文模式

  1. 键盘上键↑，下键↓，用于上下移动光标来选中菜单项
  2. 确认键Enter 或 空格键Space，用于勾选菜单或进入子菜单
  3. 退出键ESC，用于返回上一级菜单
  4. Q键，用于退出Klipper固件配置页面
  5. Y键，在退出Klipper固件配置页面时，如果有提示，则点击Y键保存配置文件
경고

如果配置页面选项比较少，请先勾选[ ] Enable extra low-level configuration options，用于显示部分隐藏选项

다음은 펌웨어를 어떻게 컴파일하는지 소개합니다:

  • SSH에 연결한 후 아래 명령어를 입력하고 엔터를 누릅니다:

    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig

  • 여기서 rm -rf ~/klipper/.config && rm -rf ~/klipper/out는 이전에 컴파일된 데이터와 펌웨어를 삭제하는 명령어이고, make menuconfig는 펌웨어를 컴파일하는 명령어입니다. 실행 후 아래와 같은 화면이 나타나야 합니다.

  • 추가적인 저수준 설정 옵션 활성화를 선택하고 엔터를 누릅니다.

  • 마이크로컨트롤러 아키텍처 메뉴로 들어가서 STMicroelectronics STM32를 선택하고 엔터를 누릅니다.

  • 프로세서 모델 메뉴로 들어가서 STM32F407을 선택하고 엔터를 누릅니다.

  • 부트로더 오프셋을 선택하고, 32KiB 부트로더를 선택합니다.

  • 통신 인터페이스를 선택하고, **USB (PA11/PA12에 연결됨)**을 선택합니다.

  • Q 키를 누르면 Save configuration이 나타납니다. 이때 Y 키를 다시 누릅니다.

  • 이제 설정이 저장되고 명령줄 인터페이스로 돌아갑니다.

  • make -j4를 입력하여 컴파일을 시작합니다. 시간이 좀 걸릴 수 있습니다.

  • 마지막으로 아래 내용이 출력되면 컴파일이 성공적으로 완료된 것입니다.

    Linking out/klipper.elf
    Creating hex file out/klipper.bin

  • 아래 이미지는 컴파일 성공 예시입니다.

펌웨어 플래싱​

주의사항
  • F404ZG는 TF 카드 플래싱을 위한 BL 펌웨어가 미리 설치되어 있습니다.

준비 작업​

  • 펌웨어를 플래시하기 전에 먼저 네트워크 연결을 통해 SSH 사용
  • 저가형 메모리 카드는 플래시에 실패할 수 있으므로 가능한 아래 권장 메모리 카드를 사용하여 펌웨어를 업데이트하세요
  • 용량이 32G를 초과하지 않는 TF 카드를 준비하고, TF 카드의 전송 속도가 class10 이상이어야 하며, 샌디스크 등의 브랜드 TF 카드를 사용하는 것이 좋습니다

펌웨어 다운로드​

  1. 이전에 준비한 TF 카드를 사용
  2. SDFormatter를 사용하여 포맷
  3. MobaXterm_Personal의 왼쪽 파일 디렉토리에서 klipper를 찾아 선택하고 out 폴더를 엽니다
  4. klipper.bin을 TF 카드에 다운로드하고 firmware.bin으로 이름을 변경
  5. 메인보드 전원을 끄고 TF 카드를 메인보드에 삽입
  6. 메인보드 상에서 10초 기다리기
  7. TF 카드를 컴퓨터에 삽입하여 SD 카드 내의 firmware.bin이 사라지고 FLY.CUR이 나타나면 플래시가 성공한 것입니다
