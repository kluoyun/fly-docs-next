# Firmware Burning Instructions

* SHT36/42 can be connected to the host computer via CAN
* CAN requires MX3.0 metal terminals for crimping
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.png').default} size="80%" align="left" />

## How to Determine Which Firmware You Need

* SHT36/42 can connect to the host computer via CAM
* <Button variant="contained" disableElevation href="./can">Burning CAN Firmware</Button>
    * Advantage: If you flash the `Katapult` firmware, you can update the firmware without touching the motherboard and it supports bus connection
    * Disadvantage: May cause errors such as homing timeout when used with `KlipperScreen`
    * Tip: If you need to use CAN, confirm whether your host computer supports CAN and ensure it is paired with UTOC or a motherboard flashed with bridge firmware
