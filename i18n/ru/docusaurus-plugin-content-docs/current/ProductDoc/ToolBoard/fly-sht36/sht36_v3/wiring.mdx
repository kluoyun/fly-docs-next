# Справочное подключение материнской платы

    :::warning Внимание

    Из-за слишком высокого разрешения изображения, невозможно рассмотреть детали при просмотре на веб-странице. Пожалуйста, поместите курсор мыши на изображение схемы подключения, щелкните правой кнопкой мыши и выберите "Сохранить изображение как...", чтобы сохранить изображение на рабочем столе для дальнейшего просмотра!!!

    :::
