

## Klipper固件编译和配置页面操作说明

    :::tip

    请确保键盘输入法为半角模式，即英文模式

    :::

    1. 键盘上键`↑`，下键`↓`，用于上下移动光标来选中菜单项
    2. 确认键`Enter` 或 空格键`Space`，用于勾选菜单或进入子菜单
    3. 退出键`ESC`，用于返回上一级菜单
    4. `Q`键，用于退出Klipper固件配置页面
    5. `Y`键，在退出Klipper固件配置页面时，如果有提示，则点击`Y`键保存配置文件

    :::warning

    如果配置页面选项比较少，请先勾选`[ ] Enable extra low-level configuration options`，用于显示部分隐藏选项

    :::
