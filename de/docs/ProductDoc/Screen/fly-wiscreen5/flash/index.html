开机图片修改

tip

请严格按着分辨率以及文件名制作图片，否则系统将不会识别

将制作好的图片，放入到SD卡/U盘根目录，在屏幕开机状态下，插入屏幕，点击升级即可

开机LOGO修改​

  • 图片要求800*480
  • 名字要求boot_logo.JPG
  • 不得大于105KB

第一界面图片修改​

  • 图片要求500*400
  • 名字要求print.png
  • 不得大于45KB

打印界面图片修改​

  • 图片要求311*480
  • 名字要求print_smail.png
  • 不得大于45KB
