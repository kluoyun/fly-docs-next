# Introduction

> Fly-D7 is a high-performance 3D printer motherboard launched by the Fly3D motherboard R&D team of Guangzhou Melun Electronic Technology Co., Ltd., supporting up to 7 stepper motors.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.png').default} size="60%" align="center" />

## **1. Product Features**

    * Uses a 32-bit ARM Cortex-M0+ series STM32F072RBT6 main control chip with a frequency of 48 MHz
    * Supports CAN bus connection, ensuring more stable data transmission, lower latency, and a more stable connection
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Board-mounted DIAG function pins, allowing for simple plug-and-play jumper caps for infinite limit switch support

## 2. Product Specifications

    * MCU: STM32F072RBT6
    * Firmware: Klipper/Marlin
    * Supports 12-24V DC power supply, providing a more stable power supply and eliminating the need for a voltage drop module
    * Driver operating modes supported: SPI, UART, STEP/DIR
    * Supported drivers: External TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
    * Expansion interfaces: BLTouch (Servos, Probe), UART X1
    * Fans: Two-wire controllable fans X2, Parallel fans X2, Always-on fans X2
    * Temperature sensors: 3 channels of NTC 100K
    * Heating interfaces: 3 controllable interfaces supporting up to 10A
    * Supported machine structures: Cartesian, Delta, Kossel, Ultimaker, CoreXY
