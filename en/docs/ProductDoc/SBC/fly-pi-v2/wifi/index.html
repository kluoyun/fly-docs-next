Connect to WiFi

tip
  • There are slight differences in the steps and methods for connecting to WiFi between the FLYOS-FAST system and the FLYOS-Armbian system. Please select the tutorial corresponding to the system you have burned!
Warning
  • The FLY host machine requires the use of an M2WE module, a 2.4G WiFi module, or a Linux driver-free USB wireless network card. Otherwise, WiFi cannot be used.
  • Please note that the host machine does not support WiFi with Chinese names, nor does it support 2.4G and 5G dual-band combined WiFi.
  • If you are using a mini pad or lite2/lite2.1, please ignore this warning.
  • Connecting to WiFi on the FLYOS-FAST System
  • Connecting to WiFi on the FLYOS-Armbian System
  • Please first connect to the host machine via serial port: FLYOS-FAST Serial Port Connection to Host Machine SSH

  1. In SSH, input the command nmtui, press the Enter key, and an interface as shown in the figure below will pop up
  1. Select the Activate a connection option, press the Enter key, and enter the next-level menu
Operation Instructions for the WiFi Configuration Page
  • Please ensure that the keyboard input method is in half-width mode, i.e., English mode.
  • The up key ‚Üë and down key ‚Üì on the keyboard are used to move the cursor up and down to select menu items.
  • The confirm key Enter is used to check the menu or enter a sub-menu.
  • The exit key ESC is used to return to the previous menu.
  1. If your router's WiFi is working properly, you will see the WiFi list as shown in the figure below. The * after the WiFi name indicates signal strength
tip
  • WiFi with Chinese names is not supported!!!
  • Support for 2.4G WiFi, but whether support for 5G WiFi depends on whether the purchased WiFi module supports 5G WiFi
  • M2WE supports 5G WiFi
  1. Select the WiFi you need to connect to, press the Enter key, and enter the next-level menu
  1. Then the following interface will pop up. Please correctly input your WiFi password, press the Enter key, and connect to WiFi
  1. If the password is correct, the symbol * will appear before the corresponding WiFi, indicating that the WiFi connection is successful
  1. Then press the Esc key twice to return to the command line interface
  1. In SSH, input ip a | grep inet, press the Enter key, and find the IP in the same segment as your router in the returned content to access
  1. Open the browser, enter the IP of the host computer queried earlier in the search box, press the Enter key, and you can enter the fluidd/mainsail WEB control console
tip
  • In the FLYOS-FAST system, you can click the mainsail/fluidd icon in the upper right corner to switch between fluidd/mainsail WEB control consoles in real time!

FLYOS-Armbian System WiFi Connection‚Äã

  1. Please first connect to the host machine via serial port: FLYOS-Armbian Serial Port Connection to Host SSH

  2. Enter the command sudo nmtui, which may prompt for a password, enter the password mellow, and press the Enter key, a pop-up interface as shown in the figure below will appear (note: the entered password is invisible)

  1. Select the Activate a connection option, press the Enter key to enter the next-level menu
Operation Instructions for the WiFi Configuration Page
  • Please ensure that the keyboard input method is in half-width mode, i.e., English mode.
  • The up key ‚Üë and down key ‚Üì on the keyboard are used to move the cursor up and down to select menu items.
  • The confirm key Enter is used to check the menu or enter a sub-menu.
  • The exit key ESC is used to return to the previous menu.
  1. If your router's WiFi is working properly, you will see the following WiFi list, the * after the WiFi name indicates signal strength
tip
  • WiFi with Chinese names is NOT supported!!!
  • Supports 2.4G WiFi, but whether it supports 5G WiFi depends on whether the purchased WiFi module supports 5G WiFi
  • M2WE supports 5G WiFi
  1. Select the WiFi you want to connect to, press the Enter key to enter the next-level menu
  1. The following interface will then pop up, please correctly enter your WiFi password, press the Enter key to connect to WiFi
  1. If the password is correct, a symbol * will appear before the corresponding WiFi, indicating that the WiFi connection is successful
  1. Then press the Esc key twice to return to the command line interface
  1. In SSH, enter ip a | grep inet, press the Enter key, and find the IP address in the returned content that matches your router to access
  1. Open a browser, enter the IP of the host machine in the search box, press the Enter key, and you can enter the fluidd/mainsail WEB control console
  1. In the browser's search box, enter IP:9999, press the Enter key, and you can enter the fly tools interface, through which you can enable various functions of the host machine
tip
  • fly tools is only available in the FLYOS-Armbian system!!! There is no fly tools in FLYOS-FAST!!!
