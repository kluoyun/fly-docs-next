Katapult 펌웨어

부팅 모드 진입​

팁

원 안에 있는 것이 BOOT 버튼입니다.

  • 부팅 모드로 진입하려면 메인보드의 전원을 완전히 차단한 상태에서, BOOT 버튼을 누른 채로 데이터 전송이 가능한 USB 케이블을 메인보드와 장치에 연결합니다.

펌웨어 플래시​

  • Windows 플래시 가이드
  • 호스트 머신 플래시 가이드
  • USB 가속도계의 Katapult 펌웨어 자동 펌웨어 다운로드
  • 데이터 전송 기능이 있는 데이터 케이블을 준비해야 합니다
  • 펌웨어를 컴퓨터로 다운로드합니다
  • BOOT 점퍼를 연결하거나 길게 누른 후 컴퓨터에 연결합니다. 그러면 RPI-RP2 디스크가 나타납니다
  • BOOT 점퍼를 제거하거나 버튼을 놓습니다!!!
  • 펌웨어를 복사할 필요가 있습니다. UF2 확장자명을 가진 펌웨어만 사용할 수 있습니다!!! (확장자명을 변경하지 마세요, 이는 무용지물입니다)
  • 참고
  • 펌웨어 복사 후 RPI-RP2 디스크가 자동으로 사라집니다. 이 시점에서 펌웨어가 완료됩니다!

  1. 설치 및 플래싱 도구

    • SSH에서 아래 명령어 입력
    • 첫 플래싱 시에만 실행, 이후에는 실행할 필요 없음
    cd ~/klipper/lib/rp2040_flash/ && make

  2. 아래 명령어를 실행하여 연결 여부 확인

    • lsusb 입력

    • 2e8a:0003 장치가 없으면 플래싱 모드로 다시 진입 시도

    • 시스템 버전 문제로 다른 이름이 표시될 수 있지만 2e8a:0003가 있으면 플래싱 모드 진입

    주의사항

    2e8a:0003이 있을 때 다음 단계로 진행

  3. 사전 컴파일된 펌웨어 가져오기

    • 여기에는 사전 컴파일된 펌웨어가 포함되어 있습니다
    • 상위 기기가 인터넷에 연결될 수 있어야 설치 가능
    • 상위 기기에서 아래 명령어를 실행하여 펌웨어 패키지 다운로드
    cd && git clone https://github.com/FengHua741/FLY_Katapult.git
    • 이미 펌웨어 패키지를 다운로드한 경우 업데이트 후 나머지 작업 수행 권장
    cd ~/FLY_Katapult && git pull
  1. 펌웨어 플래시
팁

하나의 펌웨어만 플래시하면 됩니다.

  • 아래는 펌웨어 플래시 명령어입니다. 
    sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/USBADXL/FLY_USBADXL.uf2

  1. 아래 메시지가 나타나면 플래시가 성공한 것입니다.

팁

플래시가 완료된 후, 메인보드의 전원을 완전히 차단해야 합니다.
