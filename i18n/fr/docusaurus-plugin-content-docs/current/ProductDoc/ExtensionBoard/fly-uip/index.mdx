# FLY U-IP
    Ce produit est un module de surveillance d'état multifonctionnel basé sur le microcontrôleur Raspberry Pi RP2040, intégrant une prise USB Type-A mâle et une prise USB Type-C femelle, prenant en charge le branchement sans pilote (Windows/Linux sans pilote). C'est un module de surveillance d'état embarqué basé sur le microcontrôleur Raspberry Pi RP2040, intégrant un écran LCD IPS de 1,14 pouces à angle de vision complet (résolution 240x135), avec une prise USB Type-A mâle et une prise USB Type-C femelle, prenant en charge le branchement sans pilote (Windows/Linux sans pilote). Conçu spécialement pour les scénarios d'impression 3D, les appareils IoT et le débogage réseau, il affiche directement les informations réseau, l'utilisation des ressources système et l'état du service de l'appareil via l'écran, sans outil terminal supplémentaire, avec une opération intuitive et pratique.

## Caractéristiques 

    * Utilise le contrôleur principal Raspberry Pi RP2040, fréquence d'horloge 200 MHz
    * Prend en charge la prise USB Type-A mâle et la prise USB Type-C femelle
    * Equipé d'un écran LCD IPS de 1,14 pouces, résolution 240x135
    * Prise en charge sans pilote sur Windows et les principales distributions Linux
    * Prend en charge l'obtention d'informations IP via Moonraker
    * Prend en charge l'obtention d'informations IP via SSH
    * Le PC peut modifier les paramètres via une page web
    * La couleur du thème de l'écran est personnalisable
    * L'écran supporte deux modes de rotation : 90° et 270°
    * La luminosité du rétroéclairage de l'écran est ajustable
    * Le temps de rafraîchissement et le temps d'affichage de toutes les informations sont personnalisables
    * Option pour afficher ou non l'adresse IPv6
    * Prend en charge le rétablissement des paramètres d'usine par un triple appui sur le bouton RST

## Remarques

    :::warning Remarques

    * Ce produit ne prend pas encore en charge les appareils Apple
    * Un système prenant en charge ECM ou Rndis est nécessaire, sinon il ne fonctionnera pas correctement
    * Ce produit est conçu pour Klipper et nécessite la coopération du service Moonraker pour afficher l'IP et les ressources système. Sinon, il faut activer l'arrière-plan et configurer le mode SSH pour l'utiliser
    
    :::
