## Preparation Before System Flashing

    * Card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, the transmission speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Flashing

:::info
Flashing to TF card and flashing to M2WE via TF card only requires using one of the methods.
:::

:::info
* If the machine is installed with M2WE and memory card, the system will default to boot from the memory card.
* It is recommended to use SD card for flashing to M2WE. Using USB burning to M2W may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Flashing to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os).
* Please note whether the downloaded system image is correct. If it does not match, the system cannot start.
:::
