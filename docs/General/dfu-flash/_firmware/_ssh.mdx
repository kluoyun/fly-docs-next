

    * 需要使用安装好`klipper`服务的上位机  <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/BasicTutorial/">通过网络连接SSH</Button>
    * 并且将设备与上位机连接好，如果是C8、C5、Gemini等板载上位机需要注意拨码位置
    * 注意：请使用普通的上位机，比如树莓派、FLY派等。如果是魔改上位机（例如WiFi棒子、红米等），这类上位机因问题太多，无法为您提供技术支持


    
    :::warning
    * 非FLY上位机，如果不知道账号与密码请咨询卖家
    * FLY上位机分两个系统FAST与FLYOS
    * FLY FAST只能使`root`用户，密码`mellow`
    * FLY OS使用`fly`用户，密码`mellow` 
    * 切换用户可以在SSH中直接输入：`su fly`，非fly用户请将`<您的上位机用户名>`,替换为您上位机的用户名！

    ```bash
     su <您的上位机用户名>
    ```
    :::
