# Connect to BambuLab

:::info Important Tips
* AirClick must be connected to the same network as the target printer.
* Only LAN local area network mode is supported to connect to BambuLab printers.
:::

1. Turn on the LAN mode (LAN Only Mode) of the BambuLab printer.

    - Turn on the BambuLab printer screen.
    - `Settings` -> `LAN Mode (LAN Only Mode)` -> `Enable`
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-1.webp').default} size="40%" align="Center" />

2. Click on the system settings on the left.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="60%" align="Center" />

3. Select `BambuLab` for `Printer Type`.
4. Click the button to refresh the device list.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-2.webp').default} size="60%" align="Center" />

5. Select the printer.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-3.webp').default} size="60%" align="Center" />

    - If your printer never refreshes and appears, you can choose to enter it custom. Manually enter the IP address and serial number of the printer.

6. Fill in the access code.
    - The access code is the LAN mode access code of the printer, which can be viewed in the printer's settings as per step 1.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-4.webp').default} size="60%" align="Center" />

5. After configuring as shown in the above figure, click `Submit` in the lower right corner to save the configuration.
6. After a correct connection, a prompt will appear at the top of the web page.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

7. If the connection fails:
    - Check if the IP address is correct.
    - Check if the printer and AirClick are connected to the same network.
    - Check if the printer has enabled the LAN mode.
    - Check if the access code is filled in incorrectly.
