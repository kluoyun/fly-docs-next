# Firmware Burning Instructions

## How to Determine Which Firmware You Need

    :::tip

    * Note that you only need to flash one of the firmwares

    :::

    * Gemini V3 can be connected to the host machine using the following methods:
    * <Button variant="contained" disableElevation href="./usb">USB Firmware</Button>
        * Advantage: Reliable and stable communication
        * Disadvantage: Data transmission may be affected if multiple USB devices are used
    * <Button variant="contained" disableElevation href="./can">USB Bridge Firmware</Button>
        * Advantage: Can connect multiple devices without occupying a USB port
        * Disadvantage: May cause errors such as homing timeouts when used with `KlipperScreen`.
