ADXL345

产品特点​

  • 适用36电机等地方
  • 可接树莓派、FLY PI、香橙派、主板等SPI接口

产品规格​

  • 板载接口：ZH1.5 6P插头
  • 通讯方式：SPI
  • 工作电压：5V

注意事项​

注意事项
  • 工作电压5V，请不要接错
  • 适用各种固件，请注意需要确定可以使用SPI连接
