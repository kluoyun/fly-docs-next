# 简介

    Fly-Pi-lite2.1是一款入门级Klipper上位机
    <ImageView image={require('./img/lite.png').default} size="80%" align="center" />

## 特点

    * CPU：高性能全志H3芯片，4核64位Cortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：支持最大128GB SDCARD
    * WIFI: 板载2.4G WiFi
    * 外设：Spi x1，URAT x2, USB x2，HDMI x1
    * 接口：预留FPC，HDMI + USB FPC一线通
    * 系统预装klipper、moonraker、mainsail、fulidd、klipper-screen、共振补偿插件 numpy与FLY-Tools，无需换源等复杂的操作，小白更容易上手（需要购买TF卡）


## 更新功能

    * TFT V1接口升级TFT V2
    * 添加重置与关机引脚
    * 添加上电指示灯
    * 删除BOOT按钮，更换重置按钮
    * 优化布线
