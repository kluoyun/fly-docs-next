RRFに接続する

重要な情報
  • AirClickは目標プリンターと同じネットワークに接続する必要があります
  1. 左側のシステム設定をクリックします

  1. Printer Type を RRF に選択します

  2. RepRapFirmware IP にRRFマシンのIPアドレスを入力します

  3. 上記の設定が完了したら、右下隅の提出をクリックして設定を保存します

  4. 正しく接続すると、ウェブページの上部にメッセージが表示されます

  5. 接続に失敗した場合

    • IPアドレスが正しいことを確認してください
    • プリンターとAirClickが同じネットワークに接続されていることを確認してください
    • RRFメインボードが正常に動作していることを確認してください
    • ブラウザでRRFのウェブページを開けるかどうかを確認してください
