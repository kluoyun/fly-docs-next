# Подключение к Wi-Fi


## Установка антенны на PI V2

    * Красной стрелкой указано место подключения антенны Wi-Fi
    * Разъем для антенны Wi-Fi — `ipex1`
    * Установите антенну на PI V2, иначе из-за плохого сигнала может возникнуть проблема с подключением к сети
