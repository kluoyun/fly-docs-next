# Connexion à l'hôte supérieur via SSH

:::danger Attention
- Si votre hôte supérieur démarre pour la première fois après avoir flashé l'image, il est impératif d'utiliser `connexion série à l'hôte supérieur via SSH`.
- Lors du premier démarrage, il est interdit de connecter quoi que ce soit d'autre que l'alimentation et le Type-C !!!
- Lors de la compilation du firmware, il est essentiel d'utiliser `connexion WiFi à l'hôte supérieur via SSH` afin de faciliter la compilation et le flashage du firmware !
:::

* Avant de commencer la connexion, veuillez télécharger : <Button variant="contained" disableElevation component="a" rel="noopener" href="https://mobaxterm.mobatek.net/download-home-edition.html ">logiciel SSH</Button>. Choisissez `Installer edition` à droite pour télécharger.

        ## Utilisation de la connexion série pour accéder à l'hôte supérieur via SSH

       :::tip Remarque
        - Assurez-vous que la carte TF ou M2WE a été correctement gravée avec le système approprié et correctement installée sur FLY-C5.
        - Le nom d'utilisateur par défaut du système FLYOS-FAST est `root`, et le mot de passe par défaut est `mellow`.
       :::

        * Connectez FLY-C5 à l'ordinateur à l'aide d'un câble de données Type-C.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * Si la carte mère est correctement connectée à l'ordinateur, vous verrez le port CH340 dans le Gestionnaire de périphériques.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/pc-device-mgr.webp').default} size="50%" align="center" />

        * Ouvrez `MobaXterm` et effectuez les paramétrages suivants, en sélectionnant le port que vous avez vu dans le Gestionnaire de périphériques.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-2.webp').default} size="60%" align="center" />

        * Si tout fonctionne correctement, vous verrez l'écran de démarrage.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial1.webp').default} size="60%" align="center" />  
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial2.webp').default} size="60%" align="center" />


        * Comme FLYOS-FAST démarre rapidement, il est probable que vous voyiez des codes incorrects, comme illustré ci-dessous. C'est un phénomène normal, ne vous inquiétez pas, **appuyez plusieurs fois sur Entrée**.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial3.webp').default} size="60%" align="center" />


        * Après avoir appuyé sur Entrée, vous verrez le contenu suivant.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial4.webp').default} size="60%" align="center" />

        * Entrez le compte : `root`, puis appuyez sur Entrée, puis entrez le mot de passe : `mellow`, puis appuyez sur Entrée. Le mot de passe n'apparaît pas lors de l'entrée, c'est normal.

        :::warning Attention 
        - Le compte système FLYOS-FAST est `root`, le mot de passe est `mellow`.
        - Le système FLYOS-FAST n'a qu'un seul compte : `root` !!!
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial5.webp').default} size="60%" align="center" />

        * Si vous souhaitez consulter l'adresse IP, tapez `ip a | grep inet` dans SSH et appuyez sur Entrée. Recherchez l'adresse IP dans la même plage que votre routeur pour y accéder.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial6.webp').default} size="60%" align="center" />

        ## Utilisation de la connexion WiFi pour accéder à l'hôte supérieur via SSH
       :::tip Remarque
        - Assurez-vous que le système de l'hôte supérieur a bien démarré.
        - Le nom d'utilisateur par défaut du système FLYOS-FAST est `root`, et le mot de passe par défaut est `mellow`.
        - La condition préalable pour se connecter à l'hôte supérieur via WiFi est que votre **hôte supérieur est déjà connecté au routeur**, et que vous savez son adresse IP.
       :::

        * Connectez FLY-C5 à l'ordinateur à l'aide d'un câble de données Type-C.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * Ouvrez MobaXterm et configurez-le comme indiqué ci-dessous, en entrant l'adresse IP de votre hôte supérieur que vous avez obtenue lors de la connexion série dans le champ `Remote host`. Vous pouvez également rechercher l'adresse IP de l'hôte supérieur via l'interface de configuration du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Note : Dans cette étape 4, entrez l'utilisateur `root` de l'hôte supérieur.**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_2.webp').default} size="60%" align="center" />

        * Lors de la première connexion, une fenêtre de confirmation s'affiche, cliquez sur `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Saisissez le mot de passe et appuyez sur Entrée.
        :::tip
        - Le mot de passe n'apparaît pas lors de l'entrée, appuyez directement sur Entrée après l'avoir saisi.
        :::
        
        Le nom d'utilisateur par défaut du système FLYOS-FAST est : `root`, et le mot de passe est : `mellow`.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_3.webp').default} size="60%" align="center" />

        **Si une fenêtre similaire à celle ci-dessous apparaît, elle vous demande si vous souhaitez enregistrer le mot de passe utilisateur. Cliquez sur `Non`.**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Si vous voyez l'image suivante, la connexion a réussi !

        :::warning Attention 
        - Le compte système FLYOS-FAST est `root`, le mot de passe est `mellow`.
        - Le système FLYOS-FAST n'a qu'un seul compte : `root` !!!
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_5.webp').default} size="60%" align="center" />



    </TabItem>
