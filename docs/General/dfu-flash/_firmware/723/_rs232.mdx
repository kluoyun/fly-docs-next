    下面介绍如何编译固件：
    * 连接到SSH后输入下面的命令并回车：

        ```bash
        cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
        ```

    * 其中` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`是删除之前编译数据与固件`make menuconfig`是编译固件,执行后应该出现下方界面

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="left" />

    * 选择**Enable extra low-level configuration options**并且回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="left" />

    * 进入菜单**Micro-controller Architecture**然后选择**STMicroelectronics STM32**回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="left" />

    * 进入菜单**Processor model**，选择**STM32H723**回车

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="left" />

    * 选择**Bootloader offset**，选择：**NO bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.png').default} size="100%" align="left" />

    * 选择**Clock Reference (8 MHz crystal)**，选择：**25 MHz crystal**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="left" />

    * **Communication interface**，选择：**Serial (on USART1 PA10/PA9)**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/uart.png').default} size="100%" align="left" />
