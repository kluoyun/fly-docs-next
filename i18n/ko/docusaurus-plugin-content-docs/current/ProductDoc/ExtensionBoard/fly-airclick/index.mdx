# AirClick 무선 버튼
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.webp').default} size="100%" align="Center" />



## 제품 특징

    * 높은 호환성: Klipper, RRF, BambuLab을 지원하며, 기계에 직접 연결하여 사전 설정된 명령을 보낼 수 있어 원활한 호환성과 작동 경험을 제공합니다.
    * 사용자 정의 매크로 명령: 사용자는 자신의 요구에 따라 명령을 사용자 정의할 수 있어 기계 작동을 더욱 유연하고 편리하게 합니다.
    * Klipper, RRF, BambuLab에 무선 연결을 지원합니다. 유선 연결은 Klipper만 지원합니다.
    * 무선 연결: ESP32의 설계를 통해 klipper에 직접 사용자 정의 명령 또는 매크로를 보내 사용자에게 높은 유연성과 확장성을 제공합니다.
    * 유선 연결: STM32F072 칩을 통해 네트워크가 Klipper에 연결되지 않는 경우에도 간단한 작업을 할 수 있습니다. 총 10개의 사용자 정의 명령 또는 매크로를 지원합니다.
    * 사용 편리성: 직관적인 사용자 인터페이스를 가지고 있어 사용자가 무선 버튼을 쉽게 추가하고 관리할 수 있으며 작동 편의성을 크게 향상시킵니다.

## 제품 사양

    * 주제어 칩: ESP32
    * 부가 칩: STM32F072
    * ESP32-WIFI를 통해 명령 전송: 제한 없음
    * STM32F072-IO를 통해 명령 전송: 10개
    * 호환성: Klipper, RRF, BambuLab을 지원
    * 전원: 5V

## 주의 사항

    :::danger 주의 사항

    * 이 제품은 5V 전원을 사용하며 `type c` 또는 `PH2.0 - 2P` 단자로 전원 공급이 가능합니다.
    * 이 제품의 온보드 안테나는 버튼 신호를 받는 데만 사용할 수 있습니다. WIFI 연결이 필요한 경우 안테나를 제대로 설치해야 합니다. 안테나 단자는 `ipex3`입니다.
    * 안테나를 제대로 설치하세요!!! 그렇지 않으면 신호가 너무 약해서 과열이 심각해집니다!!!
    * 오작동을 방지하기 위해 버튼을 15초 이상 길게 누르면 시스템이 자동으로 종료됩니다.

    :::
