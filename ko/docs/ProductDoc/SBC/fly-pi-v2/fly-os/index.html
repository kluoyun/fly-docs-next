상위 기기 초보자 운영 가이드​

상위 기기 초보자 운영 가이드

시스템 번인 준비​

  • 카드 리더기
  • TF 카드 1장, 용량은 16GB 이상 128GB 이하, TF 카드의 전송 속도는 class10 이상이어야 합니다.
  • 해당 시스템 이미지 다운로드
  • 필요한 번인 도구 다운로드

시스템 번인​

정보

TF 카드 번인과 TF 카드를 통해 M2WE에 번인하는 것은 둘 중 하나만 사용하면 됩니다.

정보
  • 기기에 M2WE와 메모리 카드가 설치된 경우, 시스템은 기본적으로 메모리 카드에서 시작합니다.
  • M2WE는 TF 카드를 사용하여 시스템을 플래시하는 것을 권장합니다. USB를 통해 M2W에 번인하는 경우 드라이버 호환성 등의 이유로 번인 실패가 발생할 수 있으므로, 메모리 카드를 통해 M2WE에 번인을 직접 사용하는 것을 권장합니다.
  • 다운로드한 시스템 이미지가 해당 기기와 일치하는지 확인하십시오. 일치하지 않으면 시스템이 시작되지 않습니다.
  • TF 카드에 번인
  • TF 카드를 통해 M2WE에 번인
  • USB를 통해 M2WE에 번인
  • M2SD Burner를 통해 M2WE 또는 TF 카드에 번인

시스템 TF 카드에 기록하기

준비 작업​

  • 먼저 Win32DiskImager 소프트웨어를 설치하십시오.
  • 시스템을 압축 해제하여 .img 확장자가 있는 파일을 얻으십시오.
  • balenaetcher를 사용하여 시스템을 기록하는 것은 권장하지 않습니다.
  • TF 카드를 카드 리더기에 삽입하고, 카드 리더기를 컴퓨터의 USB 포트에 연결하십시오.
  • TF 카드에 데이터가 있는 경우, 시스템 기록 후 시스템 카드를 분리하지 않는 것이 좋습니다. M2WE에 기록할 필요가 있는 경우에만 TF 카드를 분리할 수 있습니다.

메모리 카드 포맷하기​

  • TFFormatter.exe 소프트웨어를 사용하여 TF 카드를 포맷하십시오.
  • 포맷할 디스크를 선택하십시오.
  • 또는 Windows 내장 디스크 관리자를 사용하여 파티션을 삭제하고, 파티션을 다시 생성하십시오.

TF 카드에 기록하기​

테스트

  • 빨간색으로 굽힐 파일을 선택
  • 주황색으로 굽힐 TF카드를 선택
  • 파란색으로 시스템을 TF카드에 굽기
  • 굽기를 확인하고 완료될 때까지 기다리기
  • 굽기 완료

TF 카드를 통해 M2WE에 기록하기

정보

시스템 마이그레이션은 TF 카드에 있는 원래 이미지를 M2WE로 완전히 이전할 수 있습니다. 마이그레이션이 완료되면 M2WE의 내용이 원래 TF 카드의 내용과 일치하게 됩니다.

  • M2WE를 설치한 후 전원을 켜서 시작합니다.
  • 먼저 시리얼을 통해 SSH에 연결합니다.
  • 아래 명령어를 실행합니다. 비밀번호를 입력하라는 메시지가 표시되면 mellow를 입력하고 엔터 키를 누릅니다. 비밀번호는 보이지 않습니다.
sudo nand-sata-install
  • Boot from eMMC - system on eMMC 옵션을 선택한 후 엔터 키를 누릅니다. (일반적으로 기본 옵션이므로 엔터 키만 누릅니다.)
  • This script will erase your eMMC. Continue? 메시지가 나타나면 Yes를 선택하고 엔터 키를 누릅니다. (일반적으로 기본 옵션이므로 엔터 키만 누릅니다.)
  • 파일 시스템 유형을 ext4로 선택하고 엔터 키를 누릅니다. (일반적으로 기본 옵션이므로 엔터 키만 누릅니다.)
  • 진행률 표시줄이 나타날 때까지 기다립니다. (2분 이상 기다려도 아래 이미지의 진행률 표시줄이 나타나지 않으면 CTRL+C를 눌러 작업을 종료한 후 장치를 재부팅하고 다시 시도합니다.)
  • 시스템 마이그레이션 중 전원을 차단하거나 다른 작업을 수행하지 마십시오. 마이그레이션이 완료될 때까지 기다립니다.
  • 마이그레이션이 완료되면 엔터 키를 직접 누르고 장치가 완전히 꺼질 때까지 기다립니다.
  • 전원을 껀 후 TF 카드를 제거하고 전원을 다시 켜면 이제 eMMC에 있는 시스템으로 부팅할 수 있습니다.

M2WE에 USB를 통해 이미지 파일을 기록하는 방법

준비 작업​

  • 먼저 Win32DiskImager 소프트웨어를 설치합니다.
  • balenaetcher를 사용하여 시스템을 기록하는 것은 권장하지 않습니다.
  • 시스템을 압축 해제한 후, 파일 이름의 확장자는 .img가 됩니다.
  • M2WE BOOT 드라이버를 다운로드합니다.

드라이버 설치​

  • Zadig 소프트웨어를 엽니다.
  • 첫 번째 Zadig는 설치할 필요 없이 직접 실행할 수 있으며, 나머지 두 개는 설치가 필요합니다. 설치 과정에서는 모두 다음 또는 Next를 클릭하면 됩니다.
  • Zadig를 열고, 상단 메뉴 바의 Options에서 List All Devices를 체크합니다.
팁

장치를 컴퓨터에 연결하기 전에 버튼을 눌러 장치를 BOOT 상태로 만드세요.

  • M2WE를 설치하고, BOOT 버튼을 누른 상태에서 이중 USB를 컴퓨터에 연결합니다.
  • FLY-C8
  • FLY-Pi v2
  • FLY-Pi(π)
  • FLY-Gemini
  • FLY-C8 포트
  • FLY-Pi v2 포트
  • FLY-Pi 포트
  • FLY-Gemini 포트
  • zadig를 엽니다
  • 정상적으로 진행되면 zadig에 Unknow Device #x 또는 USB Device(VID_1f3a_PID_efe8)가 추가되며, 이를 선택합니다.
  • 장치를 선택한 후 USB ID가 위 그림과 일치하는지 확인하고, 일치하지 않으면 다른 장치를 선택합니다.
  • 위 그림의 두 번째 부분은 WinUSB(vxxxxxxx)만 선택할 수 있으며, 버전 번호는 중요하지 않습니다.
  • Install Driver를 클릭하여 드라이버를 설치합니다.
  • USB 드라이버 설치 성공 (드라이버는 한 번만 설치하면 됩니다. 나중에 다시 기록할 때는 드라이버를 다시 설치할 필요가 없습니다.)
  • 설치된 FLY-BOOT를 열고 장치 확인을 클릭합니다.
  • 이전 단계가 올바르게 수행되었다면 사용 가능한 장치가 발견되었다는 메시지가 표시됩니다.
  • BOOT를 시작하고, 정상적으로 진행되면 BOOT가 성공적으로 활성화되었다는 메시지가 표시됩니다.

M2WE 포맷​

  • TFFormatter.exe 소프트웨어를 사용하여 BOOT 디스크를 포맷합니다.
  • 또는 Windows 내장 디스크 관리자를 사용하여 파티션을 삭제하고, 파티션을 다시 생성합니다.
  • 이제 파일 탐색기를 열 수 있으며, 14.5GB의 USB 드라이브가 추가됩니다(16GB eMMC는 14.5GB로 표시됩니다).
  • 이제 Win32diskimager를 사용하여 시스템 이미지를 기록할 수 있으며, 드라이브 문자는 이 14.5GB USB 드라이브의 문자를 선택합니다.

M2WE 또는 SD 카드를 M2WE에 설치​

주의사항
  • 본 제품은 USB3.0을 지원합니다
  • 본 제품은 TF와 M2WE를 동시에 설치하여 사용할 수 없습니다
  • M2WE 또는 SD 카드를 설치한 후 컴퓨터 USB 포트에 연결하여 포맷할 수 있습니다
  • NVME 등 다른 장치를 설치하면 기기 손상이 발생할 수 있습니다!!!
  • M2WE 설치
  • TF 설치

  • TF 카드에 데이터가 있는 경우 백업을 하고, 시스템 번인 후 시스템 카드를 분리하지 않는 것이 좋습니다. M2WE에 번인한 후에만 TF 카드를 분리할 수 있습니다

시스템 설정​

FLYOS_Env 사용법

팁
  • 3.3.4 시스템부터는 FLY-TOOLS를 사용하여 시스템 설정을 할 수 있으며, boot로 들어가서 설정을 수정할 필요가 없습니다.
  • 이전 시스템은 FLY-TOOLS를 직접 사용하여 설정을 지원하지 않습니다.
  • FLY_TOOLS 사용 가이드

WIFI 설정 설명​

주의사항
  • WIFI 설정 방법이 복잡하여, 시리얼 연결을 통해 상위 컴퓨터에 접속한 후 WIFI 연결을 진행해야 합니다.

TF 카드 또는 M2WE를 PI V2에 설치​

  • TF 카드를 PI V2에 설치
  • M2WE 카드를 PI V2에 설치
  • 시스템을 TF 카드에 번인한 후, TF 카드를 PI V2에 연결해야 합니다. 그렇지 않으면 시스템이 시작되지 않습니다.
  • TF 카드가 있는 경우 시스템은 기본적으로 TF 카드에서 시작합니다.
  • 고정 나사를 1.27 육각 렌치로 고정해야 합니다.
