# BL固件

    :::tip

    * 请注意出厂已经预刷了`bootloader`无需在刷

    :::

## 进入烧录模式


    * 进入烧录模式有两种方法
    * 第一种：主板完全断电情况下，短接BOOT然后将带有数据传输的数据线一头接主板一头接你设备
    * 第二种：在将带有数据传输的数据线一头接主板一头接你设备后，短接BOOT然后按下RST键，然后松开RST键最后断开短接BOOT跳线
        <ImageView image={require('./img/boot.png').default} size="100%" align="center" />

            4. 烧录固件

            :::tip

            只需要烧录其中一个固件

            :::


            * 下方是bootloader固件刷写指令
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/Gemini/GeminiV3.bin
                ```

            5. 出现下图箭头处所示内容``File downloaded successfully``则烧录成功

            :::warning

            红色箭头所指这一行**后面的报错**请忽略！！！

            :::
