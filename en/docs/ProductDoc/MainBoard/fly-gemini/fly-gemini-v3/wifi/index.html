Connect WIFI

Install the Antenna on Gemini‚Äã

  • The place pointed to by the red arrow is the WIFI antenna interface
  • The WIFI antenna port is ipex1
  • Install the antenna on Gemini, otherwise it may not be able to connect to the network due to poor signal

FLYOS-Armbian System WiFi Connection‚Äã

  1. Please first connect to the host machine via serial port: FLYOS-FAST Serial Port Connection to Host SSH

  2. Enter the command sudo nmtui, which may prompt for a password. Enter the password mellow and press the Enter key, which will pop up the interface shown in the figure below (note: the input password is not visible)

  1. Select the Activate a connection option, press the Enter key to enter the next level menu
Operation Instructions for the WiFi Configuration Page
  • Please ensure that the keyboard input method is in half-width mode, i.e., English mode.
  • The up key ‚Üë and down key ‚Üì on the keyboard are used to move the cursor up and down to select menu items.
  • The confirm key Enter is used to check the menu or enter a sub-menu.
  • The exit key ESC is used to return to the previous menu.
  1. If your router's WiFi is working properly, you will see the WiFi list as shown in the figure below. The * after the WiFi name indicates signal strength
tip
  • WiFi with Chinese names is NOT supported!!!
  • 2.4G WiFi is supported, but whether 5G WiFi is supported depends on whether the purchased WiFi module supports 5G WiFi
  • M2WE supports 5G WiFi
  1. Select the WiFi you want to connect to, press the Enter key to enter the next level menu
  1. Next, the following interface will pop up. Please correctly enter your WiFi password, press the Enter key, and connect to WiFi
  1. If the password is correct, a symbol * will appear before the corresponding WiFi, indicating that the WiFi connection is successful
  1. Then press the Esc key twice to return to the command line interface
  1. In SSH, enter ip a | grep inet, press the Enter key, and find the IP address in the returned content that is in the same segment as your router to access it
  1. Open the browser, enter the IP of the host machine you just queried in the search bar, and press the Enter key to enter the WEB control console of fluidd/mainsail
  1. In the browser's search bar, enter IP:9999, press the Enter key, and enter the fly tools interface. Through the fly tools interface, you can enable a series of functions of the host machine
tip
  • fly tools is ONLY available in the FLYOS-Armbian system!!! There is NO fly tools in FLYOS-FAST!!!
