Gravure du système FLYOS-FAST

Gravure du système​

Tutoriel de gravage du système FLYOS-FAST : tutoriel de gravage du système FLYOS-FAST




Présentation du système​

  • FlyOS-Fast est un système Linux allégé, spécialement conçu pour l'impression 3D, avec un démarrage rapide (seulement 30 secondes). Compatible avec toutes les plateformes supérieures Fly.
  • Noyau optimisé pour améliorer les performances temps réel de Klipper.
  • Système de fichiers optimisé pour éviter les dommages au système lors d'une coupure de courant directe.
  • Préinstallé avec des services comme Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen.
  • Prise en charge de la bascule entre Klipper et RRF.
  • Prise en charge de l'activation en ligne entre Mainsail et Fluidd.
  • Prise en charge de l'impression via USB.
  • Avec le module KPPM, il est possible de reprendre une impression après une coupure de courant.
  • Prise en charge des paramètres prédéfinis tels que la langue par défaut de Fluidd et le nom d'imprimante par défaut.
  • Pour plus de fonctionnalités personnalisées OEM, comme le logo de démarrage, les fichiers de configuration préconfigurés ou la masquage des pages Mainsail, contactez-nous.

Préparation​

  1. Télécharger l'image système

    • Lien de téléchargement : Image système Piv2 (support uniquement Windows)
    • Lien de téléchargement : Image système Piv2 (support Windows, MacOS, Linux)

  2. Télécharger l'outil de gravure. Le tutoriel utilisera Rufus en exemple. En pratique, d'autres outils peuvent être utilisés, comme Etcher, Win32 Disk Imager.

    • Adresse de téléchargement de Rufus : Cliquer pour télécharger en chinois ou Cliquez pour télécharger en anglais (support uniquement Windows).
    • Adresse de téléchargement d'Etcher : Cliquer pour télécharger (support Windows, MacOS, Linux).
    • Adresse de téléchargement de Win32 Disk Imager : Cliquer pour télécharger (support uniquement Windows).

  3. Le tutoriel utilisera FlyOS-Fast-H5.exe en exemple.

  4. Décompresser le fichier image système pour obtenir le fichier .img.

    • Double-cliquez ou ouvrez-le avec un clic droit sur FlyOS-Fast-H5.exe, attendez quelques secondes jusqu'à ce que la fenêtre suivante s'affiche (si le système affiche un avertissement de sécurité, choisissez « Plus d'informations » -> « Exécuter »).
    • Vous pouvez choisir le dossier de décompression, puis cliquez sur le bouton Extraire, attendez que l'extraction soit terminée.
    • Après l'extraction, un fichier FlyOS-Fast-H5-vx.x.x.img sera généré dans le dossier de décompression. x.x.x représente le numéro de version du système.

Gravure de l'image​

  • Gravure vers carte TF
  • Gravure via USB vers M2WE
  • Gravure via M2SD Burner vers M2WE ou carte TF

Gravure vers carte TF​

    • La carte TF utilisée pour la gravure doit être supérieure à 4 Go, il est recommandé d'utiliser une carte SD SanDisk de 32 Go et classe 10 (les tests ont montré que de nombreuses cartes bon marché ne peuvent pas démarrer après la gravure).
    • La carte TF utilisée pour la gravure doit être insérée dans un lecteur de carte SD, puis insérée dans l'ordinateur.
  2. Ouvrir Rufus, sélectionner l'appareil. L'appareil sélectionné doit correspondre en capacité à la carte gravée insérée dans l'ordinateur.
  1. Sélectionner le fichier image, cliquez sur le bouton Type de guide sélectionné à droite et sélectionnez le fichier .img extrait.
  2. Après avoir sélectionné le fichier image, Type de guide sélectionné deviendra le nom du fichier image.
  1. Cliquez sur le bouton Démarrer, si une fenêtre apparaît pour demander de supprimer les données du disque, cliquez sur OK.
  1. Après la gravure, l'ordinateur émettra un signal sonore et la barre de progression indiquera Prêt.
  1. Après la gravure, ne retirez pas immédiatement la carte gravée.
  2. Ouvrez l'Explorateur de fichiers, vérifiez si un lecteur amovible ou une clé USB nommé FlyOS-Conf existe.
  1. Ouvrez FlyOS-Conf, si les deux fichiers suivants sont présents, cela signifie que la gravure a réussi.
  1. Parmi eux, config.txt est le fichier de configuration système, veuillez consulter la page suivante pour son contenu. Modifier le fichier de configuration.

Gravure via USB vers M2WE​

  1. Télécharger les pilotes :

    • Pilote M2WE
    • Outil de démarrage M2WE

  2. Installer les pilotes :

    • Ouvrir Zadig, cochez List All Devices dans le menu déroulant Options
  1. Installez M2WE, puis appuyez sur le bouton BOOT sur la carte mère, et connectez l'ordinateur supérieur via un câble USB double-tête
  1. Ouvrir zadig
  1. Si tout se passe bien, Zadig affichera un Device Inconnu #x ou USB Device(VID_1f3a_PID_efe8), sélectionnez-le
  1. Vérifiez que l'ID USB correspond à celui de l'image ci-dessus, sinon sélectionnez un autre appareil
  2. Dans le deuxième endroit, vous pouvez seulement sélectionner WinUSB(vxxxxxxx), la version du pilote n'est pas importante
  3. Cliquez sur Installer le pilote pour installer le pilote, le pilote doit être installé une seule fois sur le même ordinateur
  1. Ouvrir le programme FLY-BOOT installé, cliquez sur Vérifier le périphérique, si les étapes précédentes sont correctement effectuées, un message apparaîtra indiquant que le périphérique est disponible
  1. Cliquez sur Démarrer BOOT, si tout va bien, un message apparaîtra indiquant que le BOOT est activé avec succès, attendez quelques secondes pour passer à l'étape suivante
  1. Ouvrir Rufus, sélectionner l'appareil. L'appareil sélectionné doit correspondre en capacité à la carte gravée insérée dans l'ordinateur.

Gravure via M2SD Burner vers M2WE ou carte TF​

Remarques importantes
  • Ce produit supporte USB 3.0
  • Ce produit ne supporte pas l'utilisation simultanée de TF et M2WE
  • Une fois que M2WE ou une carte SD est installée, il peut être formaté en la connectant à un port USB de l'ordinateur
  • Ne pas installer NVME ou d'autres dispositifs, sinon cela pourrait endommager le dispositif !!!

  1. Insérer M2WE ou une carte SD dans M2SD. Si une carte TF est utilisée pour la gravure, elle doit être supérieure à 4 Go, il est recommandé d'utiliser une carte SanDisk de 32 Go et classe 10 (les tests ont montré que de nombreuses cartes bon marché ne peuvent pas démarrer après la gravure)

    • Installation de M2WE
    • Installation de la carte SD

  1. Sélectionner le fichier image, cliquez sur le bouton Type de guide sélectionné à droite et sélectionnez le fichier .img extrait.
  2. Après avoir sélectionné le fichier image, Type de guide sélectionné deviendra le nom du fichier image.
  1. Cliquez sur le bouton Démarrer, si une fenêtre apparaît pour demander de supprimer les données du disque, cliquez sur OK.

Configuration du WiFi​

  1. Utiliser vscode ou un autre éditeur de texte professionnel pour ouvrir config.txt (Essayez d'éviter d'utiliser le bloc-notes intégré de Windows pour éditer le fichier de configuration)
  1. Dans l'option WIFI_SSID="", remplissez les "" avec votre nom de réseau WiFi
    Dans l'option WIFI_PASS=""", remplissez les "" avec votre mot de passe WiFi
    Enregistrez et fermez, faites sortir le lecteur amovible ou la clé USB nommé FlyOS-Conf.

Démarrage de la station de travail​

  1. Avant de démarrer, assurez-vous que la carte mémoire ou M2WE est correctement installée sur la station de travail
  1. Installer l'antenne du module M2WE ou du module 2.4G WiFi
note
  • La station de travail FLY nécessite l'utilisation du module M2WE ou du module 2.4G WiFi, sans quoi elle ne pourra pas utiliser le WiFi
  • Notez que la station de travail ne prend pas en charge les noms de réseau WiFi en chinois, et ne prend pas en charge les réseaux WiFi 2.4G et 5G combinés
  • L'endroit marqué en rouge est l'interface de l'antenne WiFi
  • L'interface WiFi est ipex1
  • Assurez-vous d'installer l'antenne sur PI V2, sinon une faible puissance du signal peut entraîner une impossibilité de connexion au routeur
  1. Alimentez la station de travail.
    Ici, votre station de travail est déjà démarrée !!
    Veuillez exécuter la prochaine étape : Connexion SSH à la station de travail
