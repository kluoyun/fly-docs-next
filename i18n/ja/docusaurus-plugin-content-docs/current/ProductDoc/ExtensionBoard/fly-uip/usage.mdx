# 使用方法

## 接続方法
    :::warning 注意事項

    * 接続方法は以下の2つの方法しか使用できません。どちらか一つを選択してください。他の接続方法はありません！！！

    :::

    ``方法一：`` U-IPのUSB A（オス）を上位機のUSB A（メス）に接続します
    
    ``方法二：`` データ線（USB to TYPE-C）を使用し、一方を上位機のUSB A（メス）に、もう一方をU-IPに接続します

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/host.webp').default} size="100%" align="left" />


    ``その後：`` 線を接続したら正常に待機してください

# U-IPの設定方法

    :::danger 注意事項

    * Appleユーザーは現在サポートされていません
    * 使用にはWINDOWSシステムが必要です
    * 接続方法は以下の2つの方法しか使用できません。どちらか一つを選択してください。他の接続方法はありません！！！

    :::
    ``方法一：`` U-IPのUSB A（オス）をコンピューターのUSB A（メス）に接続します
    
    ``方法二：`` データ線（USB to TYPE-C）を使用し、一方をコンピューターのUSB A（メス）に、もう一方をU-IPに接続します
    
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/pc.webp').default} size="100%" align="left" />


    ``その後：`` 5秒待ってからWindowsのブラウザを開き、アドレスバーに``http://192.168.4.1/``と入力してEnterを押すと、下記の画面が表示されます

## インターフェース紹介

### ネットワーク設定
    * U-IPのIPアドレス、MACアドレス、ホスト名、およびバックエンド管理ポートを設定できます
    * 誤って設定しないでください
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/network.webp').default} size="100%" align="left" />

### 表示設定
    * 表示方向、バックライトの明るさ、接続されたマシンのIPv6アドレスを表示するかどうかなどを設定できます
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/display.webp').default} size="100%" align="left" />

### システム制御
    * 固件バージョンなどを設定でき、デバイスを再起動したり、U-IPの書き込みモードに入ることができます
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/system_control.webp').default} size="100%" align="left" />

## システム接続設定

### Moonraker 設定
    * 最初はMoonraker接続を有効にするかどうかです。デフォルトではオンになっています
    * 2つ目はマシンに自動的に割り当てるIPアドレスです
    * 3つ目はMoonrakerのポート番号で、デフォルトは7125です
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/mmoonraker.webp').default} size="100%" align="left" />

### SSH 設定
    * 特殊なシステムではこの方法を試すことができますが、すべてのマシンがこの機能を正常に使用できるわけではありません
    * SSH接続をオンにしてから、マシンのSSHアカウントとパスワードを入力します。ポート番号はデフォルトで22です
    * このオプションではSSHでマシンにログインし、マシンのIPとリソース情報を取得します。取得できない場合は起動画面で無限ループが発生します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ssh.webp').default} size="100%" align="left" />
