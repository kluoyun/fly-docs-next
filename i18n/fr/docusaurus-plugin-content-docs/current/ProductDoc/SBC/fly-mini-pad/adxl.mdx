# Utilisation de l'accéléromètre

    :::tip

    * Le Mini Pad peut utiliser l'interface embarquée pour se connecter directement à un accéléromètre ou utiliser une version de l'outil avec un accéléromètre intégré

    :::

## Interface embarquée pour connecter un accéléromètre externe

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.webp').default} size="80%" align="left" />

    * Configuration de référence

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # un exemple
    ```
