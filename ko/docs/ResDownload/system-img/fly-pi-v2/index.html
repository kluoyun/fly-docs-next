FLY-Pi v2 시스템 이미지

FLY-Pi-v2 시스템 다운로드

경고

SD카드 주의사항:

  • 용량은 16GB 이상 128GB 이하의 TF 카드여야 합니다.
  • TF 카드의 전송 속도는 class10 등급 이상이어야 하며, 샌디스크 등 브랜드의 TF 카드 사용을 권장합니다.
주의
  • 서버 문제로 인해 시스템이나 Klipper를 무작정 업데이트하지 마십시오.
  • FLY OS 시스템을 다운로드하십시오. 원본 Armbian은 사용이 어려워 설치를 권장하지 않습니다.
중요한 알림
  • 다운로드한 시스템은 압축을 해제해야 하며, 후위를 변경하거나 직접 번인하지 마십시오!!!
  • 압축을 해제하지 않고 번인하면 부팅할 수 없습니다!!!

시스템 다운로드​

  • FLYOS(아르비안 기반)
  • 원본 Armbian
팁
  • 시스템에는 Klipper, Moonraker, Mainsail, Fluidd, Klipper-Screen, Crowsnest, 공진 보정 플러그인, Numpy 및 FLY-Tools가 미리 설치되어 있어 소스를 변경하거나 복잡한 작업 없이도 초보자도 쉽게 접근할 수 있습니다. (TF 카드 또는 M2WE 구매 필요)
시스템 이미지 다운로드
  • 일반 사용자
  • 루트 사용자

기본 계정: fly

기본 비밀번호: mellow

기본 계정: root

기본 비밀번호: mellow

경고
  • 이 시스템은 원본 Armbian으로, 어떤 서비스도 포함되어 있지 않으며, 직접 설치해야 합니다.
  • 이 시스템은 M2WE 부팅을 지원하지 않습니다.
