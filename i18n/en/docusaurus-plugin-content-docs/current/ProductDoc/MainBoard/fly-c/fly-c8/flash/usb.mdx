# USB Firmware Compilation

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press the `Q` key, and when **Save configuration** appears, press the `Y` key again.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * You should now have saved the configuration and exited to the command line interface.
    * Enter `make -j4` to start compiling; this will take some time.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * When you see the following output, the compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware Flashing

:::tip
Run the following command to flash the firmware.
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
