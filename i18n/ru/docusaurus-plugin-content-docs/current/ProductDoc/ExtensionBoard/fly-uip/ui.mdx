## Первый интерфейс
    :::warning Важные замечания

    * Требуется поддержка системы ECM или Rndis, иначе функционирование невозможно
    * Данное устройство разработано специально для Klipper и требует совместной работы с сервисом Moonraker для отображения IP и ресурсов; в противном случае необходимо включить后台 и настроить режим SSH для использования
    
    :::
    * Отображение IP происходит на основе информации, считываемой с подключенного устройства; если информация не может быть прочитана или не получена IP-адрес, будет показано соответствующее уведомление
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ip.webp').default} size="100%" align="left" />
    * При успешном чтении отображается следующий интерфейс (IP отображается в соответствии с верхним уровнем)
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/ip1.webp').default} size="100%" align="left" />

## Второй интерфейс

    * Отображает использование процессора и оперативной памяти подключенного устройства
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-uip/img/cpu.webp').default} size="100%" align="left" />

## Третий интерфейс

    * Отображает запущенные службы верхнего уровня; обратите внимание, что отображаются только службы, связанные с klippper
    * Если служба не запущена, перед ней отображается красная точка
    * Если служба запущена, перед ней отображается синяя точка
