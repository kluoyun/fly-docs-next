소개

1. 제품 소개

FLY TOOL-LITE-CAN은 광저우 매륜 전자 과학 기술 유한 회사가 3D 프린터를 위해 설계한 툴보드입니다. 이 툴보드는 xt30/kf 인터페이스를 제공하며, 복잡한 배선을 대체하여 기존의 메인보드와 프린트 헤드 간의 복잡한 연결선을 최적화합니다. TOOL-LITE-CAN은 36/42 스텝 모터에 적합하며 CANBus 수신기, TypeC 포트, TMC2209 드라이버, LIS2DW 가속도 센서, 2개의 제어 가능한 팬 등의 기타 기능을 통합한 툴보드입니다. 이 툴보드는 Klipper 펌웨어에 적합합니다.

1.1 제품 특징​

  • 부트 버튼이 예약되어 있어 사용자는 TypeC를 통해 부트 모드로 진입하여 펌웨어를 업데이트할 수 있습니다.
  • Klipper 펌웨어를 지원합니다.
  • CANBus 모드를 지원합니다.
  • CANBus 모드에서는 더 높은 전송 속도와 강력한 안정성을 제공합니다.
  • 온도 센서 지원: ntc100k 및 PT1000 온도 센서.
  • 두 개의 제어 가능한 팬을 지원하며, 최대 출력 전류는 1A까지 가능합니다.
  • 보드에 LIS2DW 가속도 센서가 내장되어 있습니다.
  • 고압 프로브 인터페이스가 예약되어 있으며, 보드에 2209 드라이버가 내장되어 있어 다양한 레벨링 센서를 지원합니다. 예를 들어 BLtouch, klicky, Voron Tap, PL08n 등
  • 12-24V 전압, 15A 전류 입력을 지원하며, 열 막대 포트를 확장하여 10A의 정격 전류를 제공합니다.

1.2 새로운 기능​

  • 1미터 전원/신호 연결선이 추가되었습니다. 연결선은 쌍絞設計으로, 방해 신호 저항 성능이 더 강합니다.

1.3 제품 사양​

  • MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
  • 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류를 지원하며, 피크 전류는 30A까지 가능합니다.
  • 펌웨어: Klipper
  • 입력 전압: DC12V-24V
  • 논리 전압: DC3.3V
  • 보드에 장착된 센서: LIS2DW
  • 열 막대 인터페이스: 열 막대(HEAT0), 10A
  • 확장 인터페이스: 고압 프로브 인터페이스, EndStop, PT100/PT1000, XH2.54-2P CAN 확장 포트
  • 팬 인터페이스: 두 개의 DC 제어 가능한 팬
  • 모터 드라이버: 보드에 장착된 TMC2209
  • 온도 센서: 100K NTC 또는 PT1000 (TH0)
