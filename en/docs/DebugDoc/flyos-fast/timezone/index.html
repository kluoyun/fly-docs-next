Must Read
  • To set the time zone, fill in the time zone ID in the localtime configuration item, for example: Asia/Shanghai.
  • It takes effect after the system is restarted.
  • The time on the system is incorrect without an internet connection, and it will automatically synchronize after connecting to the internet.
