Ajout du bouton sans fil

Remarque importante
  • Le bouton sans fil nécessite l'installation d'une batterie, le modèle de batterie est CR927
  • Retournez à la page d'accueil et cliquez sur le + dans le cadre rouge de l'image
  • Une fois cette interface apparue, appuyez normalement sur le bouton sans fil avec la batterie installée
  • Un ID unique pour le bouton sans fil sera automatiquement généré dans le cadre rouge
  • Saisissez le nom que vous souhaitez donner au bouton dans le cadre rouge
  • Cliquez sur le premier cadre rouge, le deuxième cadre bleu apparaîtra
À savoir
  • Le GCODE personnalisé et la demande d'API sont contrôlés via la connexion sans fil
  • L'activation du GPIO est contrôlée par câble

GCODE personnalisé​

  • Personnalisez le GCODE, vous pouvez entrer le code ou le nom de la macro que vous souhaitez exécuter dans le cadre rouge, AirClick l'enverra automatiquement à klipper pour exécution

Demande d'API​

  • Vous pouvez personnaliser des macros, des arrêts d'urgence, des redémarrages de klipper, des redémarrages de firmware, l'arrêt du système, etc.

Activation du GPIO​

  • L'activation du GPIO nécessite une connexion entre le STM32F072 et klipper

  • Pour le flashage du firmware et la connexion, veuillez consulter ce document Flashage du firmware et connexion

  • Les IO disponibles sont PB0 à PB9, soit dix IO au total. Une fois le firmware correctement flashé et connecté à Klipper, une configuration normale suffit. Voici un exemple de configuration :

    [mcu AirClick]
    serial: /dev/serial/by-id/usb-Klipper_stm32f072xb_AirClick-if00
    ### Vous devez rechercher l'ID USB et modifier cette partie

    [gcode_button _test]
