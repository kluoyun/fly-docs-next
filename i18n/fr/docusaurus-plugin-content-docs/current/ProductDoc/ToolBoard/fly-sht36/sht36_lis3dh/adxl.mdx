# Utilisation de l'accéléromètre


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Mise à jour de Klipper

    * Klipper a ajouté le support pour l'accéléromètre LIS3DH le 13 novembre 2024. Si vous souhaitez utiliser l'accéléromètre LIS3DH, vous devez mettre à jour Klipper.
    * Après vous être connecté à l'ordinateur hôte via le réseau, entrez le code ci-dessous (la mise à jour peut échouer en raison de problèmes de réseau) :
        ```bash
        cd ~/klipper 
        git pull
        ```
## Ajout de la configuration


    ```cfg
    #####################################################################
    # Configuration de l'accéléromètre LIS3DH (à activer si nécessaire)
    #####################################################################
    [lis3dh]
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis3dh
    probe_points:
        125, 125, 20  # Ici, définissez les coordonnées où la tête d'impression se trouve lors de la mesure de la résonance


```
