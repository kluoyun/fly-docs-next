# 项目结构介绍

## 介绍
#### Logic文件夹
* 我们99%的ui代码编程都在logic文件夹，其它文件夹无需关注
* gcodeLogic.cc类似于main.c,    99%的ui代码都存在这里，其他文件无需多关注

#### i18n文件夹
* 存放各国语言翻译表

#### resources文件夹
* 存放ui使用的图片资源

#### ui文件夹
* 存放ui界面文件，双击打开显示
* 双击左下角箭头所指“小眼睛”可以打开隐藏的界面
