# FLY U-IP
    Данное устройство представляет собой многофункциональный модуль мониторинга состояния на базе микроконтроллера Raspberry Pi RP2040, оснащённого разъёмами Type-A Male и Type-C Female, обеспечивающий возможность Plug and Play (Windows/данный продукт является встраиваемым модулем мониторинга состояния на базе микроконтроллера Raspberry Pi RP2040, оборудованным 1,14-дюймовым IPS-экраном полного обзора (разрешение 240x135), а также разъёмами Type-A Male и Type-C Female, поддерживающим Plug and Play (без драйверов для Windows/Linux). Разработано специально для сценариев работы с 3D-принтерами, интернета вещей и отладки сетей; предоставляет возможность напрямую с экрана отображать сетевую информацию устройства, использование системных ресурсов и статусы запущенных служб без необходимости использования дополнительных терминальных инструментов, что делает работу интуитивно понятной и удобной.

## Особенности 

    * Использует микроконтроллер Raspberry Pi RP2040 с тактовой частотой 200 МГц
    * Поддерживает USB Type-A Male и USB Type-C Female
    * Снабжено 1,14-дюймовым ЖК-экраном IPS с полным обзором, разрешением 240x135
    * Без драйверов для Windows и распространённых дистрибутивов Linux
    * Поддерживает получение IP-адресов и другой информации через Moonraker
    * Поддерживает получение IP-адресов и другой информации через SSH
    * Позволяет компьютеру изменять параметры через веб-интерфейс
    * Цветовая тема экрана настраивается
    * Экран может поворачиваться на 90° и 270°
    * Яркость подсветки экрана регулируется
    * Время обновления и отображения всех данных настраивается
    * Возможность выбора отображения или скрытия IPv6-адресов
    * Поддерживает триггер RST для восстановления заводских настроек

## Важные замечания

    :::warning Важные замечания

    * Данный продукт пока не поддерживает устройства Apple
    * Требуется поддержка ECM или Rndis, иначе устройство невозможно использовать
    * Данное устройство разработано специально для Klipper, требуется совместная работа с сервисом Moonraker для отображения IP-адресов и системных ресурсов; в противном случае необходимо включить фоновые службы и настроить режим SSH для использования
    
    :::
