# Product Introduction

    FLY-ERCF is a multi-color control board for 3D printers based on the RP2040, primarily used in the Enraged Rabbit project. It can also be used for other two-axis stepper motor control situations.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.png').default} size="100%" align="left" />

## Product Features

    * MCU: Raspberry Pi rp2040, Dual-core ARM Cortex-M0+ @ 133MHz
    * Two drive positions, supporting UART and SPI drivers
    * Compatible with the original ERCF EASY Board
    * Extreme miniaturization, using a 4-layer PCB design
    * Onboard 2.54mm dual-row 7-pin expansion port, expandable for limit switches and other uses
    * Supports 12-24V voltage
    * Supports CAN bus and RS232 connections, ensuring more stable data transmission with lower latency and more stable connections
    * CAN interface uses XT30 (2+2), supporting 15A current

## Product Specifications

    * MCU: Raspberry Pi rp2040, Dual-core ARM Cortex-M0+ @ 133MHz
    * Firmware: Klipper
    * Input Voltage: DC12V-24V
    * Interfaces: 2.54mm dual-row 7-pin expansion port, SERVO X1, ENDSTOP X2, EXTRA X1
    * Drivers: Supports UART/SPI and STEP/DIR
