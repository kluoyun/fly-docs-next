# システム紹介とダウンロード

## 概要

* FlyOS-Fastは、軽量化されたLinuxシステムで、3Dプリントに特化しており、Flyシリーズのすべての上位機プラットフォームをサポートしています。
* FlyOS-Fastは軽量化されており、高速に起動します。
* カーネルがKlipperのリアルタイム性を保証するために最適化されています。
* ファイルシステムが最適化され、直ちに電源が切れた場合でも起動できないリスクを減らします。
* 高速起動で、30秒以内にKlipperscreenを起動します。
* Klipper、RRF、Mainsail、Fluidd、Moonraker、KlipperScreenなどのサービスが事前にインストールされています。
* KlipperとRRFのシステム切り替えが可能です。
* MainsailとFluiddのオンラインでのワンクリック切り替えが可能です。
* USBメモリでの印刷が可能です。
* 停電時の継続印刷が可能です。
* シリアルポートスクリーンがサポートされています。
* fly-c8、fly-geminiv3での自動ファームウェア書き込みがサポートされています。
* FluiddなどのフロントエンドUIのデフォルト言語やデフォルトプリンタ名を事前に設定できます。
* 更なるOEMカスタマイズ機能（例：起動ロゴ、事前設定済みの構成ファイル、Mainsailタブページの非表示設定など）についてはお問い合わせください。

## システムイメージファイル名の説明

* システムファイルの命名規則：`FlyOS-Fast-[チップシリーズ]-[バージョン番号].img`、例：`FlyOS-Fast-H5-v1.0.0.img`。
* ソースファイルは`.img`形式ですが、ダウンロードリンクでは`.exe`と`.7z`形式の圧縮ファイルのみ提供されます。
* Windowsシステムの場合、`.exe`ファイルをダウンロードしてください。このファイルは7z自解凍圧縮パッケージです。ダウンロード後にダブルクリックして開くことで、`.img`ファイルを取得できます（この解凍はWindows専用です）。
* MacOSやLinuxシステムの場合、`.7z`ファイルをダウンロードしてください。このファイルは7z圧縮パッケージです。ダウンロード後に7z形式に対応したツールを使用して解凍すると、`.img`ファイルが得られます。
* システムファイルは非常に大きいため、他の圧縮形式やソースイメージのダウンロードは一時的に提供されません。

:::tip システム選択

* fly-C5（全志H618チップを使用するFlyシリーズ製品）は、`FlyOS-Fast-H618.exe`または`FlyOS-Fast-H618.7z`を選択してください。
* fly-piシリーズ、fly-c8シリーズ、fly-geminiシリーズなど、全志H5チップを使用するすべてのFlyシリーズ製品は、`FlyOS-Fast-H5.exe`または`FlyOS-Fast-H5.7z`を選択してください。
* fly-pi-lite2シリーズ、fly-miniPadシリーズなど、全志H3チップを使用するすべてのFlyシリーズ製品は、`FlyOS-Fast-H3.exe`または`FlyOS-Fast-H3.7z`を選択してください。
* チップモデルが異なるため、システムファイルも異なります。実際の状況に応じて選択してください。

:::

## システムイメージのダウンロード

* FlyOS-Fastの公式ダウンロードアドレス：[FlyOS-Fast製品リポジトリ](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/packages)

- デフォルトのユーザー名：`root`
- デフォルトのパスワード：`mellow`

:::tip 注意

このシステムは軽量化されたカスタムシステムであり、組み込まれているKlipperなどのコンポーネントやサービスは公式のオリジナルではありません。ユーザーによる変更やアップグレードはサポートされません。

:::

## システムイメージのダウンロード手順

* FlyOS-Fastイメージのダウンロード手順：

    1. ブラウザで上のFlyOS-Fast製品リポジトリにアクセスします。

    2. 使用する圧縮形式を選択し、ファイルページに進みます。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down1.webp').default} size="50%" align="center" />

    3. バージョンを選択します（最新バージョンがデフォルト）。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down2.webp').default} size="50%" align="center" />

    4. ダウンロードボタンが見えない場合は、マウスをファイルリストに移動し、Shiftキーを押しながらマウスホイールをスクロールすることで、ファイルリストを左右にスクロールできます。
    5. または、下部のスクロールバーにマウスを置き、左ボタンを押し続けながら右方向にドラッグすることで、ダウンロードボタンを見ることができます。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down3.webp').default} size="50%" align="center" />

    6. ダウンロードボタンをクリックすると、対応するバージョンのシステムファイルをダウンロードできます。
