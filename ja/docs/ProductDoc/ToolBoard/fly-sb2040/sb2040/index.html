製品紹介​

FLY SB2040は、広州镁伦電子科技有限公司がVORON Stealthburberプリントヘッド用に開発したノズルツールボードです。このツールボードを使用することで、従来のメインボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化できます。SB2040は36ステップモータに対応し、CANトランスシーバー、USBポート、TMC2209ドライバ、ADXL345加速度センサー、2つの制御可能なファンなど、他の機能を統合したツールボードです。このツールボードはKlipperファームウェアに対応しています。

  • CANバス接続をサポートし、データ伝送が安定し、遅延が少なく、接続が安定
  • 内蔵TMC2209ドライバ、UARTモードで使用可能、押出機に使用
  • 内蔵ADXL345加速度センサー
  • BLTOUCH、PL08Nなどのレベリングセンサーをサポート、XYリミットスイッチを統合
  • 12-24Vの電圧入力に対応
  • MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
  • 4.7K抵抗（ntcサーモで使用）または1K抵抗（PT1000で使用）の切り替えを追加
  • canインターフェースはXT30（2+2）を使用、15A電流、ピーク30Aをサポート
  • 内蔵NTC100k抵抗を追加、倉庫温度測定に使用
  • ADXL345のINT1とINT2をmcuに接続、一部のユーザーはレベリングに使用
  • 高圧入力リミットをサポート
  • 3メートルのCAN接続ケーブルを無料で提供、ユーザーが正しく端子を圧着できないことによる信号喪失の問題を回避
  • 2/3/4線ファンに対応
  • 分離設計、Stealthburnerの分解が簡単
  • hartkのStealthburner Toolhead Boardと互換（ファンの小型ボードは交換不要）
  • インターフェース：制御可能なファン3、RGB1、リミット3、ヒーティングチューブ1、サーモ*1

製品仕様​

  • Klipper CANモード、RS232モード、RRFCANモードの相互切り替えをサポート
  • 接続インターフェースはXT30(2+2)コネクタを使用、15A電流、ピーク30Aをサポート
  • MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
  • 入力電圧：DC12-24V
  • ヒーティングロッドの最大出力電流：5A
  • ファンインターフェースの最大出力電流1A
