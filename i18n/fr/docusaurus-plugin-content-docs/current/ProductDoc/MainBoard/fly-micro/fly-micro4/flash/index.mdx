# Instructions de gravure du firmware

:::danger À savoir
* La gravure du firmware de la carte mère doit être effectuée via Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin d'effectuer la dernière étape de gravure du firmware Katapult, cela n'est que pour prévenir une éventuelle erreur de gravage du firmware Katapult
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère, si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult
:::

## Comment déterminer quel type de firmware vous avez besoin

    :::tip

    * Notez bien que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * E3 V2 peut utiliser les méthodes suivantes pour se connecter à l'ordinateur principal
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont présents, cela pourrait affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantage : peut connecter plusieurs appareils sans occuper de ports USB
        * Inconvénient : peut provoquer des erreurs telles qu'un dépassement de temps d'attente lors de l'utilisation conjointe de `KlipperScreen`
:::
