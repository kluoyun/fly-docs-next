FLY-MiniPad系统下载

warning

SD卡注意事项:

  • 容量必须不小于16GB或不大于128GTF卡
  • TF卡的传输速度必须为class10级或class10级以上，建议使用闪迪等品牌的TF卡
须知
  • 因服务器问题请不要随意更新系统或者更新klipper
  • 请下载FLY OS系统，原版Armbian使用难度过大不建议安装
重要提示
  • 请注意下载好的系统是需要解压，而不是直接改后缀或者直接烧录！！！！
  • 不解压后烧录是无法启动！！！
