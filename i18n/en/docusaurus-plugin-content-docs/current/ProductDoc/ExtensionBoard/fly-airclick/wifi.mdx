# Connect to WiFi

:::info Important Tips

* The on-board antenna of this product can only be used for reception. The transmitting antenna needs to be installed by yourself, and the antenna terminal is `ipex3`.
* Please install the antenna in place!!! Otherwise, the poor signal will cause serious heat generation!!!
* For the first use, you need to manually connect to the AirClick hotspot, and then connect to the corresponding WIFI.

:::


## Connect to WIFI via AirClick

* Connect to the hotspot of AirClick, and the hotspot name is `FLY-WBTN-XXXXXX`.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
* After connecting to the hotspot, access [http://192.168.4.1/](http://192.168.4.1/#/setting) through the browser.
* At this time, the WIFI setting interface will be displayed. You need to manually connect to the same router as the printer, enter the password, and then click the blue box below to connect to WIFI.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
* After connecting to WiFi, the IP address of AirClick after connecting to WiFi will be prompted. Click the blue box below to restart the device.
    * The one circled by number 1 is the IP address of AirClick.
    * The one circled by number 2 is to copy the IP and restart the device.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
* Please note that you need to manually switch your current network to the corresponding router, otherwise, you cannot access it.
* After a successful connection, it is recommended to power off and restart AirClick.
