# Фирменное ПО BL

    :::tip

    * Обратите внимание, что при出厂е уже предварительно записан `bootloader`, поэтому повторная запись не требуется
    * C5 может использовать только родной ПО для записи
    :::

## Подключение по SSH к ПО

    :::danger Предупреждение
    - Перед записью Фирменного ПО BL необходимо подключиться к ПО через `wi-fi SSH`
    :::

 * Сначала выполните: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh#подключение-по-wifi-к-ssh-у-ПО">Подключение по wi-fi к SSH у ПО</Button>

## Запись Фирменного ПО


    * Переведите переключатель вверх, подключите нижнее устройство к верхнему через USB
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * Введите следующую команду для начала записи BL


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```

    * Когда появится изображение ниже, это означает успешную запись и можно отпустить кнопку BOOT
       <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/hid3.webp').default} size="70%" align="center" />
