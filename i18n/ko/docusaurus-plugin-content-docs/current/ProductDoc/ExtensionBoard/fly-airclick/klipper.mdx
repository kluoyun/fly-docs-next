# Klipper에 연결하기

    :::info 중요한 팁
    * AirClick는 목표 프린터와 동일한 네트워크에 연결되어야 합니다.
    :::

1. 왼쪽 시스템 설정을 클릭합니다.

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. `Printer Type`을 `Klipper`로 선택합니다.
3. `Moonraker IP`에 klipper 기계의 IP 주소를 입력합니다.
4. `Moonraker Port`에 klipper 상위 기계의 포트를 입력합니다. 기본값은 `7125`이며, 상위 기계가 다른 포트를 사용하는 경우 해당 포트를 직접 입력해야 합니다.
5. `Apikey`는 무시하고 관리하지 않습니다. 현재 Apikey는 지원되지 않습니다.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. 위의 그림과 같이 구성한 후 오른쪽 아래의 `제출`을 클릭하여 구성을 저장합니다.
7. 정확하게 연결되면 웹 페이지 상단에 알림이 표시됩니다.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. 연결에 실패한 경우
    - IP 주소와 포트가 올바른지 확인합니다.
    - 프린터와 AirClick가 동일한 네트워크에 연결되어 있는지 확인합니다.
    - klipper 상위 기계가 정상적으로 작동하는지 확인합니다.
    - 브라우저에서 klipper 상위 기계의 웹 페이지를 열 수 있는지 확인합니다.
