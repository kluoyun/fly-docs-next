# Connexion à BambuLab

    :::info Remarque importante
    * AirClick doit être connecté au même réseau que l'imprimante cible.
    * Seule la connexion en mode LAN (réseau local) est prise en charge pour les imprimantes BambuLab.
    :::

1. Activez le mode LAN (LAN Only Mode) de l'imprimante BambuLab

    - Allumez l'écran de l'imprimante BambuLab.
    - `Paramètres` -> `Mode LAN (LAN Only Mode)` -> `Activer`
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-1.webp').default} size="40%" align="Center" />

2. Cliquez sur les paramètres système à gauche

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="60%" align="Center" />

3. Sélectionnez `BambuLab` pour `Printer Type`
4. Cliquez sur le bouton de rafraîchissement de la liste des appareils

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-2.webp').default} size="60%" align="Center" />

5. Sélectionnez l'imprimante

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-3.webp').default} size="60%" align="Center" />

    - Si votre imprimante n'apparaît jamais lors du rafraîchissement, vous pouvez choisir d'entrer manuellement. Entrez manuellement l'adresse IP et le numéro de série de l'imprimante.

6. Remplissez le code d'accès
    - Le code d'accès est le code d'accès en mode LAN de l'imprimante, vous pouvez le consulter dans les paramètres de l'imprimante en suivant l'étape 1.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-4.webp').default} size="60%" align="Center" />

5. Une fois que vous avez configuré comme indiqué ci-dessus, cliquez sur `Soumettre` en bas à droite pour enregistrer la configuration.
6. Une fois connecté correctement, un message s'affichera en haut de la page web.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

7. Si la connexion échoue
    - Vérifiez si l'adresse IP est correcte.
    - Vérifiez si l'imprimante et AirClick sont connectés au même réseau.
    - Vérifiez si le mode LAN de l'imprimante est activé.
    - Vérifiez si le code d'accès est mal saisi.
