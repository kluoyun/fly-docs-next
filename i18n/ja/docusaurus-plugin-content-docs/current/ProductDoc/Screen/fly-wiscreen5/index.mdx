# 導入

```
Fly_WiScreenは、广州镁伦电子科技有限公司のFly3Dマザーボード開発チームが提供する高性能なKlipper制御ディスプレイです。WiFiを通じて3Dプリンターを制御でき、SDカードやUSBメモリでの印刷ファイルの転送をサポートします。画面の応答速度が非常に速く、ユーザーに優れた体験を提供します。マスター・スレーブモードの設定をサポートし、複数の機器を同期操作することでユーザーが機器を管理しやすくなります。5Vの電源ポートは、バッテリー改造用の充電モジュールを接続して持ち運びを容易にすることができます。
    
<ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen5/img/wiscreen.webp').default} size="100%" align="left" />
```

## 製品の特徴

```
* Klipperがインストールされた他のホストコンピュータへのWIFI経由の接続をサポート
* WIFI検索をサポートしており、WIFIに接続するだけでホストコンピュータを自分で検索できます
* 複数のプリンターのクラスター制御をサポート
* USBとTF カードを使用したアップロードと印刷をサポート
* カスタマイズされたマクロにより、操作とデバッグがより便利になります
```

## 製品仕様

```
* CPU: Allwinner T113-S3、デュアルコア CORTEX-A7 最大 1.2GHZ
* メモリ: 統合 128M DDR3
* ストレージ: 128MB NANDフラッシュ、外部:TFカードサポート
* ディスプレイ: 800X480静電容量式スクリーン
* 外部インターフェース: USB 2.0 x 1、TF カードホルダー x 1
* WIFIプロトコル: 802.11b/g/n
* 画面タイプ: IPS
* 画面サイズ：5インチ
* マルチタッチ: サポート
* 定格電圧: 5V
```
