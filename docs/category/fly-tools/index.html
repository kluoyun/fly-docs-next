Fly-Tools

📄️ 简介

FLY-Tools是广州镁伦电子科技有限公司针对Klipper所定制的一款Web调试工具，不仅适用于Fly上位机（系统自带，无需安装），还兼容市面是大部分安装了Klipper服务的上位机。

📄️ FLY_TOOLS安装

1. 拉取最新版本Klipper

📄️ 搜索ID

* 支持FLY主板固件编译

📄️ 固件编译

支持一键复制设备ID，自动识别ID并且前面对应的配置添加serial ，device:*

📄️ 生成负载图等

* 支持生成Klipper默认负载图

📄️ 网页SSH

* 无需通过SSH工具即可连接上位机SSH

📄️ 系统配置

* 支持主机名修改
