ファームウェア更新と書き込み

注意事項
  • 出荷時に既にファームウェアがプリインストールされています

ファームウェアのダウンロード​

  • ファームウェアの説明
  • wbtn-vX.X.X-X-XXXXXX-full.binは完全なファームウェアで、0x0アドレスから書き込む必要があり、書き込み後にすべての設定が保持されません
  • wbtn-vX.X.X-X-XXXXXX-delta.binは差分ファームウェアで、0x10000アドレスから書き込む必要があり、既存の設定に影響を与えずにファームウェアのみを更新します
  • wbtn-vX.X.X-X-XXXXXX-www.binはUIファームウェアで、0x290000アドレスから書き込む必要があり、既存のファームウェアや設定に影響を与えずにウェブフロントエンドUIのみを更新します
  • ファームウェアパッケージのダウンロード

ファームウェア更新​

  • AirClickを電源オフにして、BOOTに従ってAirClickのESP32書き込みポートをコンピューターに接続する必要があります
  1. USB書き込みツールのダウンロードと解凍 ダウンロード
  2. flash_download_tool_3.9.4.exeを開きます。注意：書き込みツールを開くとまずCMDウィンドウが表示され、その後書き込みツールのインターフェースが表示されます。CMDウィンドウを絶対に閉じてはいけません
  3. 書き込みツールでChipType:にESP32-C3を選択
  4. 書き込みツールでWorkMode:にDevelopを選択
  5. 書き込みツールでLoadMode:にUSBを選択
  6. 最後にOKをクリックし、下の図を参照してください
  7. クリック後、下の画像が表示されます
ヒント
  • 上記の画像の緑色の枠で囲まれた部分は画像と一致していることを確認してください、自主的に変更しないでください
  1. AirClickをUSB-Typecケーブルでコンピューターに接続
  2. 上記の画像の青い領域1ボタンをクリックして、書き込むファームウェアを選択
  3. 上記の画像の青い領域2に書き込みアドレスを入力
  4. 上記の画像の青い領域3でAirClickのシリアルポートを選択（表示されない場合は、AirClickの書き込みボタンを押し続けてからコンピューターに接続してください）
  5. 上記の画像の青い領域4で最大921600のボーレートを選択
  6. 上記の画像の青い領域5をクリックして書き込みを開始し、書き込み完了を待ってからAirClickのファームウェア更新が完了します（書き込み完了の表示が出るまで、接続を切断しないでください）
