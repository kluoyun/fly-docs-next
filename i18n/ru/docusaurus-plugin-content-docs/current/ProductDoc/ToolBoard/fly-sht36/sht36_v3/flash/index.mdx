# Инструкция по прошивке прошивки
    
    * Для компиляции прошивки и её записи требуется ПО верхнего уровня
    * SHT36 V3 может подключаться к ПО верхнего уровня как через RS232, так и через CAN
    * RS232 и CAN подключаются с использованием `XT30(2+2)-F`
    * Ниже приведён вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Прошивка материнской платы осуществляется через Katapult для записи Klipper
* Если материнская плата уже содержит прошивку Katapult, то последний шаг по записи прошивки Katapult не требуется — это просто мера на случай неправильной записи прошивки
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли светодиод на материнской плате. Если да, то это указывает на наличие прошивки Katapult
:::

## Как определить нужную прошивку

    * SHT36 V3 может подключаться к ПО верхнего уровня следующим образом
    * <Button variant="contained" disableElevation href="./can">CAN-прошивка</Button>
        * Преимущество: если установлена прошивка Katapult, обновление прошивки возможно без физического контакта с материнской платой, а также поддерживаются шинные соединения
        * Недостаток: совместное использование с `KlipperScreen` может вызвать ошибки типа превышения времени ожидания при возврате домой
        * Подсказка: если требуется использовать CAN, убедитесь, что ПО верхнего уровня поддерживает CAN, и используйте его совместно с UTOC или материнской платой с прошивкой моста
    * <Button variant="contained" disableElevation href="./rs232">RS232-прошивка</Button>
        * Преимущество: не требует специального ПО для использования, надёжная и стабильная связь
        * Недостаток: для повторной прошивки требуется войти в DFU вручную или дважды нажать кнопку RST для входа в Katapult и обновления прошивки
        * Подсказка: необходимы G2T или UTOR для работы
